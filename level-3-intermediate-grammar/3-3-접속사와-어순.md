# 3-3. 접속사와 어순 (Voegwoorden)

네덜란드어의 어순은 접속사에 따라 크게 달라집니다. 이것은 네덜란드어 문법에서 가장 중요한 규칙 중 하나입니다.

### 등위접속사 (Nevenschikkende voegwoorden) — 어순 변화 없음

등위접속사는 두 개의 독립적인 절을 연결합니다. 어순에 영향을 주지 않습니다.

| 접속사 | 뜻 | 예시 |
|--------|-----|------|
| **en** | 그리고 | Ik werk en hij studeert. |
| **maar** | 그러나 | Ik wil komen, maar ik heb geen tijd. |
| **of** | 또는 | Wil je koffie of thee? |
| **want** | 왜냐하면 | Ik blijf thuis, want ik ben ziek. |
| **dus** | 그래서 | Het regent, dus ik neem een paraplu. |

> 등위접속사 뒤에서는 어순이 평서문과 동일합니다: **주어 + 동사**

**등위접속사 추가 예문:**

```
en:   Ik kook en mijn man wast af.
      (나는 요리하고 남편은 설거지한다.)

en:   Zij studeert Nederlands en ze werkt bij een bedrijf.
      (그녀는 네덜란드어를 공부하고 회사에서 일한다.)

maar: Het eten was duur, maar het was heel lekker.
      (음식이 비쌌지만, 아주 맛있었다.)

maar: Ik spreek een beetje Nederlands, maar ik maak nog veel fouten.
      (네덜란드어를 조금 하지만, 아직 실수를 많이 한다.)

of:   Gaan we naar het strand of blijven we thuis?
      (해변에 갈까 아니면 집에 있을까?)

of:   Je kunt een broodje of een salade nemen.
      (샌드위치나 샐러드를 먹을 수 있어.)

want: Ik ga vroeg slapen, want ik ben heel moe.
      (일찍 자러 갈게, 왜냐면 아주 피곤하거든.)

want: We nemen de auto, want het is te ver om te fietsen.
      (차를 가져갈게, 자전거 타기엔 너무 멀거든.)

dus:  Ik heb geen geld meer, dus ik kan niet mee-eten.
      (돈이 더 이상 없어서 같이 식사할 수 없어.)

dus:  Het was mooi weer, dus we zijn naar het park gegaan.
      (날씨가 좋았기 때문에 공원에 갔다.)
```

> **want vs omdat**: 둘 다 "왜냐하면"이지만, **want**는 등위접속사(어순 변화 없음)이고 **omdat**는 종속접속사(동사가 끝으로)입니다.

```
Ik blijf thuis, want ik ben ziek.     (want: 주어+동사 순서 유지)
Ik blijf thuis omdat ik ziek ben.     (omdat: 동사가 절의 끝으로)
```

### 종속접속사 (Onderschikkende voegwoorden) — 동사가 끝으로!

종속접속사 뒤에서는 활용된 동사가 **절의 끝**으로 이동합니다. 이것이 네덜란드어 어순의 핵심입니다.

**기본 종속접속사:**

| 접속사 | 뜻 | 예시 |
|--------|-----|------|
| **dat** | ~라고 (that) | Ik denk **dat** hij ziek **is**. |
| **omdat** | 왜냐하면 (because) | Ik blijf thuis **omdat** ik ziek **ben**. |
| **als** | 만약/~할 때 (if/when) | **Als** het regent, blijf ik thuis. |
| **wanneer** | ~할 때 (when) | Ik bel je **wanneer** ik thuis **ben**. |
| **toen** | ~했을 때 (when, 과거) | **Toen** ik jong **was**, woonde ik in Seoul. |
| **terwijl** | ~하는 동안 | Ik lees **terwijl** hij **kookt**. |
| **voordat** | ~하기 전에 | Ik eet **voordat** ik **vertrek**. |
| **nadat** | ~한 후에 | **Nadat** hij gegeten **had**, ging hij slapen. |
| **hoewel** | ~에도 불구하고 | **Hoewel** het regent, ga ik fietsen. |
| **of** | ~인지 (whether) | Ik weet niet **of** hij **komt**. |
| **zodat** | ~하기 위해 | Ik studeer hard **zodat** ik kan **slagen**. |

**추가 종속접속사:**

| 접속사 | 뜻 | 예시 |
|--------|-----|------|
| **totdat** | ~할 때까지 | Ik wacht **totdat** je klaar **bent**. (네가 준비될 때까지 기다릴게.) |
| **zodra** | ~하자마자 | **Zodra** ik thuis **ben**, bel ik je. (집에 도착하자마자 전화할게.) |
| **tenzij** | ~하지 않는 한 | Ik kom morgen, **tenzij** het **regent**. (비가 오지 않는 한 내일 갈게.) |
| **doordat** | ~때문에 (원인) | Hij viel **doordat** het glad **was**. (미끄러웠기 때문에 그는 넘어졌다.) |
| **nu** | ~하는 이상/지금 | **Nu** je hier **bent**, kunnen we beginnen. (네가 여기 있으니 시작할 수 있어.) |
| **sinds/sedert** | ~한 이후로 | **Sinds** ik in Nederland **woon**, spreek ik Nederlands. (네덜란드에 산 이후로 네덜란드어를 한다.) |
| **ofschoon** | ~에도 불구하고 (격식) | **Ofschoon** hij moe **was**, werkte hij door. (피곤했음에도 계속 일했다.) |
| **indien** | 만약 (격식) | **Indien** u vragen **heeft**, kunt u ons bellen. (질문이 있으시면 전화해주세요.) |
| **mits** | ~하는 조건으로 | Je mag gaan, **mits** je op tijd terug **bent**. (제시간에 돌아오는 조건으로 가도 돼.) |
| **naarmate** | ~함에 따라 | **Naarmate** je meer **oefent**, word je beter. (연습할수록 나아진다.) |
| **opdat** | ~하기 위해 (격식) | Ik spreek langzaam **opdat** je me kunt **verstaan**. (네가 이해할 수 있도록 천천히 말한다.) |
| **zoals** | ~처럼/~와 같이 | Doe het **zoals** ik het **doe**. (내가 하는 것처럼 해.) |

> **핵심**: 종속접속사 뒤에서는 활용된 동사가 **절의 끝**으로 갑니다!

```
주절:    Ik denk       (나는 생각한다)
종속절:  dat hij ziek is  (그가 아프다고)
         ↑ 접속사  ↑ 동사가 끝으로

전체: Ik denk dat hij ziek is.
```

**als vs wanneer vs toen — 시간 접속사 구분:**

| 접속사 | 시제 | 용법 | 예시 |
|--------|------|------|------|
| **als** | 현재/미래 | ~할 때, 만약 | Als ik tijd heb, ga ik sporten. |
| **wanneer** | 현재/미래 | ~할 때 (als보다 격식) | Wanneer komt de trein aan? |
| **toen** | **과거만** | ~했을 때 | Toen ik kind was, woonde ik in Busan. |

> **한국어 화자 주의**: 한국어에서는 "~할 때"를 시제에 관계없이 같은 형태로 쓰지만, 네덜란드어에서는 과거에는 반드시 **toen**을 써야 합니다. ~~Als ik kind was~~는 틀린 문장입니다!

```
현재: Als het mooi weer is, ga ik naar het strand.
      (날씨가 좋으면 해변에 간다.)

과거: Toen het mooi weer was, ging ik naar het strand.
      (날씨가 좋았을 때 해변에 갔다.)

미래: Als het morgen mooi weer is, ga ik naar het strand.
      (내일 날씨가 좋으면 해변에 갈 거야.)
```

**omdat vs doordat — "왜냐하면"의 차이:**

| 접속사 | 용법 | 예시 |
|--------|------|------|
| **omdat** | 일반적인 이유/동기 | Ik ga niet mee omdat ik geen zin heb. (하고 싶지 않아서 안 간다.) |
| **doordat** | 직접적 원인 (물리적) | De weg is nat doordat het heeft geregend. (비가 와서 도로가 젖었다.) |

### 종속절이 먼저 오는 경우

종속절이 문장 앞에 오면, 주절에서 **도치**가 일어납니다:

```
Omdat ik ziek ben,  blijf  ik  thuis.
종속절              동사   주어  나머지
왜냐하면 아프기 때문에, 있다  나는  집에.
```

> **도치 원칙**: 네덜란드어에서 동사는 항상 **주절의 두 번째 위치**에 옵니다. 종속절 전체가 첫 번째 위치를 차지하므로, 그 다음에 바로 동사가 와야 합니다.

**다양한 종속절 도치 예문:**

```
① Omdat het regent, neem ik een paraplu mee.
   (비가 오기 때문에, 우산을 가져간다.)
   → 종속절이 먼저 → 도치: neem + ik

② Als ik klaar ben, ga ik naar huis.
   (끝나면, 집에 갈 거야.)
   → 종속절이 먼저 → 도치: ga + ik

③ Toen ik in Nederland aankwam, sprak ik geen woord Nederlands.
   (네덜란드에 도착했을 때, 네덜란드어를 한 마디도 못 했다.)
   → 종속절이 먼저 → 도치: sprak + ik

④ Hoewel hij moe was, werkte hij de hele nacht door.
   (피곤했음에도 불구하고, 밤새 일했다.)
   → 종속절이 먼저 → 도치: werkte + hij

⑤ Voordat je vertrekt, moet je je kamer opruimen.
   (떠나기 전에, 방을 정리해야 해.)
   → 종속절이 먼저 → 도치: moet + je

⑥ Zodra de winkel opengaat, ga ik boodschappen doen.
   (가게가 열자마자, 장을 보러 갈 거야.)
   → 종속절이 먼저 → 도치: ga + ik

⑦ Sinds ik hier woon, heb ik veel vrienden gemaakt.
   (여기 산 이후로, 많은 친구를 사겼다.)
   → 종속절이 먼저 → 도치: heb + ik
```

### 종속절에서 동사가 여러 개인 경우

조동사나 완료시제가 포함된 종속절에서는 모든 동사가 절의 끝으로 갑니다:

```
① 조동사 + 원형동사:
   Ik weet dat hij goed Nederlands kan spreken.
   (나는 그가 네덜란드어를 잘 할 수 있다는 것을 안다.)
   → kan spreken (조동사 + 원형) 둘 다 끝으로

② 현재 완료:
   Ik weet dat hij een boek heeft gelezen.
   (나는 그가 책을 읽었다는 것을 안다.)
   → heeft gelezen (조동사 + 과거분사) 둘 다 끝으로

③ 조동사 + 완료:
   Ik denk dat hij het boek moet hebben gelezen.
   (나는 그가 그 책을 읽었어야 한다고 생각한다.)
   → moet hebben gelezen 모두 끝으로
```

> **한국어 화자에게 좋은 소식**: 종속절에서 동사가 끝으로 가는 것은 한국어의 "~한다고/~하기 때문에" 등과 비슷한 구조입니다. 한국어도 동사가 절의 끝에 오는 SOV 어순이므로, 이 부분은 한국어 화자에게 상대적으로 자연스럽습니다.

### 복합 문장 연습

**여러 접속사를 사용한 복합 문장들:**

```
① Ik wil naar Nederland verhuizen, maar ik moet eerst Nederlands leren,
   want de meeste mensen spreken Nederlands op het werk.
   (네덜란드로 이사하고 싶은데, 먼저 네덜란드어를 배워야 해,
   왜냐하면 직장에서 대부분 네덜란드어를 쓰거든.)

② Hoewel ik al drie jaar Nederlands studeer, vind ik het nog steeds
   moeilijk als mensen snel praten.
   (3년째 네덜란드어를 공부하고 있지만, 사람들이 빨리 말하면
   아직도 어렵다.)

③ Toen ik voor het eerst naar Amsterdam ging, was ik verbaasd
   dat zoveel mensen Engels spraken.
   (처음 암스테르담에 갔을 때, 그렇게 많은 사람들이 영어를
   하는 것에 놀랐다.)

④ Zodra je je diploma hebt gehaald, kun je een baan zoeken,
   tenzij je wilt doorstuderen.
   (졸업장을 받자마자 일자리를 구할 수 있어,
   계속 공부하고 싶지 않다면.)

⑤ Sinds ik in Nederland woon, heb ik geleerd dat je altijd
   een paraplu moet meenemen, omdat het weer heel snel verandert.
   (네덜란드에 산 이후로, 날씨가 아주 빨리 변하기 때문에
   항상 우산을 가지고 다녀야 한다는 것을 배웠다.)
```

### 연습 대화: 접속사 활용

```
A: Waarom leer je Nederlands?
   (왜 네덜란드어를 배워?)

B: Ik leer Nederlands omdat ik in Amsterdam wil werken.
   (암스테르담에서 일하고 싶기 때문에 네덜란드어를 배워.)

A: Hoe lang studeer je al?
   (얼마나 오래 공부했어?)

B: Sinds ik vorig jaar naar Nederland ben verhuisd,
   studeer ik elke dag.
   (작년에 네덜란드로 이사한 이후로 매일 공부해.)

A: Vind je het moeilijk?
   (어려워?)

B: Ja, vooral als mensen snel praten.
   Maar ik merk dat ik steeds meer versta,
   naarmate ik meer oefen.
   (응, 특히 사람들이 빨리 말할 때.
   하지만 연습할수록 점점 더 많이 알아듣는 것 같아.)

A: Dat is goed! Als je wilt, kunnen we samen oefenen.
   (잘됐다! 원하면 같이 연습할 수 있어.)

B: Dat zou ik leuk vinden, mits we in het Nederlands praten
   en niet in het Engels!
   (좋지, 영어가 아닌 네덜란드어로 말하는 조건으로!)
```
