# 3-2. 현재 완료 (Voltooid tegenwoordige tijd)

### 구조: **hebben/zijn + 과거분사 (voltooid deelwoord)**

```
Ik heb gewerkt.        (나는 일했다.)
Zij is gekomen.        (그녀는 왔다.)
Wij hebben gegeten.    (우리는 먹었다.)
Hij is gevallen.       (그는 넘어졌다.)
```

> **중요**: 네덜란드어 일상 대화에서 과거를 표현할 때 가장 많이 쓰는 시제입니다. 한국어의 "~했다"에 해당하는 표현 대부분이 VTT로 표현됩니다.

### 과거분사 만들기

#### 규칙 동사

**ge- + 어간 + -t/-d**

- 어간이 't kofschip' 자음으로 끝나면 → **-t**
- 그 외 → **-d**

| 원형 | 어간 | 과거분사 | 't kofschip? |
|------|------|---------|-------------|
| werken (일하다) | werk | ge**werk**t | k → -t |
| fietsen (자전거 타다) | fiets | ge**fiets**t | s → -t |
| wonen (살다) | woon | ge**woon**d | n → -d |
| leren (배우다) | leer | ge**leer**d | r → -d |
| studeren (공부하다) | studeer | ge**studeer**d | r → -d |
| koken (요리하다) | kook | ge**kook**t | k → -t |
| praten (이야기하다) | praat | ge**praat** | t → 이미 t로 끝남 |
| maken (만들다) | maak | ge**maak**t | k → -t |
| hopen (바라다) | hoop | ge**hoop**t | p → -t |
| lachen (웃다) | lach | ge**lach**t | ch → -t |
| spelen (놀다) | speel | ge**speel**d | l → -d |
| bouwen (짓다) | bouw | ge**bouw**d | w → -d |
| horen (듣다) | hoor | ge**hoor**d | r → -d |
| tekenen (그리다) | teken | ge**tekend** | n → -d |
| antwoorden (대답하다) | antwoord | ge**antwoord** | d → 이미 d로 끝남 |

> **주의**: 어간이 이미 **-t**로 끝나는 동사는 -t를 하나만 씁니다 (praten → gepraat, 아님 ~~gepraatt~~). 마찬가지로 어간이 **-d**로 끝나면 -d를 하나만 씁니다 (antwoorden → geantwoord).

**v/z 변환과 과거분사:**

| 원형 | 어간 변환 | 과거분사 | 설명 |
|------|----------|---------|------|
| leven (살다) | leef (v→f) | ge**leefd** | 원래 v → -d |
| geloven (믿다) | geloof (v→f) | ge**loofd** | be/ge/ver 접두사 → ge- 없음! |
| verhuizen (이사하다) | verhuis (z→s) | ver**huisd** | ver- 접두사 → ge- 없음! |
| reizen (여행하다) | reis (z→s) | ge**reisd** | 원래 z → -d |

#### ge- 를 붙이지 않는 경우

**비분리 접두사** 로 시작하는 동사에는 ge-를 붙이지 않습니다:

| 접두사 | 원형 | 과거분사 | 뜻 |
|--------|------|---------|-----|
| **be-** | betalen | **betaald** | 지불하다 |
| **be-** | bezoeken | **bezocht** | 방문하다 |
| **be-** | beginnen | **begonnen** | 시작하다 |
| **be-** | beloven | **beloofd** | 약속하다 |
| **er-** | erkennen | **erkend** | 인정하다 |
| **ge-** | gebruiken | **gebruikt** | 사용하다 |
| **ge-** | geloven | **geloofd** | 믿다 |
| **ge-** | gebeuren | **gebeurd** | 일어나다 |
| **her-** | herhalen | **herhaald** | 반복하다 |
| **her-** | herinneren | **herinnerd** | 기억하다 |
| **ont-** | ontmoeten | **ontmoet** | 만나다 |
| **ont-** | ontdekken | **ontdekt** | 발견하다 |
| **ont-** | ontvangen | **ontvangen** | 받다 |
| **ver-** | vertellen | **verteld** | 말해주다 |
| **ver-** | verliezen | **verloren** | 잃다 |
| **ver-** | verstaan | **verstaan** | 이해하다 |
| **ver-** | vergeten | **vergeten** | 잊다 |
| **ver-** | verkopen | **verkocht** | 팔다 |

> **한국어 화자가 자주 하는 실수**: "gebruiken"의 과거분사를 ~~gegebruikt~~로 쓰는 경우가 많습니다. ge-로 시작하는 동사이므로 ge-를 또 붙이지 않습니다 → **gebruikt**

**-eren 으로 끝나는 외래어 동사:**

| 원형 | 과거분사 | ge- 여부 | 뜻 |
|------|---------|---------|-----|
| telefoneren | ge**telefoneerd** | ge- 있음 | 전화하다 |
| studeren | ge**studeerd** | ge- 있음 | 공부하다 |
| reserveren | ge**reserveerd** | ge- 있음 | 예약하다 |
| repareren | ge**repareerd** | ge- 있음 | 수리하다 |
| parkeren | ge**parkeerd** | ge- 있음 | 주차하다 |
| informeren | ge**informeerd** | ge- 있음 | 알리다 |
| organiseren | ge**organiseerd** | ge- 있음 | 조직하다 |

> **주의**: -eren으로 끝나도 비분리 접두사가 있으면 ge-가 없습니다: proberen → ge**probeerd** (있음), maar: verklaren → **verklaard** (없음, ver- 접두사)

#### 분리동사의 과거분사

ge-가 분리 접두사와 어간 사이에 들어갑니다:

| 원형 | 과거분사 | 뜻 |
|------|---------|-----|
| opbellen | **op-ge-beld** | 전화하다 |
| aankomen | **aan-ge-komen** | 도착하다 |
| meenemen | **mee-ge-nomen** | 가져가다 |
| opruimen | **op-ge-ruimd** | 정리하다 |
| uitnodigen | **uit-ge-nodigd** | 초대하다 |
| thuiskomen | **thuis-ge-komen** | 귀가하다 |
| binnenkomen | **binnen-ge-komen** | 들어오다 |
| afwassen | **af-ge-wassen** | 설거지하다 |
| opstaan | **op-ge-staan** | 일어나다 |
| meebrengen | **mee-ge-bracht** | 가져오다 |
| uitzetten | **uit-ge-zet** | 끄다 |
| aantrekken | **aan-ge-trokken** | 입다(옷) |
| uitdoen | **uit-ge-daan** | 끄다/벗다 |

```
Ik heb mijn moeder opgebeld.           (나는 어머니께 전화했다.)
De trein is om drie uur aangekomen.    (기차가 3시에 도착했다.)
Heb je je jas aangetrokken?            (외투 입었어?)
Ik heb de keuken opgeruimd.            (나는 부엌을 정리했다.)
We hebben onze vrienden uitgenodigd.   (우리는 친구들을 초대했다.)
```

### hebben vs zijn 선택

대부분의 동사는 **hebben**을 씁니다. **zijn**을 쓰는 동사:

| 조건 | 예시 |
|------|------|
| 이동/방향 변화 동사 | gaan (가다), komen (오다), rijden (운전하다), lopen (걷다), vliegen (날다), vertrekken (출발하다), aankomen (도착하다), reizen (여행하다), verhuizen (이사하다), emigreren (이민하다) |
| 상태 변화 동사 | worden (되다), beginnen (시작하다), stoppen (멈추다), groeien (자라다), sterven (죽다), veranderen (변하다), slagen (합격하다), trouwen (결혼하다), schrikken (놀라다) |
| zijn, blijven, gebeuren | zijn (이다), blijven (머물다), gebeuren (일어나다), lukken (성공하다), mislukken (실패하다) |

**zijn을 사용하는 동사 전체 목록:**

| 동사 | 과거분사 | 예문 |
|------|---------|------|
| gaan | gegaan | Ik **ben** naar huis **gegaan**. (집에 갔다.) |
| komen | gekomen | Zij **is** laat **gekomen**. (그녀는 늦게 왔다.) |
| worden | geworden | Hij **is** arts **geworden**. (그는 의사가 되었다.) |
| blijven | gebleven | Wij **zijn** thuis **gebleven**. (우리는 집에 있었다.) |
| zijn | geweest | Ik **ben** in Amsterdam **geweest**. (암스테르담에 갔었다.) |
| vertrekken | vertrokken | De trein **is** al **vertrokken**. (기차가 이미 출발했다.) |
| aankomen | aangekomen | Het pakket **is** **aangekomen**. (소포가 도착했다.) |
| vallen | gevallen | Het kind **is** **gevallen**. (아이가 넘어졌다.) |
| sterven | gestorven | De plant **is** **gestorven**. (식물이 죽었다.) |
| groeien | gegroeid | De economie **is** **gegroeid**. (경제가 성장했다.) |
| rijden | gereden | Hij **is** naar Utrecht **gereden**. (그는 위트레흐트로 운전해 갔다.) |
| lopen | gelopen | Zij **is** naar school **gelopen**. (그녀는 학교까지 걸어갔다.) |
| vliegen | gevlogen | We **zijn** naar Korea **gevlogen**. (우리는 한국으로 비행했다.) |
| verhuizen | verhuisd | Ze **zijn** naar Leiden **verhuisd**. (그들은 레이던으로 이사했다.) |
| gebeuren | gebeurd | Wat **is** er **gebeurd**? (무슨 일이 있었어?) |
| slagen | geslaagd | Ik **ben** voor het examen **geslaagd**. (시험에 합격했다.) |
| stoppen | gestopt | De bus **is** **gestopt**. (버스가 멈췄다.) |

**hebben/zijn 둘 다 가능한 동사:**

일부 동사는 의미에 따라 hebben 또는 zijn을 선택합니다:

| 동사 | hebben (행위 자체) | zijn (이동/방향) |
|------|-------------------|----------------|
| rijden | Ik **heb** twee uur **gereden**. (2시간 운전했다.) | Ik **ben** naar huis **gereden**. (집으로 운전해 갔다.) |
| lopen | Ik **heb** een uur **gelopen**. (1시간 걸었다.) | Ik **ben** naar school **gelopen**. (학교까지 걸어갔다.) |
| fietsen | Ik **heb** lang **gefietst**. (오래 자전거 탔다.) | Ik **ben** naar het station **gefietst**. (역까지 자전거로 갔다.) |
| zwemmen | Ik **heb** een uur **gezwommen**. (1시간 수영했다.) | Ik **ben** naar de overkant **gezwommen**. (건너편까지 수영해 갔다.) |

> **핵심 원칙**: 목적지/방향이 있으면 **zijn**, 행위 자체에 초점이면 **hebben**

```
Ik heb een boek gelezen.              (나는 책을 읽었다.)  → hebben
Ik ben naar Amsterdam gegaan.          (나는 암스테르담에 갔다.) → zijn
Hij is dokter geworden.                (그는 의사가 되었다.)  → zijn
Zij heeft de hele dag gewerkt.         (그녀는 하루 종일 일했다.) → hebben
We zijn om acht uur vertrokken.        (우리는 8시에 출발했다.) → zijn
Het ongeluk is gisteren gebeurd.       (사고가 어제 일어났다.) → zijn
Ik heb je gisteren gebeld.            (나는 어제 너에게 전화했다.) → hebben
De kinderen hebben urenlang gespeeld.  (아이들이 몇 시간이나 놀았다.) → hebben
```

### 어순: 과거분사는 문장 끝으로

```
Ik   heb   gisteren   een boek   gelezen.
주어  조동사  부사       목적어     과거분사
나는  했다   어제        책을      읽었다.
```

**다양한 어순 예시:**

```
평서문:   Ik heb gisteren een cadeau gekocht.
          (나는 어제 선물을 샀다.)

의문문:   Heb jij gisteren een cadeau gekocht?
          (너 어제 선물 샀어?)

Wh-질문:  Wat heb jij gisteren gekocht?
          (너 어제 뭐 샀어?)

부정문:   Ik heb gisteren geen cadeau gekocht.
          (나는 어제 선물을 사지 않았다.)

종속절:   ... omdat ik gisteren een cadeau heb gekocht.
          (... 어제 선물을 샀기 때문에.)
          → 종속절에서는 조동사(heb)도 끝으로!
```

### 현재 완료의 일상 표현

자주 사용되는 현재 완료 표현을 익혀두세요:

```
Ben je ooit in Nederland geweest?     (네덜란드에 가본 적 있어?)
Ik heb dat al gedaan.                  (나는 그것을 이미 했다.)
Heb je al gegeten?                     (벌써 밥 먹었어?)
Ik ben net aangekomen.                 (방금 도착했어.)
We hebben lang gewacht.                (우리는 오래 기다렸다.)
Wat is er gebeurd?                     (무슨 일이 있었어?)
Ik heb het niet begrepen.              (이해하지 못했어.)
Ze is in slaap gevallen.               (그녀는 잠들었다.)
Heb je het gehoord?                    (그거 들었어?)
Ik ben mijn portemonnee verloren.      (지갑을 잃어버렸어.)
Hij heeft me een bericht gestuurd.     (그가 나에게 메시지를 보냈다.)
We zijn gisteren verhuisd.             (우리는 어제 이사했다.)
```

### 연습 대화: 현재 완료 사용하기

```
A: Hoi! Hoe was je weekend?               (안녕! 주말 어땠어?)
B: Heel leuk! Ik ben naar Utrecht geweest. (아주 좋았어! 위트레흐트에 갔었어.)
A: Oh, wat heb je daar gedaan?             (오, 거기서 뭐 했어?)
B: Ik heb een museum bezocht en           (박물관 방문하고)
   daarna heb ik lekker gegeten.          (그 다음에 맛있게 먹었어.)
A: Leuk! Ben je alleen gegaan?             (좋다! 혼자 갔어?)
B: Nee, ik ben met vrienden gegaan.        (아니, 친구들이랑 갔어.)
   We hebben ook een boottocht gemaakt.    (보트 투어도 했어.)
A: Dat klinkt geweldig! Ik heb het        (대단하다! 나는 주말 내내)
   hele weekend gestudeerd.                (공부했어.)
B: Oh, dat is jammer. Volgende keer       (오, 그건 아쉽다. 다음에는)
   ga je mee!                              (같이 가자!)
```
