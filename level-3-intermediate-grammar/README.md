# Level 3: 중급 문법

> 과거시제, 완료시제, 접속사와 복합 어순, 비교급/최상급을 배웁니다.

---

## 3-1. 과거시제 (Verleden tijd)

네덜란드어에는 두 가지 과거시제가 있습니다:
- **단순 과거 (Onvoltooid verleden tijd, OVT)** — 영어 simple past와 비슷
- **현재 완료 (Voltooid tegenwoordige tijd, VTT)** — 영어 present perfect와 비슷

> 일상 회화에서는 **현재 완료(VTT)**를 훨씬 더 많이 사용합니다. 단순 과거는 글쓰기, 이야기, 뉴스에서 주로 사용됩니다.

### 단순 과거: 규칙 동사

어간의 마지막 자음에 따라 **-te/-ten** 또는 **-de/-den**을 붙입니다.

**'t kofschip** 규칙: 어간이 **t, k, f, s, ch, p** 로 끝나면 → **-te/-ten**

그 외에는 → **-de/-den**

| 원형 | 어간 | 과거 단수 | 과거 복수 |
|------|------|----------|----------|
| werken (일하다) | werk | werk**te** | werk**ten** |
| fietsen (자전거 타다) | fiets | fiets**te** | fiets**ten** |
| leven (살다) | leef* | leef**de** | leef**den** |
| reizen (여행하다) | reis* | reis**de** | reis**den** |
| wonen (살다) | woon | woon**de** | woon**den** |
| leren (배우다) | leer | leer**de** | leer**den** |

> *v → f, z → s 로 변합니다 (어간에서)

| 인칭 | 단수 (werken) | 복수 (werken) |
|------|-------------|-------------|
| ik | werkte | - |
| jij/u | werkte | - |
| hij/zij/het | werkte | - |
| wij/jullie/zij | - | werkten |

### 단순 과거: 불규칙 동사 (강변화 동사)

불규칙 동사는 모음이 변합니다. 암기가 필요합니다.

| 원형 | 뜻 | 과거 단수 | 과거 복수 | 과거분사 |
|------|-----|----------|----------|---------|
| zijn | 이다 | was | waren | geweest |
| hebben | 가지다 | had | hadden | gehad |
| gaan | 가다 | ging | gingen | gegaan |
| komen | 오다 | kwam | kwamen | gekomen |
| zien | 보다 | zag | zagen | gezien |
| geven | 주다 | gaf | gaven | gegeven |
| nemen | 가지다/취하다 | nam | namen | genomen |
| spreken | 말하다 | sprak | spraken | gesproken |
| schrijven | 쓰다 | schreef | schreven | geschreven |
| lezen | 읽다 | las | lazen | gelezen |
| eten | 먹다 | at | aten | gegeten |
| drinken | 마시다 | dronk | dronken | gedronken |
| slapen | 자다 | sliep | sliepen | geslapen |
| rijden | (차를) 운전하다 | reed | reden | gereden |
| lopen | 걷다 | liep | liepen | gelopen |
| zitten | 앉다 | zat | zaten | gezeten |
| staan | 서다 | stond | stonden | gestaan |
| liggen | 눕다 | lag | lagen | gelegen |
| beginnen | 시작하다 | begon | begonnen | begonnen |
| vinden | 찾다/생각하다 | vond | vonden | gevonden |

| kopen | 사다 | kocht | kochten | gekocht |

| weten | 알다 | wist | wisten | geweten |

## 3-2. 현재 완료 (Voltooid tegenwoordige tijd)

### 구조: **hebben/zijn + 과거분사 (voltooid deelwoord)**

```
Ik heb gewerkt.        (나는 일했다.)
Zij is gekomen.        (그녀는 왔다.)
```

### 과거분사 만들기

#### 규칙 동사

**ge- + 어간 + -t/-d**

- 어간이 't kofschip' 자음으로 끝나면 → **-t**
- 그 외 → **-d**

| 원형 | 어간 | 과거분사 |
|------|------|---------|
| werken | werk | ge**werk**t |
| fietsen | fiets | ge**fiets**t |
| wonen | woon | ge**woon**d |
| leren | leer | ge**leer**d |
| studeren | studeer | ge**studeer**d |

#### ge- 를 붙이지 않는 경우

- **be-, er-, ge-, her-, ont-, ver-** 로 시작하는 동사: betalen → **betaald** (ge 없음)
- **-eren** 으로 끝나는 외래어 동사: telefoneren → **getelefoneerd** (ge 있음) / studeren → **gestudeerd**

#### 분리동사의 과거분사

ge-가 분리 접두사와 어간 사이에 들어갑니다:

- opbellen (전화하다) → **op-ge-beld**
- aankomen (도착하다) → **aan-ge-komen**
- meenemen (가져가다) → **mee-ge-nomen**

### hebben vs zijn 선택

대부분의 동사는 **hebben**을 씁니다. **zijn**을 쓰는 동사:

| 조건 | 예시 |
|------|------|
| 이동/방향 변화 동사 | gaan (가다), komen (오다), rijden (운전하다), lopen (걷다), vliegen (날다), vertrekken (출발하다), aankomen (도착하다) |
| 상태 변화 동사 | worden (되다), beginnen (시작하다), stoppen (멈추다), groeien (자라다), sterven (죽다) |
| zijn, blijven, gebeuren | zijn (이다), blijven (머물다), gebeuren (일어나다) |

```
Ik heb een boek gelezen.     (나는 책을 읽었다.)  → hebben
Ik ben naar Amsterdam gegaan. (나는 암스테르담에 갔다.) → zijn
Hij is dokter geworden.       (그는 의사가 되었다.)  → zijn
```

### 어순: 과거분사는 문장 끝으로

```
Ik   heb   gisteren   een boek   gelezen.
주어  조동사  부사       목적어     과거분사
나는  했다   어제        책을      읽었다.
```

## 3-3. 접속사와 어순 (Voegwoorden)

### 등위접속사 (Nevenschikkende voegwoorden) — 어순 변화 없음

| 접속사 | 뜻 | 예시 |
|--------|-----|------|
| **en** | 그리고 | Ik werk en hij studeert. |
| **maar** | 그러나 | Ik wil komen, maar ik heb geen tijd. |
| **of** | 또는 | Wil je koffie of thee? |
| **want** | 왜냐하면 | Ik blijf thuis, want ik ben ziek. |
| **dus** | 그래서 | Het regent, dus ik neem een paraplu. |

> 등위접속사 뒤에서는 어순이 평서문과 동일합니다: **주어 + 동사**

### 종속접속사 (Onderschikkende voegwoorden) — 동사가 끝으로!

| 접속사 | 뜻 | 예시 |
|--------|-----|------|
| **dat** | ~라고 (that) | Ik denk **dat** hij ziek **is**. |
| **omdat** | 왜냐하면 (because) | Ik blijf thuis **omdat** ik ziek **ben**. |
| **als** | 만약/~할 때 (if/when) | **Als** het regent, blijf ik thuis. |
| **wanneer** | ~할 때 (when) | Ik bel je **wanneer** ik thuis **ben**. |
| **toen** | ~했을 때 (when, 과거) | **Toen** ik jong **was**, woonde ik in Seoul. |
| **terwijl** | ~하는 동안 | Ik lees **terwijl** hij **kookt**. |
| **voordat** | ~하기 전에 | Ik eet **voordat** ik **vertrek**. |
| **nadat** | ~한 후에 | **Nadat** hij gegeten **had**, ging hij slapen. |
| **hoewel** | ~에도 불구하고 | **Hoewel** het regent, ga ik fietsen. |
| **of** | ~인지 (whether) | Ik weet niet **of** hij **komt**. |
| **zodat** | ~하기 위해 | Ik studeer hard **zodat** ik kan **slagen**. |

> **핵심**: 종속접속사 뒤에서는 활용된 동사가 **절의 끝**으로 갑니다!

```
주절:    Ik denk       (나는 생각한다)
종속절:  dat hij ziek is  (그가 아프다고)
         ↑ 접속사  ↑ 동사가 끝으로

전체: Ik denk dat hij ziek is.
```

### 종속절이 먼저 오는 경우

종속절이 문장 앞에 오면, 주절에서 **도치**가 일어납니다:

```
Omdat ik ziek ben,  blijf  ik  thuis.
종속절              동사   주어  나머지
왜냐하면 아프기 때문에, 있다  나는  집에.
```

## 3-4. 조동사 (Modale werkwoorden)

| 조동사 | 뜻 | 예시 |
|--------|-----|------|
| **kunnen** | ~할 수 있다 | Ik **kan** zwemmen. (나는 수영할 수 있다.) |
| **mogen** | ~해도 된다 | Je **mag** hier roken. (여기서 흡연해도 된다.) |
| **moeten** | ~해야 한다 | Ik **moet** werken. (나는 일해야 한다.) |
| **willen** | ~하고 싶다 | Ik **wil** eten. (나는 먹고 싶다.) |
| **zullen** | ~할 것이다 | Ik **zal** komen. (나는 올 것이다.) |
| **hoeven** | ~할 필요가 있다 (부정) | Je **hoeft** niet te komen. (올 필요 없다.) |

### 어순: 조동사 + ... + 원형동사 (끝)

```
Ik   kan   goed   Nederlands   spreken.
주어  조동사  부사   목적어       원형동사
나는  할 수 있다  잘  네덜란드어를  말하다.
```

## 3-5. 형용사 (Bijvoeglijke naamwoorden)

### 형용사 어미변화

| 조건 | 어미 | 예시 |
|------|------|------|
| **de 명사** 앞 | **-e** | de groot**e** man (큰 남자) |
| **het 명사 + 정관사** 앞 | **-e** | het groot**e** huis (그 큰 집) |
| **het 명사 + 부정관사** 앞 | **변화 없음** | een groot huis (큰 집 하나) |
| 복수 명사 앞 | **-e** | grote huizen (큰 집들) |

> **요약**: **het 명사 + een(부정관사/무관사)** 인 경우에만 어미를 안 붙입니다.

```
de grote tafel       (큰 테이블)     → de 명사 + -e
het grote huis       (그 큰 집)      → het 명사 + 정관사 + -e
een grote tafel      (큰 테이블 하나)  → de 명사 + -e
een groot huis       (큰 집 하나)     → het 명사 + 부정관사 → 변화 없음!
```

### 비교급과 최상급

| 원급 | 비교급 (-er) | 최상급 (-st) | 뜻 |
|------|------------|------------|-----|
| groot | grot**er** | groot**st** | 큰 |
| klein | klein**er** | klein**st** | 작은 |
| mooi | mooi**er** | mooi**st** | 아름다운 |
| duur | duur**der** | duur**st** | 비싼 |
| lang | lang**er** | lang**st** | 긴/키가 큰 |
| kort | kort**er** | kort**st** | 짧은 |
| oud | oud**er** | oud**st** | 오래된 |
| jong | jong**er** | jong**st** | 젊은 |

#### 불규칙 비교급

| 원급 | 비교급 | 최상급 | 뜻 |
|------|--------|--------|-----|
| goed | **beter** | **best** | 좋은 |
| veel | **meer** | **meest** | 많은 |
| weinig | **minder** | **minst** | 적은 |
| graag | **liever** | **het liefst** | 기꺼이 |

#### 비교 구문

```
A is groter dan B.          (A는 B보다 크다.)
A is even groot als B.      (A는 B만큼 크다.)
A is het grootst.           (A가 가장 크다.)
Dit is het mooiste huis.    (이것이 가장 아름다운 집이다.)
```

## 3-6. er의 다양한 용법

네덜란드어에서 **er**는 매우 다양하게 쓰입니다.

| 용법 | 뜻 | 예시 |
|------|-----|------|
| 장소 | 거기에 | Ik woon **er**. (나는 거기에 산다.) |
| 비인칭 주어 | (there) | **Er** zijn veel mensen. (많은 사람들이 있다.) |
| 전치사 + er | 그것에 대해 등 | Ik denk **eraan**. (나는 그것에 대해 생각한다.) |
| 수량 | (of it/them) | Ik heb **er** drie. (나는 그것 세 개를 가지고 있다.) |
