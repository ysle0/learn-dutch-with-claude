# 자주 쓰는 형용사 어휘

**색깔 형용사:**

| 네덜란드어 | 한국어 | -e 형태 | 예시 |
|-----------|--------|---------|------|
| rood | 빨간 | rode | een rode roos (빨간 장미) |
| blauw | 파란 | blauwe | de blauwe lucht (파란 하늘) |
| groen | 초록 | groene | het groene gras (초록 잔디) |
| geel | 노란 | gele | een gele banaan (노란 바나나) |
| wit | 하얀 | witte | een witte muur (하얀 벽) |
| zwart | 검은 | zwarte | de zwarte kat (검은 고양이) |
| bruin | 갈색 | bruine | bruine schoenen (갈색 구두) |
| grijs | 회색 | grijze | een grijze dag (흐린 날) |
| roze | 분홍 | roze | een roze jurk (분홍 드레스) |
| oranje | 주황 | oranje | de oranje vlag (주황 깃발) |
| paars | 보라 | paarse | een paarse bloem (보라 꽃) |
| lichtblauw | 하늘색 | lichtblauwe | een lichtblauwe hemel (하늘색 하늘) |
| donkergroen | 짙은 초록 | donkergroene | een donkergroene jas (짙은 초록 외투) |

> **참고**: roze와 oranje는 어미가 변하지 않습니다 (이미 -e로 끝남).

# 형용사 관련 표현

```
Hoe vind je het eten?                (음식 어때?)
Ik vind het heerlijk/lekker!         (아주 맛있어!)

Wat een mooi weer!                   (날씨가 정말 좋다!)
Wat een groot huis!                  (정말 큰 집이다!)
Wat een lieve hond!                  (정말 귀여운 강아지!)

Het is te duur voor mij.             (나한테 너무 비싸.)
Het is niet groot genoeg.            (충분히 크지 않다.)
Hij is ontzettend aardig.            (그는 엄청 친절하다.)
```
