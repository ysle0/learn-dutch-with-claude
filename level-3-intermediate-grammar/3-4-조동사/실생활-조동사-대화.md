# 실생활 조동사 대화

```
A: Kun je vanavond komen eten?          (오늘 저녁에 밥 먹으러 올 수 있어?)
B: Ik wil wel komen, maar ik moet       (가고 싶은데, 오늘 밤에
   vanavond werken.                      일해야 해.)
A: Moet dat echt? Kun je niet           (꼭 그래야 해? 다른 날에
   een andere dag werken?                일할 수는 없어?)
B: Nee, ik moet dit project vandaag     (아니, 이 프로젝트를 오늘
   afmaken.                              끝내야 해.)
A: Jammer. Zullen we dan morgen gaan?   (아쉽다. 그럼 내일 갈까?)
B: Ja, dat zou leuk zijn! Zal ik        (응, 좋겠다! 내가 뭔가
   iets meenemen?                        가져갈까?)
A: Nee, je hoeft niets mee te nemen.    (아니, 아무것도 가져올 필요 없어.)
   Ik zal alles klaarmaken.              (내가 다 준비할게.)
```
