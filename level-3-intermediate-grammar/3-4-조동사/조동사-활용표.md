# 조동사 활용표

#### kunnen (~할 수 있다: 능력, 가능성)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | kan | kon |
| jij | kunt/kan | kon |
| u | kunt/kan | kon |
| hij/zij/het | kan | kon |
| wij | kunnen | konden |
| jullie | kunnen | konden |
| zij | kunnen | konden |

```
Ik kan goed zwemmen.                     (나는 수영을 잘 할 수 있다.)
Kun je me helpen?                         (나 좀 도와줄 수 있어?)
Hij kan heel goed koken.                  (그는 요리를 아주 잘 한다.)
Zij konden niet komen vanwege het weer.   (날씨 때문에 올 수 없었다.)
Kun je een beetje Nederlands spreken?     (네덜란드어 조금 할 수 있어?)
Dat kan niet waar zijn!                   (그게 사실일 리 없어!)
Ik kon gisteren niet slapen.              (어제 잠을 잘 수 없었다.)
Je kunt hier parkeren.                    (여기 주차할 수 있어요.)
```

> **뉘앙스**: kunnen은 능력("~할 줄 알다")과 가능성("~할 수 있다") 두 가지 의미가 있습니다.
> - 능력: Ik kan pianospelen. (피아노를 칠 줄 안다.)
> - 가능성: Het kan morgen regenen. (내일 비가 올 수 있다.)

#### mogen (~해도 된다: 허가, 허락)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | mag | mocht |
| jij | mag | mocht |
| u | mag | mocht |
| hij/zij/het | mag | mocht |
| wij | mogen | mochten |
| jullie | mogen | mochten |
| zij | mogen | mochten |

```
Mag ik hier zitten?                       (여기 앉아도 될까요?)
Je mag hier niet roken.                   (여기서 흡연하면 안 됩니다.)
Mag ik iets vragen?                       (뭐 좀 물어봐도 될까요?)
De kinderen mogen niet laat opblijven.    (아이들은 늦게까지 안 자면 안 된다.)
Mocht ik van mijn ouders naar het feest?  (부모님이 파티에 가도 된다고 했어?)
U mag hier niet parkeren.                 (여기 주차하시면 안 됩니다.)
Mag ik even langs?                        (잠깐 지나가도 될까요?)
Je mag best trots zijn op jezelf.         (스스로 자랑스러워해도 돼.)
```

> **mogen vs kunnen**: mogen은 **허가** (규칙상/사회적으로 괜찮은지), kunnen은 **능력/가능성** (물리적으로/능력상 가능한지)
> - Mag ik hier zwemmen? (여기서 수영해도 돼요?) — 허가를 묻는 것
> - Kan ik hier zwemmen? (여기서 수영할 수 있어요?) — 물리적으로 가능한지

#### moeten (~해야 한다: 의무, 필요)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | moet | moest |
| jij | moet | moest |
| u | moet | moest |
| hij/zij/het | moet | moest |
| wij | moeten | moesten |
| jullie | moeten | moesten |
| zij | moeten | moesten |

```
Ik moet morgen vroeg opstaan.             (나는 내일 일찍 일어나야 한다.)
Je moet je huiswerk maken.                (숙제를 해야 해.)
We moeten nu vertrekken.                  (지금 출발해야 해.)
Hij moest gisteren overwerken.            (그는 어제 야근해야 했다.)
Moet ik een afspraak maken?               (예약을 해야 하나요?)
Je moet niet zo veel suiker eten.         (설탕을 그렇게 많이 먹으면 안 돼.)
Zij moest naar de dokter.                 (그녀는 병원에 가야 했다.)
Moet je vanavond werken?                  (오늘 저녁에 일해야 해?)
U moet rechtsaf slaan bij het stoplicht.  (신호등에서 우회전하셔야 합니다.)
```

> **moeten의 다양한 강도**:
> - 강한 의무: Je **moet** je belasting betalen. (세금을 내야 한다.) — 법적 의무
> - 강한 권고: Je **moet** dit boek lezen! (이 책 꼭 읽어봐!) — 강력 추천
> - 추측: Het **moet** koud zijn buiten. (밖이 추울 것이다.) — ~임에 틀림없다

#### willen (~하고 싶다: 의지, 소망)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | wil | wilde/wou |
| jij | wilt/wil | wilde/wou |
| u | wilt | wilde/wou |
| hij/zij/het | wil | wilde/wou |
| wij | willen | wilden/wouden |
| jullie | willen | wilden/wouden |
| zij | willen | wilden/wouden |

```
Ik wil een kopje koffie.                  (커피 한 잔 마시고 싶어.)
Wil je iets drinken?                      (뭐 마실래?)
Zij wil arts worden.                      (그녀는 의사가 되고 싶다.)
We willen graag bestellen.                (주문하고 싶습니다.)
Ik wilde/wou je iets vragen.              (너한테 뭐 물어보려고 했어.)
Wilt u mij volgen?                        (저를 따라오시겠습니까?)
De kinderen willen buiten spelen.         (아이들이 밖에서 놀고 싶어한다.)
Ik wilde eigenlijk naar huis gaan.        (사실 집에 가고 싶었어.)
```

> **정중한 표현**: "Ik wil"은 직접적입니다. 더 정중하게 말하려면:
> - Ik **zou graag** een koffie **willen**. (커피 한 잔 부탁드립니다.) — 가장 정중
> - Ik **wil graag** een koffie. (커피 주세요.) — 보통
> - Ik **wil** een koffie. (커피요.) — 직접적/무례할 수 있음

#### zullen (~할 것이다: 미래, 제안)

| 인칭 | 현재형 | 과거형 (zou) |
|------|--------|-------------|
| ik | zal | zou |
| jij | zult/zal | zou |
| u | zult/zal | zou |
| hij/zij/het | zal | zou |
| wij | zullen | zouden |
| jullie | zullen | zouden |
| zij | zullen | zouden |

```
Ik zal je morgen bellen.                  (내일 전화할게.)
Zal ik je helpen?                         (내가 도와줄까?)
Het zal morgen regenen.                   (내일 비가 올 것이다.)
Zullen we naar de bioscoop gaan?          (영화 보러 갈까?)
Ik zou graag een tafel reserveren.        (테이블을 예약하고 싶습니다.)
Zou je me kunnen helpen?                  (나를 도와줄 수 있을까요?)
We zouden morgen vertrekken.              (우리는 내일 출발할 예정이었다.)
Dat zou leuk zijn!                        (그거 좋겠다!)
Hij zou om drie uur komen.                (그는 3시에 올 예정이었다.)
```

> **zullen의 용법 정리**:
> - 미래: Ik zal morgen komen. (내일 올게.)
> - 제안: Zal ik koffie zetten? (내가 커피 탈까?)
> - 약속: Ik zal het niet vergeten. (잊지 않을게.)
> - **zou** (과거/조건법): Zou je me willen helpen? (도와줄 수 있을까요?) — 정중한 요청
> - **zou** (비현실): Als ik rijk was, zou ik een huis kopen. (부자라면 집을 살 텐데.)

#### hoeven (~할 필요가 있다: 항상 부정/의문과 함께)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | hoef | hoefde |
| jij | hoeft | hoefde |
| u | hoeft | hoefde |
| hij/zij/het | hoeft | hoefde |
| wij | hoeven | hoefden |
| jullie | hoeven | hoefden |
| zij | hoeven | hoefden |

> **중요**: hoeven은 거의 항상 **niet** 또는 의문문과 함께 사용됩니다. 그리고 뒤에 오는 동사 앞에 **te**가 필요합니다!

```
Je hoeft niet te komen.                   (올 필요 없어.)
Je hoeft je geen zorgen te maken.         (걱정할 필요 없어.)
Hij hoefde niet te werken.                (그는 일할 필요가 없었다.)
Hoef ik dit formulier in te vullen?       (이 양식을 작성해야 하나요?)
Je hoeft alleen maar te bellen.           (전화만 하면 돼.)
We hoeven vandaag niet te koken.          (오늘 요리할 필요 없어.)
Dat hoeft niet!                           (그럴 필요 없어요!)
Je hoeft niet zo hard te schreeuwen.      (그렇게 크게 소리 지를 필요 없어.)
```

> **moeten niet vs niet hoeven te**:
> - Je **moet** niet liegen. (거짓말하면 **안 된다**.) — 금지
> - Je **hoeft** niet te komen. (올 **필요 없다**.) — 의무 없음
>
> 한국어 화자가 자주 혼동합니다! "~할 필요 없다"는 **hoeven niet te**이고, "~하면 안 된다"는 **niet mogen** 또는 **moet niet**입니다.
