# 조동사 조합

두 개의 조동사를 함께 사용할 수도 있습니다:

```
Ik moet kunnen zwemmen.                (수영할 줄 알아야 한다.)
      moeten + kunnen + zwemmen

Je zou moeten studeren.                (공부해야 할 텐데.)
      zou + moeten + studeren

Ik wil kunnen autorijden.              (운전할 줄 알고 싶다.)
      willen + kunnen + autorijden

Hij zal wel moeten werken.             (그는 아마 일해야 할 것이다.)
      zullen + moeten + werken
```
