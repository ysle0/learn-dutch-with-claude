# 연습 문제

1. 다음 동사를 과거형(OVT)으로 바꾸세요:
   - ik werk → ik ___
   - hij gaat → hij ___
   - wij komen → wij ___
   - zij leest → zij ___

2. 현재 완료형으로 문장을 만드세요:
   - Ik ___ een boek ___ (lezen).
   - Zij ___ naar Amsterdam ___ (gaan).
   - Wij ___ Nederlands ___ (leren).

3. 접속사를 사용해 두 문장을 연결하세요:
   - Ik blijf thuis. Ik ben ziek. (omdat)
   - Hij komt morgen. Ik denk het. (dat)

4. 형용사에 알맞은 어미를 붙이세요:
   - een mooi___ dag (de dag)
   - het klein___ kind (het kind)
   - een groot___ huis (het huis)
