# 3-1. 과거시제 (Verleden tijd)

네덜란드어에는 두 가지 과거시제가 있습니다:
- **단순 과거 (Onvoltooid verleden tijd, OVT)** — 영어 simple past와 비슷
- **현재 완료 (Voltooid tegenwoordige tijd, VTT)** — 영어 present perfect와 비슷

> 일상 회화에서는 **현재 완료(VTT)**를 훨씬 더 많이 사용합니다. 단순 과거는 글쓰기, 이야기, 뉴스에서 주로 사용됩니다.
