# OVT vs VTT: 언제 어떤 것을 사용할까?

| 상황 | 사용하는 시제 | 예시 |
|------|-------------|------|
| 일상 대화에서 과거를 말할 때 | **VTT (현재 완료)** | Ik heb gisteren een film gezien. |
| 소설, 이야기, 동화 | **OVT (단순 과거)** | Er was eens een prinses... (옛날에 공주가 있었다...) |
| 뉴스 보도 | **OVT** | De minister sprak over het nieuwe beleid. |
| zijn, hebben, modal verbs | **OVT 선호** | Ik was ziek. / Ik had geen tijd. / Ik kon niet komen. |
| 편지, 이메일 (격식체) | **OVT** | Ik schreef u om te informeren dat... |
| 구어체 과거 경험 | **VTT** | Ben je ooit in Japan geweest? (일본에 가본 적 있어?) |
| 감정/상태 표현 (과거) | **OVT 선호** | Ik wist het niet. / Ik dacht dat... |

> **한국어 화자 주의**: 한국어에는 단순 과거와 현재 완료의 구분이 영어/네덜란드어만큼 뚜렷하지 않습니다. 네덜란드어에서는 **일상 대화에서 거의 항상 VTT를 사용**한다는 것을 기억하세요. "어제 뭐 했어?"는 "Wat heb je gisteren gedaan?" (VTT)이 자연스럽습니다.
