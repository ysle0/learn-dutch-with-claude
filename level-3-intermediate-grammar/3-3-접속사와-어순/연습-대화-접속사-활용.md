# 연습 대화: 접속사 활용

```
A: Waarom leer je Nederlands?
   (왜 네덜란드어를 배워?)

B: Ik leer Nederlands omdat ik in Amsterdam wil werken.
   (암스테르담에서 일하고 싶기 때문에 네덜란드어를 배워.)

A: Hoe lang studeer je al?
   (얼마나 오래 공부했어?)

B: Sinds ik vorig jaar naar Nederland ben verhuisd,
   studeer ik elke dag.
   (작년에 네덜란드로 이사한 이후로 매일 공부해.)

A: Vind je het moeilijk?
   (어려워?)

B: Ja, vooral als mensen snel praten.
   Maar ik merk dat ik steeds meer versta,
   naarmate ik meer oefen.
   (응, 특히 사람들이 빨리 말할 때.
   하지만 연습할수록 점점 더 많이 알아듣는 것 같아.)

A: Dat is goed! Als je wilt, kunnen we samen oefenen.
   (잘됐다! 원하면 같이 연습할 수 있어.)

B: Dat zou ik leuk vinden, mits we in het Nederlands praten
   en niet in het Engels!
   (좋지, 영어가 아닌 네덜란드어로 말하는 조건으로!)
```
