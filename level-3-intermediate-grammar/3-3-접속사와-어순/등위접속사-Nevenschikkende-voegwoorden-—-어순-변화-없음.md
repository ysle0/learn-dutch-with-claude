# 등위접속사 (Nevenschikkende voegwoorden) — 어순 변화 없음

등위접속사는 두 개의 독립적인 절을 연결합니다. 어순에 영향을 주지 않습니다.

| 접속사 | 뜻 | 예시 |
|--------|-----|------|
| **en** | 그리고 | Ik werk en hij studeert. |
| **maar** | 그러나 | Ik wil komen, maar ik heb geen tijd. |
| **of** | 또는 | Wil je koffie of thee? |
| **want** | 왜냐하면 | Ik blijf thuis, want ik ben ziek. |
| **dus** | 그래서 | Het regent, dus ik neem een paraplu. |

> 등위접속사 뒤에서는 어순이 평서문과 동일합니다: **주어 + 동사**

**등위접속사 추가 예문:**

```
en:   Ik kook en mijn man wast af.
      (나는 요리하고 남편은 설거지한다.)

en:   Zij studeert Nederlands en ze werkt bij een bedrijf.
      (그녀는 네덜란드어를 공부하고 회사에서 일한다.)

maar: Het eten was duur, maar het was heel lekker.
      (음식이 비쌌지만, 아주 맛있었다.)

maar: Ik spreek een beetje Nederlands, maar ik maak nog veel fouten.
      (네덜란드어를 조금 하지만, 아직 실수를 많이 한다.)

of:   Gaan we naar het strand of blijven we thuis?
      (해변에 갈까 아니면 집에 있을까?)

of:   Je kunt een broodje of een salade nemen.
      (샌드위치나 샐러드를 먹을 수 있어.)

want: Ik ga vroeg slapen, want ik ben heel moe.
      (일찍 자러 갈게, 왜냐면 아주 피곤하거든.)

want: We nemen de auto, want het is te ver om te fietsen.
      (차를 가져갈게, 자전거 타기엔 너무 멀거든.)

dus:  Ik heb geen geld meer, dus ik kan niet mee-eten.
      (돈이 더 이상 없어서 같이 식사할 수 없어.)

dus:  Het was mooi weer, dus we zijn naar het park gegaan.
      (날씨가 좋았기 때문에 공원에 갔다.)
```

> **want vs omdat**: 둘 다 "왜냐하면"이지만, **want**는 등위접속사(어순 변화 없음)이고 **omdat**는 종속접속사(동사가 끝으로)입니다.

```
Ik blijf thuis, want ik ben ziek.     (want: 주어+동사 순서 유지)
Ik blijf thuis omdat ik ziek ben.     (omdat: 동사가 절의 끝으로)
```
