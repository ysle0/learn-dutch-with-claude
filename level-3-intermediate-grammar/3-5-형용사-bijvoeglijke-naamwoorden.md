# 3-5. 형용사 (Bijvoeglijke naamwoorden)

### 형용사 어미변화

네덜란드어 형용사는 명사 앞에서 어미가 변할 수 있습니다. 이 규칙은 네덜란드어에서 가장 많이 연습해야 하는 부분 중 하나입니다.

| 조건 | 어미 | 예시 |
|------|------|------|
| **de 명사** 앞 | **-e** | de groot**e** man (큰 남자) |
| **het 명사 + 정관사** 앞 | **-e** | het groot**e** huis (그 큰 집) |
| **het 명사 + 부정관사** 앞 | **변화 없음** | een groot huis (큰 집 하나) |
| 복수 명사 앞 | **-e** | grote huizen (큰 집들) |

> **요약**: **het 명사 + een(부정관사/무관사)** 인 경우에만 어미를 안 붙입니다.

```
de grote tafel       (큰 테이블)     → de 명사 + -e
het grote huis       (그 큰 집)      → het 명사 + 정관사 + -e
een grote tafel      (큰 테이블 하나)  → de 명사 + -e
een groot huis       (큰 집 하나)     → het 명사 + 부정관사 → 변화 없음!
```

**더 많은 예시로 확인하기:**

| de 명사 | 정관사 + -e | 부정관사 + -e | 무관사 + -e |
|---------|-----------|------------|-----------|
| de man | de oude man | een oude man | oude mannen |
| de vrouw | de mooie vrouw | een mooie vrouw | mooie vrouwen |
| de auto | de rode auto | een rode auto | rode auto's |
| de kat | de dikke kat | een dikke kat | dikke katten |

| het 명사 | 정관사 + -e | 부정관사 + 변화없음! | 무관사 + 변화없음! |
|---------|-----------|-------------------|-----------------|
| het huis | het grote huis | een groot huis | groot huis |
| het kind | het kleine kind | een klein kind | klein kind |
| het boek | het interessante boek | een interessant boek | interessant boek |
| het meisje | het leuke meisje | een leuk meisje | leuk meisje |

> **한국어 화자 주의**: 한국어에는 관사가 없으므로 이 규칙이 어렵게 느껴질 수 있습니다. 핵심은 **het 명사 + 부정관사/무관사**일 때만 예외라는 것입니다. 그 외에는 항상 -e를 붙이세요!

**-e를 붙일 때 철자 변화:**

| 원형 | -e 형태 | 규칙 |
|------|---------|------|
| groot | grot**e** | 모음 하나 + 자음 하나 → 자음 하나 탈락 (oo → o) |
| wit | witt**e** | 짧은 모음 + 자음 → 자음 두 배 (t → tt) |
| lief | liev**e** | f → v |
| doof | dov**e** | f → v |
| grijs | grijz**e** | s → z |
| dik | dikk**e** | 짧은 모음 + k → kk |
| nat | natt**e** | 짧은 모음 + t → tt |
| dun | dunn**e** | 짧은 모음 + n → nn |
| bloot | blot**e** | oo → o |
| boos | boz**e** | oo → o, s → z |

```
een grote man (큰 남자)          → groot → grote (oo → o)
een witte kat (하얀 고양이)       → wit → witte (t → tt)
een lieve vrouw (다정한 여자)     → lief → lieve (f → v)
een grijze auto (회색 차)        → grijs → grijze (s → z)
een dikke hond (뚱뚱한 개)       → dik → dikke (k → kk)
een natte jas (젖은 외투)        → nat → natte (t → tt)
```

**-e를 절대 붙이지 않는 형용사들:**

일부 형용사는 -e를 붙이지 않습니다:

| 형용사 | 뜻 | 예시 |
|--------|-----|------|
| open | 열린 | een open deur (열린 문) |
| plastic | 플라스틱의 | een plastic tas (플라스틱 가방) |
| aluminium | 알루미늄의 | een aluminium pan (알루미늄 냄비) |
| tevreden | 만족한 | een tevreden klant (만족한 고객) |
| eigen | 자신의 | mijn eigen huis (나만의 집) |

### 비교급과 최상급

#### 규칙 비교급/최상급

| 원급 | 비교급 (-er) | 최상급 (-st) | 뜻 |
|------|------------|------------|-----|
| groot | grot**er** | groot**st** | 큰 |
| klein | klein**er** | klein**st** | 작은 |
| mooi | mooi**er** | mooi**st** | 아름다운 |
| duur | duur**der** | duur**st** | 비싼 |
| lang | lang**er** | lang**st** | 긴/키가 큰 |
| kort | kort**er** | kort**st** | 짧은 |
| oud | oud**er** | oud**st** | 오래된 |
| jong | jong**er** | jong**st** | 젊은 |
| lief | liev**er** | lief**st** | 다정한 |
| snel | snell**er** | snel**st** | 빠른 |
| warm | warm**er** | warm**st** | 따뜻한 |
| koud | koud**er** | koud**st** | 차가운 |
| slim | slimm**er** | slim**st** | 똑똑한 |
| dik | dikk**er** | dik**st** | 두꺼운 |
| arm | arm**er** | arm**st** | 가난한 |
| rijk | rijk**er** | rijk**st** | 부유한 |
| breed | bred**er** | breed**st** | 넓은 |
| laat | lat**er** | laat**st** | 늦은 |

> **철자 규칙**: 비교급에서도 형용사 어미변화와 같은 철자 규칙이 적용됩니다:
> - groot → grot**er** (oo → o, 모음 앞이므로 열린 음절)
> - dik → dikk**er** (k → kk, 짧은 모음 보존)
> - lief → liev**er** (f → v, 모음 앞)

**-r/-st로 끝나는 형용사의 비교급/최상급:**

| 원급 | 비교급 | 최상급 | 뜻 |
|------|--------|--------|-----|
| duur | duur**der** | duur**st** | 비싼 |
| ver | ver**der** | ver**st** | 먼 |
| lekker | lekker**der** | lekker**st** | 맛있는 |
| mager | mager**der** | mager**st** | 마른 |

> -r로 끝나는 형용사는 비교급에 **-der**를 붙입니다.

#### 불규칙 비교급

| 원급 | 비교급 | 최상급 | 뜻 |
|------|--------|--------|-----|
| goed | **beter** | **best** | 좋은 |
| veel | **meer** | **meest** | 많은 |
| weinig | **minder** | **minst** | 적은 |
| graag | **liever** | **het liefst** | 기꺼이 |

```
Dit boek is beter dan dat boek.           (이 책이 저 책보다 낫다.)
Ik heb meer ervaring dan jij.             (나는 너보다 경험이 많다.)
Er zijn minder mensen dan gisteren.       (어제보다 사람이 적다.)
Ik drink liever thee dan koffie.          (커피보다 차를 더 좋아한다.)
Dit is het beste restaurant in de stad.   (이것이 도시 최고의 레스토랑이다.)
```

#### 비교 구문

```
비교급 + dan:
  A is groter dan B.              (A는 B보다 크다.)
  Amsterdam is groter dan Utrecht. (암스테르담이 위트레흐트보다 크다.)
  Deze film is beter dan die film. (이 영화가 저 영화보다 낫다.)
  Zij is slimmer dan haar broer.   (그녀는 오빠보다 똑똑하다.)

동등 비교 (even ... als):
  A is even groot als B.          (A는 B만큼 크다.)
  Hij is even oud als ik.          (그는 나와 나이가 같다.)
  Dit boek is even interessant als dat boek. (이 책은 저 책만큼 흥미롭다.)
  Nederlands is even moeilijk als Duits. (네덜란드어는 독일어만큼 어렵다.)

최상급:
  A is het grootst.               (A가 가장 크다.)
  Dit is het mooiste huis.        (이것이 가장 아름다운 집이다.)
  Zij is de slimste van de klas.   (그녀가 반에서 가장 똑똑하다.)
  Dat was de leukste vakantie ooit! (그것이 지금까지 가장 즐거운 휴가였다!)
  Amsterdam is de grootste stad van Nederland. (암스테르담은 네덜란드에서 가장 큰 도시다.)
```

**최상급의 두 가지 형태:**

```
서술적 (명사 없이):
  Hij is het slimst.              (그가 가장 똑똑하다.)
  → het + 최상급 (어미 없음)

한정적 (명사 앞에):
  Hij is de slimste student.      (그가 가장 똑똑한 학생이다.)
  → 최상급 + -e (항상)

  Dit is het mooiste uitzicht.    (이것이 가장 아름다운 경치이다.)
  → het 명사 앞이라도 최상급은 항상 -e!
```

> **주의**: 비교급과 최상급이 명사 앞에 올 때도 형용사 어미변화 규칙이 적용됩니다:
> - een groter**e** stad (더 큰 도시) — de 명사 + -e
> - een groter huis (더 큰 집) — het 명사 + een → 어미 없음
> - het grootst**e** huis (가장 큰 집) — 최상급은 항상 -e!

### 형용사 관련 표현

```
Hoe vind je het eten?                (음식 어때?)
Ik vind het heerlijk/lekker!         (아주 맛있어!)

Wat een mooi weer!                   (날씨가 정말 좋다!)
Wat een groot huis!                  (정말 큰 집이다!)
Wat een lieve hond!                  (정말 귀여운 강아지!)

Het is te duur voor mij.             (나한테 너무 비싸.)
Het is niet groot genoeg.            (충분히 크지 않다.)
Hij is ontzettend aardig.            (그는 엄청 친절하다.)
```
