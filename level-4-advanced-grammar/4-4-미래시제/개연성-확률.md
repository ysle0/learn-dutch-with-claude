# 개연성/확률 표현

zullen + wel은 추측(~일 것이다)을 나타냅니다:

```
확실:  Hij komt morgen.                  (그는 내일 온다.)
추측:  Hij zal morgen wel komen.         (그는 내일 아마 올 거야.)
약한:  Hij zou morgen kunnen komen.      (그는 내일 올 수도 있어.)
불확실: Misschien komt hij morgen.        (어쩌면 내일 올지도.)
```

# 미래완료: zullen + 과거분사 + hebben/zijn

미래의 특정 시점에 완료되어 있을 일을 표현합니다:

```
Tegen vrijdag zal ik het rapport hebben afgemaakt.
(금요일까지 보고서를 끝내 놓을 것이다.)

Over een jaar zullen we zijn verhuisd.
(1년 후면 이사를 마친 상태일 것이다.)

Voordat jij thuiskomt, zal ik al hebben gegeten.
(네가 집에 오기 전에 이미 먹어 놓을 것이다.)

Tegen die tijd zullen ze het probleem hebben opgelost.
(그때쯤이면 그들이 문제를 해결해 놓을 것이다.)
```

> 미래완료는 격식적 표현으로, 일상 대화에서는 잘 사용하지 않습니다.

# 세 가지 방법 비교

| 상황 | 추천 방법 | 예시 |
|------|----------|------|
| 확정된 계획/일정 | 현재시제 + 시간 부사 | De trein vertrekt om 3 uur. |
| 의도/계획 (비격식) | gaan + 원형 | Ik ga morgen studeren. |
| 약속/의지 | zullen | Ik zal het doen, beloofd! |
| 추측/예측 | zullen + wel | Het zal wel lukken. |
| 제안 | zullen (의문문) | Zullen we gaan? |
| 공식적/격식적 | zullen | De wet zal worden aangepast. |
