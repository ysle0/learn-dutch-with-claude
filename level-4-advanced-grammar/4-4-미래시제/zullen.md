# 방법 1: zullen + 원형동사 (의지/약속/추측/격식)

**zullen** 활용:

| 인칭 | 활용 |
|------|------|
| ik | zal |
| jij | zal / zult (격식) |
| hij/zij/het | zal |
| wij | zullen |
| jullie | zullen |
| zij (복수) | zullen |

**용법 1 — 약속/의지:**
```
Ik zal morgen komen.                 (나는 내일 올 것이다.)
Ik zal het nooit meer doen.          (다시는 안 할게.)
Wij zullen ons best doen.           (우리는 최선을 다할 것입니다.)
Ik zal je niet teleurstellen.        (실망시키지 않을게.)
```

**용법 2 — 추측/예측:**
```
Het zal wel goed komen.              (잘 될 거야.)
Hij zal nu wel thuis zijn.           (그는 지금 아마 집에 있을 거야.)
Dat zal niet makkelijk zijn.         (그건 쉽지 않을 거야.)
Het zal morgen wel regenen.          (내일 아마 비가 올 거야.)
Ze zal wel moe zijn na de reis.      (여행 후에 피곤할 거야.)
```

**용법 3 — 제안/요청:**
```
Zal ik je helpen?                    (도와줄까?)
Zal ik de deur opendoen?             (문 열어줄까?)
Zullen we naar de film gaan?         (영화 보러 갈까?)
Zullen we om 8 uur afspreken?        (8시에 만날까?)
Zal ik koffie voor je halen?         (커피 가져다줄까?)
```

**용법 4 — 격식적/공식적 문서:**
```
De vergadering zal om 10 uur beginnen.  (회의는 10시에 시작될 것입니다.)
De minister zal een toespraak houden.   (장관이 연설할 것입니다.)
Het project zal volgend jaar worden afgerond. (프로젝트는 내년에 완료될 것입니다.)
```
