# 4-4. 미래시제 (Toekomende tijd)

네덜란드어에서 미래를 표현하는 방법은 세 가지가 있으며, 각각 뉘앙스가 다릅니다. 한국어에서 "-ㄹ 것이다", "-ㄹ 거야", "-ㄹ게" 등의 차이와 비슷합니다. 영어의 will/be going to/현재진행형과도 대응됩니다.
