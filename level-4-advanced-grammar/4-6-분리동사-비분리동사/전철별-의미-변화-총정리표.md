# 전철별 의미 변화 총정리표

| 분류 | 전철 | 핵심 의미 | 예시 |
|------|------|----------|------|
| **분리** | aan- | 접근, 시작, 켜기 | aandoen (켜다), aankomen (도착하다) |
| | af- | 떨어짐, 완료, 제거 | afmaken (끝내다), afnemen (줄다) |
| | bij- | 추가, 곁에, 회복 | bijkomen (회복하다), bijdragen (기여하다) |
| | in- | 안으로, 삽입 | instappen (탑승), invullen (기입) |
| | mee- | 함께, 동반 | meenemen (가져가다), meedoen (참여) |
| | na- | 뒤따름, 모방, 확인 | nadenken (숙고), nadoen (흉내) |
| | op- | 위로, 시작, 소진 | opstaan (일어나다), opeten (다 먹다) |
| | open- | 열림 | opendoen (열다), opengaan (열리다) |
| | rond- | 주위에, 돌아다님 | rondlopen (돌아다니다), rondkijken (둘러보다) |
| | samen- | 함께, 합침 | samenwerken (협업), samenwonen (동거) |
| | tegen- | 반대, 대항 | tegenkomen (마주치다), tegenhouden (저지) |
| | terug- | 되돌아감 | terugkomen (돌아오다), teruggeven (돌려주다) |
| | thuis- | 집에서/집으로 | thuiskomen (귀가), thuiswerken (재택근무) |
| | toe- | 향함, 닫힘, 허가 | toestaan (허용), toevoegen (추가) |
| | uit- | 밖으로, 끄기, 완료 | uitgaan (외출), uitleggen (설명) |
| | vast- | 고정, 붙잡힘 | vasthouden (붙잡다), vastzitten (갇히다) |
| | verder- | 계속, 더 멀리 | verdergaan (계속하다), verderlezen (계속 읽다) |
| | voor- | 앞에, 미리, 시범 | voorstellen (소개), voorbereiden (준비) |
| | weg- | 떠남, 사라짐 | weggaan (떠나다), weggooien (버리다) |
| **비분리** | be- | 타동사화, 대상에 영향 | betalen (지불), begrijpen (이해) |
| | ver- | 변화, 강화, 잘못됨 | veranderen (바꾸다), vergeten (잊다) |
| | ont- | 제거, 시작, 벗어남 | ontdekken (발견), ontsnappen (탈출) |
| | her- | 반복, 복원 | herhalen (반복), herstellen (수리) |
| | ge- | 상태, 지속 | gebruiken (사용), gebeuren (일어나다) |
| | er- | 달성, 인정 | erkennen (인정), ervaren (경험) |
| | mis- | 잘못됨, 실패 | mislukken (실패), misbruiken (오용) |
| **양용** | door- | 분리: 계속/관통, 비분리: 완전히 수행 | doorgaan ↔ doorzoeken |
| | over- | 분리: 다시/넘어, 비분리: 전체에 걸침 | overstappen ↔ overleggen |
| | onder- | 분리: 아래로, 비분리: 경험/영향 | ondergaan ↔ onderzoeken |
| | om- | 분리: 넘어짐/회전, 비분리: 둘러쌈 | omvallen ↔ omschrijven |
| | voor- | 분리: 앞에/미리, 비분리: 예방 | voorstellen ↔ voorkomen |
