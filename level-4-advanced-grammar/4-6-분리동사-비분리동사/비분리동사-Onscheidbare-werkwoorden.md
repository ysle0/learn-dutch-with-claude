# 비분리동사 (Onscheidbare werkwoorden)

**be-, er-, ge-, her-, mis-, ont-, ver-** 접두사는 절대 분리되지 않으며, 과거분사에서도 **ge-를 붙이지 않습니다**.

> **핵심 규칙:** 비분리 전철은 강세가 없고(동사 어근에 강세), 과거분사에 ge-가 붙지 않습니다.
> ```
> betalen → betaald (✅)    (×gebetaald)
> vergeten → vergeten (✅)  (×gevergeten)
> ```

#### 비분리 전철 완전 목록

---

**1. be- (타동사화, 대상에 영향을 미침)**

> **의미 변화:** 가장 중요한 기능은 **자동사를 타동사로** 만드는 것입니다. 또한 동작이 대상 전체에 영향을 미친다는 뉘앙스를 줍니다.
> - 타동사화: 전치사가 필요했던 동사를 직접목적어를 취하는 타동사로 변환
>   ```
>   kijken naar → bekijken    (보다 → 살펴보다)
>   spreken over → bespreken  (말하다 → 논의하다)
>   stijgen op → bestijgen    (오르다 → 등반하다)
>   ```
> - 대상에 영향: 동작이 대상을 완전히 감싸거나 영향을 미침
>   ```
>   schrijven (쓰다) → beschrijven (묘사하다 — 대상을 글로 감쌈)
>   ```

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **be**talen | 지불하다 | Ik betaal de rekening. | betaald |
| **be**grijpen | 이해하다 | Ik begrijp het. | begrepen |
| **be**zoeken | 방문하다 | We bezoeken het museum. | bezocht |
| **be**schrijven | 묘사하다 | Hij beschrijft de situatie. | beschreven |
| **be**slissen | 결정하다 | Zij beslist morgen. | beslist |
| **be**danken | 감사하다 | Ik bedank je voor je hulp. | bedankt |
| **be**denken | 고안하다/생각해내다 | Wie heeft dit bedacht? | bedacht |
| **be**ginnen | 시작하다 | De les begint om 9 uur. | begonnen |
| **be**horen | ~에 속하다 | Dat behoort tot het verleden. | behoord |
| **be**loven | 약속하다 | Ik beloof het je. | beloofd |
| **be**oordelen | 평가하다 | De jury beoordeelt de kandidaten. | beoordeeld |
| **be**reiken | 도달하다 | We bereiken de top. | bereikt |
| **be**schermen | 보호하다 | De muur beschermt de stad. | beschermd |
| **be**staan | 존재하다 | Dat woord bestaat niet. | bestaan |
| **be**studeren | 연구하다 | Zij bestudeert de documenten. | bestudeerd |
| **be**sturen | 운전하다/관리하다 | Hij bestuurt het bedrijf. | bestuurd |
| **be**tekenen | 의미하다 | Wat betekent dit woord? | betekend |
| **be**invloeden | 영향을 미치다 | Het weer beïnvloedt mijn humeur. | beïnvloed |
| **be**handelen | 다루다/치료하다 | De arts behandelt de patiënt. | behandeld |
| **be**wijzen | 증명하다 | Kun je dat bewijzen? | bewezen |
| **be**kijken | 살펴보다 | Bekijk deze foto eens. | bekeken |
| **be**groeten | 인사하다 | Ik begroet mijn collega's. | begroet |

**2. ver- (변화, 강화, 잘못됨, 이동)**

> **의미 변화:** 매우 다양한 의미를 가진 전철입니다. 크게 네 가지 기능이 있습니다:
> - **변화:** 상태의 변화 (anderen → **ver**anderen = 바꾸다)
> - **강화:** 의미를 강하게 함 (groot → **ver**groten = 확대하다)
> - **잘못됨:** 실수·부정적 행위 (kijken → **ver**kijken = 잘못 보다)
>   ```
>   spreken → verspreken (zich) = 말실수하다
>   slapen → verslapen (zich) = 늦잠 자다
>   schrijven → verschrijven (zich) = 잘못 쓰다
>   ```
> - **이동·멀어짐:** 떠나거나 사라짐 (trekken → **ver**trekken = 출발하다)

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **ver**staan | 이해하다(듣기) | Ik versta je niet. | verstaan |
| **ver**tellen | 이야기하다 | Zij vertelt een verhaal. | verteld |
| **ver**geten | 잊다 | Ik vergeet het altijd. | vergeten |
| **ver**huizen | 이사하다 | We verhuizen volgende maand. | verhuisd |
| **ver**anderen | 바꾸다 | Het weer verandert snel. | veranderd |
| **ver**kopen | 팔다 | Hij verkoopt auto's. | verkocht |
| **ver**trekken | 출발하다 | De trein vertrekt om 5 uur. | vertrokken |
| **ver**bieden | 금지하다 | Roken is hier verboden. | verboden |
| **ver**binden | 연결하다 | Dit verbindt de twee steden. | verbonden |
| **ver**bergen | 숨기다 | Zij verbergt haar gevoelens. | verborgen |
| **ver**dienen | 벌다/~할 자격이 있다 | Hij verdient goed. | verdiend |
| **ver**dwijnen | 사라지다 | De zon verdwijnt achter de wolken. | verdwenen |
| **ver**groten | 확대하다 | We vergroten de foto. | vergroot |
| **ver**liezen | 잃다 | Ik verlies altijd mijn sleutels. | verloren |
| **ver**wachten | 기대하다 | Ik verwacht je om 8 uur. | verwacht |
| **ver**zamelen | 수집하다 | Hij verzamelt postzegels. | verzameld |
| **ver**zorgen | 돌보다 | Zij verzorgt haar moeder. | verzorgd |
| **ver**slapen (zich) | 늦잠 자다 | Ik heb me verslapen. | verslapen |
| **ver**spreken (zich) | 말실수하다 | Hij verspreekt zich vaak. | versproken |
| **ver**beteren | 개선하다 | Het weer verbetert. | verbeterd |
| **ver**klaren | 설명하다/선언하다 | De minister verklaart de situatie. | verklaard |
| **ver**lengen | 연장하다 | Ik verleng mijn abonnement. | verlengd |
| **ver**trouwen | 신뢰하다 | Ik vertrouw je. | vertrouwd |
| **ver**nietigen | 파괴하다 | De storm vernietigt alles. | vernietigd |
| **ver**spillen | 낭비하다 | Verspil geen water. | verspild |
| **ver**gissen (zich) | 실수하다 | Ik heb me vergist. | vergist |

**3. ont- (제거, 시작, 역행, 벗어남)**

> **의미 변화:** 주로 세 가지 기능이 있습니다:
> - **제거·부정:** 원래 상태를 없앰/빼앗음 (kleden → **ont**kleden = 옷을 벗기다)
> - **시작:** 무언가의 시작 (staan → **ont**staan = 발생하다)
> - **벗어남:** ~에서 벗어남 (snappen → **ont**snappen = 탈출하다)
>   ```
>   laden (싣다) → ontladen (방전하다/내리다)
>   doen (하다) → ontdoen (제거하다)
>   dekken (덮다) → ontdekken (발견하다 — 덮개를 벗기다)
>   ```

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **ont**moeten | 만나다 | Ik ontmoet haar morgen. | ontmoet |
| **ont**vangen | 받다 | We ontvangen gasten. | ontvangen |
| **ont**dekken | 발견하다 | Zij ontdekt nieuwe dingen. | ontdekt |
| **ont**snappen | 탈출하다 | De gevangene ontsnapt. | ontsnapt |
| **ont**houden | 기억하다 | Ik onthoud het. | onthouden |
| **ont**staan | 발생하다 | Er ontstaat een probleem. | ontstaan |
| **ont**breken | 부족하다 | Er ontbreekt één puzzelstuk. | ontbroken |
| **ont**kennen | 부인하다 | Hij ontkent alles. | ontkend |
| **ont**slaan | 해고하다 | Het bedrijf ontslaat werknemers. | ontslagen |
| **ont**spannen | 긴장을 풀다 | Ik ontspan me in het weekend. | ontspannen |
| **ont**wikkelen | 개발하다/발전하다 | We ontwikkelen een nieuwe app. | ontwikkeld |
| **ont**vluchten | 도망치다 | Zij ontvlucht het gevaar. | ontvlucht |
| **ont**ploffen | 폭발하다 | De bom ontploft. | ontploft |
| **ont**laden | 방전하다/짐을 내리다 | We ontladen de vrachtwagen. | ontladen |
| **ont**bijten | 아침을 먹다 | Wij ontbijten om 8 uur. | ontbeten |

---

**4. her- (반복, 다시, 복원)**

> **의미 변화:** 기본동사에 "다시·반복", "복원·수리" 의미를 부여합니다.
> - 반복: 다시 함 (halen → **her**halen = 반복하다)
> - 복원: 원래 상태로 되돌림 (stellen → **her**stellen = 수리하다/회복하다)

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **her**halen | 반복하다 | Ik herhaal de zin. | herhaald |
| **her**kennen | 알아보다 | Ik herken je! | herkend |
| **her**inneren | 기억나게 하다 | Dit herinnert me aan vroeger. | herinnerd |
| **her**stellen | 수리하다/회복하다 | Hij herstelt snel. | hersteld |
| **her**vormen | 개혁하다 | De regering hervormt het onderwijs. | hervormd |
| **her**zien | 수정하다/재검토하다 | We herzien het plan. | herzien |
| **her**openen | 재개하다 | Het restaurant heropent volgende week. | heropend |
| **her**gebruiken | 재사용하다 | Hergebruik plastic flessen. | hergebruikt |
| **her**bouwen | 재건하다 | Ze herbouwen de kerk. | herbouwd |
| **her**overwegen | 재고하다 | Ik heroverweeg mijn besluit. | heroverwogen |

---

**5. ge- (상태, 지속)**

> **의미 변화:** 비분리 전철로서의 ge-는 동사에 "상태", "지속적 행위" 의미를 부여합니다. 개수가 적고 독립적인 규칙보다는 개별 암기가 필요합니다.
>
> 주의: 과거분사의 ge- 접두사와 혼동하지 마세요. 여기서의 ge-는 동사 원형에 이미 포함된 비분리 전철입니다.

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **ge**bruiken | 사용하다 | Ik gebruik mijn telefoon. | gebruikt |
| **ge**beuren | 일어나다 | Wat gebeurt er? | gebeurd |
| **ge**loven | 믿다 | Ik geloof je. | geloofd |
| **ge**nieten | 즐기다 | Ik geniet van de zon. | genoten |
| **ge**lijken | 닮다 | Hij gelijkt op zijn vader. | geleken |
| **ge**hoorzamen | 복종하다 | De hond gehoorzaamt zijn baas. | gehoorzaamd |
| **ge**lukken | 성공하다 | Het is me gelukt! | gelukt |

---

**6. er- (달성, 상태변화)**

> **의미 변화:** 독일어에서 차용된 전철로, 네덜란드어에서는 소수의 동사에만 나타납니다. "달성", "경험", "인정" 등의 의미를 부여합니다.

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **er**kennen | 인정하다 | Hij erkent zijn fout. | erkend |
| **er**varen | 경험하다 | Ik ervaar dat als positief. | ervaren |

---

**7. mis- (잘못됨, 실패)**

> **의미 변화:** 기본동사에 "잘못됨", "실패", "오류" 의미를 부여합니다. 영어의 mis-와 거의 동일합니다.

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **mis**lukken | 실패하다 | Het plan mislukt. | mislukt |
| **mis**bruiken | 오용/학대하다 | Hij misbruikt zijn macht. | misbruikt |
| **mis**verstaan | 오해하다 | Je misverstaat me. | misverstaan |
| **mis**handelen | 학대하다 | Mishandel dieren niet. | mishandeld |
| **mis**gaan | 잘못되다 | Wat kan er misgaan? | misgegaan* |
| **mis**lopen | 놓치다 | Ik misloop de trein. | misgelopen* |

> *misgaan과 mislopen은 분리동사로 취급되는 경우도 있어 ge-가 들어갑니다. mis-는 비분리와 분리 사이의 경계에 있는 전철입니다.
