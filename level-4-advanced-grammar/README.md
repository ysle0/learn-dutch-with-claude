# Level 4: 고급 문법

> 수동태, 관계절, 가정법, 조건문, 과거완료, 미래시제를 배웁니다.

---

## 4-1. 수동태 (Lijdende vorm)

### 현재 수동태: **worden + 과거분사**

```
능동: De man leest het boek.        (남자가 책을 읽는다.)
수동: Het boek wordt gelezen.       (책이 읽힌다.)
```

| 인칭 | 활용 |
|------|------|
| ik | word + 과거분사 |
| jij | wordt + 과거분사 |
| hij/zij/het | wordt + 과거분사 |
| wij/jullie/zij | worden + 과거분사 |

### 과거 수동태: **werd(en) + 과거분사**

```
Het boek werd gelezen.              (책이 읽혔다.)
De boeken werden gelezen.           (책들이 읽혔다.)
```

### 완료 수동태: **zijn + 과거분사**

```
Het boek is gelezen.                (책이 읽혔다. — 현재완료 수동)
Het boek was gelezen.               (책이 읽혀 있었다. — 과거완료 수동)
```

> **worden vs zijn (수동태에서)**:
> - **worden**: 동작/과정에 초점 ("읽히고 있다")
> - **zijn**: 결과/상태에 초점 ("읽힌 상태이다")

### 행위자 표시: **door**

```
Het boek wordt door de student gelezen.
(그 책은 학생에 의해 읽힌다.)
```

## 4-2. 관계절 (Betrekkelijke bijzinnen)

### 관계대명사

| 선행사 | 관계대명사 | 예시 |
|--------|-----------|------|
| **de 명사** (주격/목적격) | **die** | De man **die** daar staat, is mijn vader. |
| **het 명사** (주격/목적격) | **dat** | Het boek **dat** ik lees, is interessant. |
| 복수 명사 | **die** | De kinderen **die** spelen, zijn blij. |
| 소유격 (누구의) | **wiens** (남성), **wier** (여성/복수) | De man **wiens** auto gestolen is... |
| 전치사 + 사물 | **waar + 전치사** | Het huis **waarin** ik woon... |
| 전치사 + 사람 | **전치사 + wie** | De man **met wie** ik werk... |

### 관계절의 어순

관계절은 종속절이므로 동사가 끝으로 갑니다:

```
De vrouw die   in Amsterdam   woont,   is mijn zus.
관계대명사      부사구         동사(끝)

암스테르담에 사는 여자는 내 여동생이다.
```

### waar + 전치사 (사물에 대해)

전치사 + 사물은 **waar-** 형태로 합칩니다:

| 전치사 구문 | 관계절 형태 | 예시 |
|------------|-----------|------|
| in het huis | **waar**...in / **waar**in | Het huis **waarin** ik woon. |
| op de tafel | **waar**...op / **waar**op | De tafel **waarop** het boek ligt. |
| over het onderwerp | **waar**...over / **waar**over | Het onderwerp **waarover** we praten. |
| aan het project | **waar**...aan / **waar**aan | Het project **waaraan** ik werk. |
| met het mes | **waar**...mee / **waar**mee | Het mes **waarmee** ik snij. |

> **met → mee**, **tot → toe** 로 형태가 변합니다.

## 4-3. 과거완료 (Voltooid verleden tijd)

### 구조: **had(den) + 과거분사** 또는 **was/waren + 과거분사**

과거의 특정 시점보다 더 이전에 일어난 일을 표현합니다.

```
Ik had al gegeten voordat hij kwam.
(그가 오기 전에 나는 이미 먹었다.)

Zij was al vertrokken toen ik aankwam.
(내가 도착했을 때 그녀는 이미 떠나 있었다.)
```

| hebben 동사 | zijn 동사 |
|------------|----------|
| ik **had** gewerkt | ik **was** gegaan |
| jij **had** gewerkt | jij **was** gegaan |
| hij **had** gewerkt | hij **was** gegaan |
| wij **hadden** gewerkt | wij **waren** gegaan |

## 4-4. 미래시제 (Toekomende tijd)

### 방법 1: **zullen + 원형동사** (의지/약속/추측)

```
Ik zal morgen komen.            (나는 내일 올 것이다.)
Het zal wel goed komen.         (잘 될 거야.)
Zal ik je helpen?               (도와줄까?)
```

### 방법 2: **gaan + 원형동사** (계획/의도)

```
Ik ga morgen werken.            (나는 내일 일할 거야.)
We gaan verhuizen.              (우리는 이사할 거야.)
```

### 방법 3: 현재시제 + 시간 부사 (가장 흔한 방법!)

```
Ik werk morgen.                 (나는 내일 일해.)
We vertrekken volgende week.    (우리는 다음 주에 떠나.)
```

> 네덜란드어에서는 미래의 일을 **현재시제 + 시간 부사**로 표현하는 것이 가장 자연스럽습니다.

## 4-5. 조건문 (Voorwaardelijke zinnen)

### 제1조건문: 현실적 조건 (als + 현재시제)

```
Als het regent, blijf ik thuis.
(비가 오면, 나는 집에 있을 거야.)

Als je hard werkt, slaag je voor het examen.
(열심히 공부하면, 시험에 합격할 거야.)
```

### 제2조건문: 비현실적 조건 (als + 과거시제)

```
Als ik rijk was, zou ik een groot huis kopen.
(내가 부자라면, 큰 집을 살 텐데.)

Als ik jou was, zou ik dat niet doen.
(내가 너라면, 그렇게 안 할 텐데.)
```

> **zou/zouden**: 영어의 would에 해당

| 인칭 | 활용 |
|------|------|
| ik | zou |
| jij | zou |
| hij/zij/het | zou |
| wij/jullie/zij | zouden |

### 제3조건문: 과거의 비현실 (als + 과거완료)

```
Als ik harder had gestudeerd, zou ik geslaagd zijn.
(더 열심히 공부했더라면, 합격했을 텐데.)

Als ik het had geweten, was ik gekomen.
(알았더라면, 왔을 텐데.)
```

## 4-6. 분리동사와 비분리동사 (Scheidbare en onscheidbare werkwoorden)

### 분리동사 (Scheidbare werkwoorden)

접두사가 강세를 받으며, 문장에서 분리됩니다.

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **op**bellen | 전화하다 | Ik bel je **op**. | **op**gebeld |
| **aan**komen | 도착하다 | De trein komt **aan**. | **aan**gekomen |
| **uit**gaan | 외출하다 | Ik ga vanavond **uit**. | **uit**gegaan |
| **mee**nemen | 가져가다 | Ik neem het boek **mee**. | **mee**genomen |
| **terug**komen | 돌아오다 | Hij komt morgen **terug**. | **terug**gekomen |
| **op**staan | 일어나다 | Ik sta vroeg **op**. | **op**gestaan |

> 분리동사가 종속절에서 사용되면 다시 붙습니다:
> - 주절: Ik bel je morgen **op**.
> - 종속절: ...dat ik je morgen **opbel**.

### 비분리동사 (Onscheidbare werkwoorden)

**be-, er-, ge-, her-, ont-, ver-** 접두사는 분리되지 않습니다.

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **be**talen | 지불하다 | Ik betaal. | betaald (ge- 없음) |
| **ver**staan | 이해하다 | Ik versta. | verstaan (ge- 없음) |
| **ont**moeten | 만나다 | Ik ontmoet. | ontmoet (ge- 없음) |
| **her**halen | 반복하다 | Ik herhaal. | herhaald (ge- 없음) |

## 4-7. 재귀동사 (Wederkerend werkwoord)

재귀대명사와 함께 쓰이는 동사입니다.

### 재귀대명사

| 인칭 | 재귀대명사 |
|------|-----------|
| ik | me / mij |
| jij | je |
| u | zich |
| hij/zij/het | zich |
| wij | ons |
| jullie | je |
| zij (복수) | zich |

### 자주 쓰이는 재귀동사

| 동사 | 뜻 | 예시 |
|------|-----|------|
| zich voelen | 느끼다 | Ik voel **me** goed. (기분이 좋다.) |
| zich wassen | 씻다 | Hij wast **zich**. (그는 씻는다.) |
| zich aankleden | 옷을 입다 | Zij kleedt **zich** aan. |
| zich vergissen | 실수하다 | Ik heb **me** vergist. |
| zich herinneren | 기억하다 | Ik herinner **me** dat. |
| zich voorstellen | 자기소개하다/상상하다 | Mag ik **me** voorstellen? |
| zich bevinden | 위치하다 | Het hotel bevindt **zich** in het centrum. |

## 4-8. 복합 어순 정리

네덜란드어 어순의 핵심을 정리합니다.

### 주절

```
위치1      V2(활용동사)  주어   중간부분          동사(끝)
Morgen     ga           ik     naar Amsterdam    fietsen.
내일       갈 것이다     나는   암스테르담으로      자전거 타러.
```

### 중간부분의 순서 (일반적 경향)

```
시간(Tijd) - 방법(Manier) - 장소(Plaats)

Ik ga    morgen    met de trein    naar Amsterdam.
         시간      방법             장소
```

### 종속절

```
접속사    주어   중간부분           동사(끝)
...dat    ik     morgen naar Amsterdam   ga.
          나는   내일 암스테르담으로       간다.
```

### 동사 여러 개일 때 (끝 동사의 순서)

```
주절: Ik heb gisteren een boek willen kopen.
      (나는 어제 책을 사고 싶었다.)

종속절: ...dat ik gisteren een boek heb willen kopen.
```

1. 수동태로 바꾸세요:
   - De leraar leest het boek. → Het boek ___
   - De studenten maken de oefeningen. → De oefeningen ___

2. 관계대명사를 채우세요:
   - De man ___ daar staat, is mijn vader. (die/dat)
   - Het boek ___ ik lees, is interessant. (die/dat)
   - De stad ___ ik woon, is groot. (waarin/waarop)

3. 조건문을 완성하세요:
   - Als ik veel geld ___ (hebben), ___ (zouden) ik een huis kopen.
   - Als het morgen ___ (regenen), blijf ik thuis.

   - Ik ___ morgen om 7 uur ___ (opstaan).
