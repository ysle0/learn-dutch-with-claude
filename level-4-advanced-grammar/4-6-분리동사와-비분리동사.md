# 4-6. 분리동사와 비분리동사 (Scheidbare en onscheidbare werkwoorden)

분리동사는 네덜란드어의 가장 독특한 특징 중 하나입니다. 한국어에는 없는 개념이며, 영어의 phrasal verbs (pick up, turn off)와 비슷하지만 규칙이 더 체계적입니다. 접두사에 강세가 있으면 분리동사, 동사 어근에 강세가 있으면 비분리동사입니다.

> **핵심 원리:** 전철(접두사, voorvoegsel)은 동사의 기본 의미를 방향, 상태, 강도 등으로 변형합니다. 같은 동사 어근이라도 어떤 전철이 붙느냐에 따라 전혀 다른 의미가 됩니다.
>
> ```
> staan (서다) → opstaan (일어나다) / verstaan (알아듣다) / bestaan (존재하다)
> komen (오다) → aankomen (도착하다) / bijkomen (의식을 되찾다) / overkomen (일어나다/발생하다)
> ```

---

### 분리동사 (Scheidbare werkwoorden)

접두사가 강세를 받으며, 주절에서 접두사가 문장 끝으로 분리됩니다. 과거분사는 접두사와 어근 사이에 **ge-**가 삽입됩니다 (예: **op**ge**staan**).

#### 분리 전철 완전 목록

---

**1. aan- (방향: ~에/~으로 / 시작, 연결, 켜기)**

> **의미 변화:** 기본동사에 "접근·접촉", "시작", "켜기/연결" 의미를 부여합니다.
> - 방향: 어떤 대상이나 장소에 다가감 (komen → **aan**komen = 도착하다)
> - 시작: 동작의 시작 (vallen → **aan**vallen = 공격하다, "덤벼들다")
> - 연결/켜기: 연결하거나 작동시킴 (doen → **aan**doen = 켜다)

| 원형 | 뜻 | 현재시제 예시 | 과거분사 |
|------|-----|-------------|---------|
| **aan**komen | 도착하다 | De trein komt om 3 uur **aan**. | **aan**gekomen |
| **aan**bieden | 제공하다 | Hij biedt zijn hulp **aan**. | **aan**geboden |
| **aan**doen | (불을) 켜다 | Doe het licht **aan**. | **aan**gedaan |
| **aan**nemen | 받아들이다/고용하다 | Het bedrijf neemt nieuwe mensen **aan**. | **aan**genomen |
| **aan**raken | 만지다 | Raak dat niet **aan**! | **aan**geraakt |
| **aan**trekken | (옷을) 입다 | Ik trek mijn jas **aan**. | **aan**getrokken |
| **aan**wijzen | 가리키다 | De leraar wijst het antwoord **aan**. | **aan**gewezen |
| **aan**kijken | 바라보다 | Kijk me **aan** als ik praat. | **aan**gekeken |
| **aan**geven | 건네다/신고하다 | Geef me het zout **aan**. | **aan**gegeven |
| **aan**passen | 맞추다/적응하다 | Ik pas me snel **aan**. | **aan**gepast |
| **aan**sluiten | 연결하다/가입하다 | Sluit de kabel **aan**. | **aan**gesloten |
| **aan**spreken | 말을 걸다 | Zij spreekt de buurman **aan**. | **aan**gesproken |
| **aan**steken | 불을 붙이다 | Hij steekt een kaars **aan**. | **aan**gestoken |
| **aan**vallen | 공격하다 | De hond valt de postbode **aan**. | **aan**gevallen |

**2. af- (방향: ~에서 떨어져 / 완료, 제거, 아래로)**

> **의미 변화:** 기본동사에 "분리·이탈", "완료·마무리", "제거" 의미를 부여합니다.
> - 떨어짐: 대상으로부터 떨어지거나 분리됨 (breken → **af**breken = 부수다)
> - 완료: 동작을 끝까지 마침 (maken → **af**maken = 끝내다)
> - 하향: 아래로 내려감 (dalen → **af**dalen = 내려가다)

| 원형 | 뜻 | 현재시제 예시 | 과거분사 |
|------|-----|-------------|---------|
| **af**maken | 끝내다 | Ik maak mijn huiswerk **af**. | **af**gemaakt |
| **af**spreken | 약속하다 | We spreken om 8 uur **af**. | **af**gesproken |
| **af**wassen | 설거지하다 | Zij wast de borden **af**. | **af**gewassen |
| **af**sluiten | 잠그다/마무리하다 | Sluit de deur **af**. | **af**gesloten |
| **af**zeggen | 취소하다 | Hij zegt de afspraak **af**. | **af**gezegd |
| **af**breken | 부수다/중단하다 | Ze breken het oude huis **af**. | **af**gebroken |
| **af**halen | 가지러 가다 | Ik haal het pakket **af**. | **af**gehaald |
| **af**nemen | 줄다/빼앗다 | De pijn neemt langzaam **af**. | **af**genomen |
| **af**rekenen | 계산하다 | We rekenen bij de kassa **af**. | **af**gerekend |
| **af**vallen | 살 빠지다/떨어지다 | Hij valt vijf kilo **af**. | **af**gevallen |
| **af**wachten | 기다리다 | We wachten het resultaat **af**. | **af**gewacht |
| **af**drogen | 닦다(건조) | Droog de borden **af**. | **af**gedroogd |

**3. bij- (곁에, 추가, 보충)**

> **의미 변화:** 기본동사에 "추가·보충", "동반·곁에", "회복" 의미를 부여합니다.
> - 추가: 기존 것에 더함 (betalen → **bij**betalen = 추가로 지불하다)
> - 회복: 원래 상태로 돌아옴 (komen → **bij**komen = 의식을 되찾다)
> - 곁에: 가까이 있음 (staan → **bij**staan = 도와주다, "곁에 서다")

| 원형 | 뜻 | 현재시제 예시 | 과거분사 |
|------|-----|-------------|---------|
| **bij**komen | 의식을 되찾다/회복하다 | Na het ongeluk komt hij langzaam **bij**. | **bij**gekomen |
| **bij**dragen | 기여하다 | Iedereen draagt iets **bij**. | **bij**gedragen |
| **bij**houden | 따라가다/관리하다 | Ik houd mijn administratie **bij**. | **bij**gehouden |
| **bij**staan | 도와주다 | Mijn vrienden staan me altijd **bij**. | **bij**gestaan |
| **bij**vullen | 보충하다 | Vul je glas **bij**. | **bij**gevuld |
| **bij**werken | 업데이트하다 | Ik werk mijn cv **bij**. | **bij**gewerkt |
| **bij**betalen | 추가로 지불하다 | Je moet tien euro **bij**betalen. | **bij**betaald |
| **bij**leren | 추가로 배우다 | Je kunt altijd iets **bij**leren. | **bij**geleerd |
| **bij**praten | 근황을 나누다 | We moeten snel **bij**praten. | **bij**gepraat |

**4. in- (방향: 안으로 / 함몰, 삽입)**

| **in**leveren | 제출하다 | Lever je huiswerk **in**. | **in**geleverd |
| **in**schenken | (음료를) 따르다 | Zij schenkt koffie **in**. | **in**geschonken |

### 분리동사의 어순 규칙

**1. 주절 — 접두사가 문장 끝으로:**
```
Ik bel je morgen op.
(나는 내일 너에게 전화할 거야.)

Zij trekt haar jas aan.
(그녀는 재킷을 입는다.)

We spreken om 8 uur af.
(우리는 8시에 약속한다.)
```

**2. 종속절 — 다시 붙여서 끝으로:**
```
...dat ik je morgen opbel.
(...내가 내일 전화한다는 것)

...omdat zij haar jas aantrekt.
(...그녀가 재킷을 입기 때문에)

...wanneer we om 8 uur afspreken.
(...우리가 8시에 약속할 때)
```

**3. 현재완료 — ge-가 접두사와 어근 사이에:**
```
Ik heb je gisteren opgebeld.
(어제 전화했어.)

Zij heeft haar jas aangetrokken.
(그녀는 재킷을 입었다.)

We hebben om 8 uur afgesproken.
(8시에 약속했다.)
```

**4. 조동사와 함께 — 원형으로 끝에:**
```
Ik moet je morgen opbellen.
(내일 전화해야 해.)

Zij wil haar jas aantrekken.
(그녀는 재킷을 입고 싶어한다.)

Kun je de deur afsluiten?
(문 잠글 수 있어?)
```

**5. te + 분리동사 — te가 사이에:**
```
Ik probeer je op te bellen.
(전화하려고 해.)

Het is moeilijk om vroeg op te staan.
(일찍 일어나는 것은 어렵다.)

Hij vergeet altijd de deur af te sluiten.
(그는 항상 문 잠그는 것을 잊는다.)
```

### 비분리동사 (Onscheidbare werkwoorden)

**be-, er-, ge-, her-, mis-, ont-, ver-** 접두사는 절대 분리되지 않으며, 과거분사에서도 **ge-를 붙이지 않습니다**.

> **핵심 규칙:** 비분리 전철은 강세가 없고(동사 어근에 강세), 과거분사에 ge-가 붙지 않습니다.
> ```
> betalen → betaald (✅)    (×gebetaald)
> vergeten → vergeten (✅)  (×gevergeten)
> ```

**2. ver- (변화, 강화, 잘못됨, 이동)**

> **의미 변화:** 매우 다양한 의미를 가진 전철입니다. 크게 네 가지 기능이 있습니다:
> - **변화:** 상태의 변화 (anderen → **ver**anderen = 바꾸다)
> - **강화:** 의미를 강하게 함 (groot → **ver**groten = 확대하다)
> - **잘못됨:** 실수·부정적 행위 (kijken → **ver**kijken = 잘못 보다)
>   ```
>   spreken → verspreken (zich) = 말실수하다
>   slapen → verslapen (zich) = 늦잠 자다
>   schrijven → verschrijven (zich) = 잘못 쓰다
>   ```
> - **이동·멀어짐:** 떠나거나 사라짐 (trekken → **ver**trekken = 출발하다)

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|

### 분리 vs 비분리 — 동일 접두사가 두 가지로 쓰이는 양용 전철

일부 접두사(**door, over, onder, om, voor**)는 **강세 위치**에 따라 분리/비분리가 달라지며, 의미도 완전히 바뀝니다. 이것은 네덜란드어에서 가장 주의해야 할 부분입니다.

> **핵심 규칙:**
> - **접두사에 강세** → 분리동사 (보통 구체적/물리적 의미)
> - **어근에 강세** → 비분리동사 (보통 추상적/비유적 의미)

---

### 전철별 의미 변화 총정리표

| 분류 | 전철 | 핵심 의미 | 예시 |
|------|------|----------|------|
| **분리** | aan- | 접근, 시작, 켜기 | aandoen (켜다), aankomen (도착하다) |
| | af- | 떨어짐, 완료, 제거 | afmaken (끝내다), afnemen (줄다) |
| | bij- | 추가, 곁에, 회복 | bijkomen (회복하다), bijdragen (기여하다) |
| | in- | 안으로, 삽입 | instappen (탑승), invullen (기입) |
| | mee- | 함께, 동반 | meenemen (가져가다), meedoen (참여) |
| | na- | 뒤따름, 모방, 확인 | nadenken (숙고), nadoen (흉내) |
| | op- | 위로, 시작, 소진 | opstaan (일어나다), opeten (다 먹다) |
| | open- | 열림 | opendoen (열다), opengaan (열리다) |
| | rond- | 주위에, 돌아다님 | rondlopen (돌아다니다), rondkijken (둘러보다) |
| | samen- | 함께, 합침 | samenwerken (협업), samenwonen (동거) |

### 한국어 화자가 자주 하는 실수

```
❌ Ik opbel je morgen. (주절에서 분리 필수)
✅ Ik bel je morgen op.

❌ ...dat ik je morgen bel op. (종속절에서는 붙여야 함)
✅ ...dat ik je morgen opbel.

❌ Ik heb je gebeld op. (과거분사: ge-가 사이에)
✅ Ik heb je opgebeld.

❌ Ik probeer op te je bellen. (te의 위치)
✅ Ik probeer je op te bellen.

❌ Ik heb geontmoeten. (비분리동사에는 ge- 없음)
✅ Ik heb ontmoet.

❌ Hij heeft gebetaald. (비분리동사에는 ge- 없음)
✅ Hij heeft betaald.

❌ De politie zoekt het huis door. (비분리 doorzoeken을 분리하면 안 됨)
✅ De politie doorzoekt het huis.

❌ Hij drijft altijd over. (비분리 overdrijven을 분리하면 안 됨)
✅ Hij overdrijft altijd.

❌ Ik heb doorgelezen het boek. (분리 doorlezen의 과거분사 위치)
✅ Ik heb het boek doorgelezen.
```

> **양용 전철 구분 팁:**
> - 발음으로 구별하세요: **DOOR**gaan (분리) vs door**ZOE**ken (비분리)
> - 구체적/물리적 → 대부분 분리 / 추상적/비유적 → 대부분 비분리
> - 사전에서 강세 표시를 확인하세요

### 분리동사 연습 대화

```
A: Hoe laat sta je morgen op?
   (내일 몇 시에 일어나?)
B: Ik sta om 7 uur op, want ik moet om 8 uur vertrekken.
   (7시에 일어나, 8시에 출발해야 하거든.)
A: Neem je de trein? Vergeet niet om een kaartje mee te nemen.
   (기차 타? 표 가져가는 거 잊지 마.)
B: Ik heb al een kaartje gekocht. Haal je me op van het station?
   (이미 표 샀어. 역에서 데리러 와줄래?)
A: Ja, ik kom je om 9 uur ophalen. Bel me op als je aankomt.
   (응, 9시에 데리러 갈게. 도착하면 전화해.)
B: Top! Ik bel je op zodra ik uitstap.
   (좋아! 하차하자마자 전화할게.)
```

### 양용 전철 연습 대화

```
A: De politie doorzoekt het hele gebouw.
   (경찰이 건물 전체를 수색하고 있어.)
B: Ik lees het rapport nog even door voordat we overleggen.
   (상의하기 전에 보고서를 끝까지 읽을게.)
A: Goed. We moeten dit probleem ook nog bespreken. Het komt steeds vaker voor.
   (좋아. 이 문제도 논의해야 해. 점점 더 자주 발생하고 있거든.)
B: Ja, we moeten voorkomen dat het erger wordt.
   (응, 더 악화되는 걸 방지해야 해.)
A: De directeur overweegt een nieuw plan. Stap je over naar de andere afdeling?
   (이사가 새 계획을 고려 중이야. 다른 부서로 옮길 거야?)
B: Dat overwegen we morgen. Nu gaan we door met het huidige werk.
   (그건 내일 고려해보자. 지금은 현재 업무를 계속하자.)
```
