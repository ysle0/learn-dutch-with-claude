# waar + 전치사 완전 목록 (사물에 대해)

전치사 + 사물은 **waar-** 형태로 합칩니다. 사물을 가리킬 때는 반드시 이 형태를 사용합니다 (전치사 + dat/die 불가):

| 전치사 | waar- 형태 | 예시 |
|--------|-----------|------|
| aan | **waaraan** | Het project **waaraan** ik werk. (내가 작업하는 프로젝트) |
| achter | **waarachter** | De muur **waarachter** hij zich verstopt. (그가 숨은 벽) |
| bij | **waarbij** | De cursus **waarbij** je een certificaat krijgt. (수료증을 받는 과정) |
| door | **waardoor** | De reden **waardoor** ik te laat was. (내가 늦은 이유) |
| in | **waarin** | Het huis **waarin** ik woon. (내가 사는 집) |
| langs | **waarlangs** | De weg **waarlangs** we reden. (우리가 달린 길) |
| mee (= met) | **waarmee** | Het mes **waarmee** ik snij. (내가 자르는 칼) |
| na | **waarna** | De vergadering **waarna** we gingen eten. (식사하러 간 후의 회의) |
| naar | **waarnaar** | Het land **waarnaar** ik op reis ga. (내가 여행 가는 나라) |
| om | **waarom** | De reden **waarom** ik dit doe. (내가 이것을 하는 이유) |
| onder | **waaronder** | De tafel **waaronder** de kat zit. (고양이가 앉아 있는 테이블) |
| op | **waarop** | De stoel **waarop** je zit. (네가 앉아 있는 의자) |
| over | **waarover** | Het onderwerp **waarover** we praten. (우리가 이야기하는 주제) |
| tegen | **waartegen** | De muur **waartegen** ik leun. (내가 기대는 벽) |
| toe (= tot) | **waartoe** | Het doel **waartoe** dit dient. (이것이 쓰이는 목적) |
| tussen | **waartussen** | De twee huizen **waartussen** het park ligt. (공원이 있는 두 집 사이) |
| uit | **waaruit** | De stad **waaruit** ik kom. (내가 출신인 도시) |
| van | **waarvan** | Het bedrijf **waarvan** ik de naam vergeten ben. (이름을 잊은 회사) |
| voor | **waarvoor** | Het examen **waarvoor** ik studeer. (내가 공부하는 시험) |

> **형태 변화 주의:** met → **mee** (waarmee), tot → **toe** (waartoe)
