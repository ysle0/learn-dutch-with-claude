# waar-의 분리와 비분리

waar-형태는 붙여 쓸 수도, 분리할 수도 있습니다. 분리하면 더 구어체적입니다:

```
붙여 쓰기 (문어체): Het huis waarin ik woon, is oud.
분리 (구어체):       Het huis waar ik in woon, is oud.
                    (내가 사는 집은 오래되었다.)

붙여 쓰기: Het mes waarmee ik snij, is scherp.
분리:      Het mes waar ik mee snij, is scherp.
           (내가 자르는 칼은 날카롭다.)

붙여 쓰기: De stoel waarop je zit, is kapot.
분리:      De stoel waar je op zit, is kapot.
           (네가 앉은 의자는 고장났다.)
```
