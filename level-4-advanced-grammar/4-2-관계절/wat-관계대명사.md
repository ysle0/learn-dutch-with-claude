# wat을 관계대명사로 사용하는 경우

선행사가 부정대명사(alles, iets, niets, veel, dat 등)이거나 절 전체를 받을 때 **wat**을 사용합니다:

```
Alles wat hij zegt, is waar.
(그가 말하는 모든 것은 사실이다.)

Iets wat ik niet begrijp...
(내가 이해하지 못하는 것...)

Niets wat je doet, kan mij boos maken.
(네가 하는 어떤 것도 나를 화나게 할 수 없다.)

Dat is het mooiste wat ik ooit heb gezien.
(그것은 내가 본 가장 아름다운 것이다.)

Hij kwam te laat, wat mij erg irriteerde.
(그는 늦게 왔는데, 그것이 나를 매우 짜증나게 했다.)

Ze is geslaagd, wat iedereen had verwacht.
(그녀가 합격했는데, 모두가 예상한 것이었다.)
```
