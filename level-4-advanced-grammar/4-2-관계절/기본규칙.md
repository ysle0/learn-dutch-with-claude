# 관계대명사 기본 규칙

| 선행사 | 관계대명사 | 예시 |
|--------|-----------|------|
| **de 명사** (주격) | **die** | De man **die** daar staat, is mijn vader. |
| **de 명사** (목적격) | **die** | De vrouw **die** ik ken, woont hier. |
| **het 명사** (주격) | **dat** | Het boek **dat** op tafel ligt, is van mij. |
| **het 명사** (목적격) | **dat** | Het boek **dat** ik lees, is interessant. |
| 복수 명사 (de/het 무관) | **die** | De kinderen **die** spelen, zijn blij. |
| 소유격 (남성) | **wiens** | De man **wiens** auto gestolen is... |
| 소유격 (여성/복수) | **wier** | De vrouw **wier** tas ik vond... |
| 전치사 + 사물 | **waar + 전치사** | Het huis **waarin** ik woon... |
| 전치사 + 사람 | **전치사 + wie** | De man **met wie** ik werk... |

> **핵심 규칙:** de-woord → **die**, het-woord → **dat**, 복수 → 항상 **die**
