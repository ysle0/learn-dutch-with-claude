# 과거완료 vs 단순 과거 비교

**단순 과거만 사용 (시간 순서가 불명확할 수 있음):**
```
Ik at en hij kwam binnen.
(나는 먹고 있었고 그가 들어왔다. — 동시에?)
```

**과거완료 사용 (시간 순서가 명확):**
```
Ik had al gegeten toen hij binnenkwam.
(그가 들어왔을 때 나는 이미 먹은 상태였다. — 먹기가 먼저)
```

**더 많은 비교 예시:**
```
단순 과거: Ik verloor mijn sleutels.
          (나는 열쇠를 잃어버렸다.)
과거완료:  Ik had mijn sleutels verloren, dus kon ik niet naar binnen.
          (나는 열쇠를 잃어버린 상태였기에, 안에 들어갈 수 없었다.)

단순 과거: Ze studeerde Nederlands.
          (그녀는 네덜란드어를 공부했다.)
과거완료:  Ze had al drie jaar Nederlands gestudeerd voordat ze naar Nederland verhuisde.
          (그녀는 네덜란드로 이사하기 전에 이미 3년간 네덜란드어를 공부한 상태였다.)
```
