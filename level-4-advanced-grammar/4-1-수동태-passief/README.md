# 4-1. 수동태 (Lijdende vorm)

> 네덜란드어의 수동태는 한국어의 피동 표현("-되다", "-당하다", "-받다")과 비슷한 기능을 합니다. 영어의 수동태(be + past participle)와 구조적으로 유사하지만, 네덜란드어는 **worden**(과정)과 **zijn**(상태)을 구분한다는 점이 독특합니다.
