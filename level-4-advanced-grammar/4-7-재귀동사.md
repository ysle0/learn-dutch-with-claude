# 4-7. 재귀동사 (Wederkerend werkwoord)

재귀동사는 동작이 주어 자신에게 돌아오는 동사입니다. 한국어에서는 "씻다", "숨다"처럼 자동사로 표현되는 경우가 많지만, 네덜란드어에서는 재귀대명사를 명시적으로 사용해야 합니다. 영어의 reflexive verbs (wash oneself, introduce oneself)와 비슷하지만, 네덜란드어에서 훨씬 더 많이 사용됩니다.

### 재귀대명사

| 인칭 | 비강세형 | 강세형 | 예시 |
|------|---------|-------|------|
| ik | me | mij | Ik was **me** / Ik was **mij**. |
| jij | je | jou | Jij wast **je** / Jij wast **jou**. |
| u | zich | zich | U wast **zich**. |
| hij | zich | zichzelf | Hij wast **zich** / Hij wast **zichzelf**. |
| zij (단수) | zich | zichzelf | Zij wast **zich**. |
| het | zich | zichzelf | Het kind wast **zich**. |
| wij | ons | onszelf | Wij wassen **ons**. |
| jullie | je | jezelf/jullie zelf | Jullie wassen **je**. |
| zij (복수) | zich | zichzelf | Zij wassen **zich**. |

> **비강세형 vs 강세형:**
> - 비강세형: 일반적으로 사용 (Ik was me.)
> - 강세형: 대조/강조할 때 (Ik was **mij**, niet jou!)
> - zichzelf: "자기 자신을" 강조 (Hij bekijkt **zichzelf** in de spiegel.)

### 진짜 재귀동사 (Echte wederkerende werkwoorden)

재귀대명사 없이는 사용할 수 없는 동사입니다. 이 동사들은 항상 재귀대명사가 필요합니다:

| 동사 | 뜻 | 예시 |
|------|-----|------|
| zich vergissen | 실수하다 | Ik heb **me** vergist. (실수했다.) |
| zich schamen | 부끄러워하다 | Hij schaamt **zich** voor zijn gedrag. (그는 자기 행동이 부끄럽다.) |
| zich gedragen | 처신하다 | Gedraag **je**! (행동 바르게 해!) |
| zich bevinden | 위치하다 | Het hotel bevindt **zich** in het centrum. (호텔은 중심에 있다.) |
| zich verzetten | 저항하다 | Zij verzet **zich** tegen de plannen. (그녀는 계획에 반대한다.) |
| zich misdragen | 잘못 행동하다 | De kinderen misdragen **zich**. (아이들이 버릇없이 군다.) |
| zich verbazen | 놀라다 | Ik verbaas **me** over het nieuws. (그 뉴스에 놀랐다.) |
| zich afvragen | 궁금해하다 | Ik vraag **me** af of het waar is. (그게 사실인지 궁금하다.) |
| zich realiseren | 깨닫다 | Ik realiseer **me** dat het laat is. (늦었다는 것을 깨달았다.) |
| zich inspannen | 노력하다 | Hij spant **zich** in voor het examen. (그는 시험을 위해 노력한다.) |

### 가짜 재귀동사 (Oneigenlijke wederkerende werkwoorden)

재귀대명사 대신 다른 목적어를 넣을 수도 있는 동사입니다:

| 동사 | 재귀 의미 | 비재귀 의미 |
|------|----------|-----------|
| zich wassen | 씻다 (자기를) | de auto wassen (차를 씻다) |
| zich aankleden | 옷을 입다 | het kind aankleden (아이에게 옷을 입히다) |
| zich voorstellen | 자기소개하다 | iemand voorstellen (누군가를 소개하다) |
| zich herinneren | 기억하다 | iemand herinneren aan... (누군가에게 상기시키다) |
| zich verstoppen | 숨다 | iets verstoppen (무언가를 숨기다) |
| zich verdedigen | 자신을 방어하다 | iemand verdedigen (누군가를 방어하다) |

```
재귀: Ik was me.              (나는 씻는다.)
비재귀: Ik was de auto.        (나는 차를 씻는다.)

재귀: Zij kleedt zich aan.    (그녀는 옷을 입는다.)
비재귀: Zij kleedt het kind aan. (그녀는 아이에게 옷을 입힌다.)

재귀: Mag ik me voorstellen?   (자기소개해도 될까요?)
비재귀: Mag ik mijn vriend voorstellen? (제 친구를 소개해도 될까요?)
```

### 자주 쓰이는 재귀동사 — 카테고리별

**일상생활/신체:**

| 동사 | 뜻 | 예시 |
|------|-----|------|
| zich wassen | 씻다 | Ik was **me** elke ochtend. |
| zich douchen | 샤워하다 | Hij doucht **zich** 's ochtends. |
| zich aankleden | 옷을 입다 | Zij kleedt **zich** snel aan. |
| zich uitkleden | 옷을 벗다 | Kleed **je** uit! |
| zich scheren | 면도하다 | Hij scheert **zich** elke dag. |
| zich opmaken | 화장하다 | Zij maakt **zich** op voor het feest. |
| zich omkleden | 옷을 갈아입다 | Ik kleed **me** even om. |

**감정/상태:**

| 동사 | 뜻 | 예시 |
|------|-----|------|
| zich voelen | 느끼다 | Ik voel **me** goed. (기분이 좋다.) |
| zich zorgen maken | 걱정하다 | Zij maakt **zich** zorgen over haar zoon. |
| zich ergeren | 짜증내다 | Ik erger **me** aan het lawaai. |
| zich vervelen | 지루해하다 | De kinderen vervelen **zich**. |
| zich amuseren | 즐기다 | We amuseren **ons** op het feest. |
| zich schamen | 부끄러워하다 | Hij schaamt **zich** diep. |

**행동/활동:**

| 동사 | 뜻 | 예시 |
|------|-----|------|
| zich voorbereiden | 준비하다 | Ik bereid **me** voor op het examen. |
| zich concentreren | 집중하다 | Concentreer **je** op je werk! |
| zich specialiseren | 전문화하다 | Zij specialiseert **zich** in taalwetenschap. |
| zich aanmelden | 등록하다 | Ik meld **me** aan voor de cursus. |
| zich verontschuldigen | 사과하다 | Hij verontschuldigt **zich** voor zijn fout. |
| zich bezighouden met | ~에 종사하다 | Zij houdt **zich** bezig met onderzoek. |
| zich interesseren voor | ~에 관심이 있다 | Ik interesseer **me** voor kunst. |

### 재귀동사의 어순

**주절:**
```
Ik was me elke ochtend.
(나는 매일 아침 씻는다.)

Zij kleedt zich snel aan.
(그녀는 빨리 옷을 입는다.)
```

**종속절 — 재귀대명사가 주어 바로 뒤에:**
```
...omdat ik me elke ochtend was.
(...내가 매일 아침 씻기 때문에)

...dat zij zich snel aankleedt.
(...그녀가 빨리 옷을 입는다는 것)
```

**도치문 — 재귀대명사가 주어 바로 뒤에 또는 앞에:**
```
Morgen was ik me niet.       (내일은 안 씻을 거야.)
Elke dag voelt hij zich goed. (매일 그는 기분이 좋다.)
```

**현재완료:**
```
Ik heb me gewassen.          (나는 씻었다.)
Zij heeft zich aangekleed.    (그녀는 옷을 입었다.)
Hij heeft zich vergist.       (그는 실수했다.)
```

### 상호 대명사: elkaar (서로)

재귀대명사가 "서로"의 의미를 가질 때 **elkaar**를 사용합니다:

```
Zij wassen zich.             (그들은 (각자) 씻는다.)
Zij wassen elkaar.           (그들은 서로를 씻긴다.)

We kennen elkaar al lang.    (우리는 오래전부터 서로 안다.)
Ze helpen elkaar altijd.     (그들은 항상 서로를 돕는다.)
We zien elkaar elke week.    (우리는 매주 서로 만난다.)
Jullie begrijpen elkaar goed. (너희는 서로를 잘 이해한다.)
Ze schrijven elkaar brieven.  (그들은 서로에게 편지를 쓴다.)
We bellen elkaar elke dag.   (우리는 매일 서로에게 전화한다.)
```

**전치사 + elkaar:**
```
We praten met elkaar.         (우리는 서로 이야기한다.)
Ze kijken naar elkaar.        (그들은 서로를 쳐다본다.)
We houden van elkaar.         (우리는 서로 사랑한다.)
Ze zijn boos op elkaar.       (그들은 서로에게 화가 나 있다.)
De huizen staan naast elkaar. (집들이 나란히 서 있다.)
Ze denken aan elkaar.         (그들은 서로를 생각한다.)
```

### 신체 부위 + 재귀동사

네덜란드어에서는 신체 부위에 소유격 대명사 대신 정관사 + 재귀대명사를 쓰는 경우가 많습니다:

```
Ik was me de handen.          (나는 손을 씻는다.)
Hij breekt zich het hoofd.    (그는 머리를 싸맨다. = 고심한다.)
Zij poetst zich de tanden.    (그녀는 이를 닦는다.)
Ik kam me het haar.           (나는 머리를 빗는다.)

비교:
네덜란드어: Ik was me de handen.   (정관사 de + 재귀대명사 me)
영어:      I wash my hands.        (소유격 my)
한국어:    나는 손을 씻는다.         (재귀 표현 없음)
```

### 한국어 화자가 자주 하는 실수

```
❌ Ik voel goed. (재귀대명사 누락)
✅ Ik voel me goed.

❌ Hij herinnert dat. (재귀대명사 누락)
✅ Hij herinnert zich dat.

❌ Ik vraag af of... (재귀대명사 누락)
✅ Ik vraag me af of...

❌ Wij wassen zich. (인칭 불일치: wij → ons)
✅ Wij wassen ons.

❌ ...dat ik voelde me goed. (종속절 어순 오류)
✅ ...dat ik me goed voelde.

❌ Zij helpen zich. (서로 → elkaar)
✅ Zij helpen elkaar. (서로를 돕는다.)
   Zij helpen zich. (각자 자신을 돕는다. — 뜻이 다름!)
```

### 재귀동사 연습 대화

```
A: Hoe voel je je vandaag?
   (오늘 기분이 어때?)
B: Ik voel me een beetje moe. Ik heb me gisteravond
   niet goed kunnen voorbereiden op het examen.
   (좀 피곤해. 어젯밤에 시험 준비를 잘 못했어.)
A: Maak je je zorgen over het examen?
   (시험이 걱정돼?)
B: Ja, ik probeer me te concentreren, maar ik verveel
   me snel als ik alleen studeer.
   (응, 집중하려고 하는데, 혼자 공부하면 금방 지루해져.)
A: Zullen we elkaar helpen? Ik bereid me ook voor
   op hetzelfde examen.
   (서로 도울까? 나도 같은 시험 준비 중이야.)
B: Goed idee! Laten we ons aanmelden voor de studiegroep.
   (좋은 생각! 스터디 그룹에 등록하자.)
A: We moeten ons wel goed gedragen in de bibliotheek!
   (도서관에서 행동 바르게 해야 해!)
```
