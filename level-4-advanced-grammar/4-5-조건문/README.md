# 4-5. 조건문 (Voorwaardelijke zinnen)

조건문은 "~하면 ~하다"라는 조건-결과 관계를 표현합니다. 한국어의 "-면", "-다면", "-었더라면"과 유사합니다. 네덜란드어의 조건문은 영어의 conditional sentences와 거의 같은 체계를 가지고 있습니다.
