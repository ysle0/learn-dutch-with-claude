# mits (provided that / ~하는 조건으로)

```
Je mag naar het feest, mits je om 12 uur thuis bent.
(12시까지 집에 오는 조건으로 파티에 가도 돼.)

Ik help je, mits je mij ook helpt.
(나도 도와주는 조건으로 도와줄게.)

Het project gaat door, mits we genoeg budget hebben.
(충분한 예산이 있는 조건으로 프로젝트를 진행한다.)
```

---

# tenzij (unless / ~하지 않는 한)

```
Ik ga wandelen, tenzij het regent.
(비가 오지 않는 한, 산책하러 갈 거야.)

We gaan door met het project, tenzij de baas het annuleert.
(상사가 취소하지 않는 한, 프로젝트를 계속할 거야.)

Tenzij je een goede reden hebt, moet je op tijd komen.
(좋은 이유가 없는 한, 제시간에 와야 해.)

Ik zal het doen, tenzij iemand anders het wil doen.
(다른 사람이 하고 싶어하지 않는 한, 내가 할게.)

Tenzij het heel koud is, ga ik met de fiets.
(매우 춥지 않는 한, 자전거로 갈 거야.)
```
