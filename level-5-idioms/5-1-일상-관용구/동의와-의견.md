# 동의와 의견

| 관용구 | 의미 |
|--------|------|
| Dat klopt. | 맞아요. (That's correct.) |
| Precies! | 정확해! |
| Inderdaad. | 정말/확실히. (Indeed.) |
| Dat vind ik ook. | 나도 그렇게 생각해. |
| Dat ben ik met je eens. | 네 말에 동의해. |
| Dat ben ik niet met je eens. | 네 말에 동의하지 않아. |
| Dat is niet waar. | 그건 사실이 아니야. |
| Nou ja... | 음, 글쎄... |
| Eerlijk gezegd... | 솔직히 말하면... |
| Aan de ene kant... aan de andere kant... | 한편으로는... 다른 한편으로는... |
