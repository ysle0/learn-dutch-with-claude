# 5-6. 비즈니스/격식 표현

| 표현 | 의미 | 사용 상황 |
|------|------|----------|
| Met alle respect... | 모든 존경을 담아... | 반대 의견 제시 전 |
| Naar mijn mening... | 제 의견으로는... | 의견 제시 |
| Ik zou graag willen... | ~을 하고 싶습니다. | 정중한 요청 |
| Zou u zo vriendelijk willen zijn om...? | ~해 주실 수 있을까요? | 매우 정중한 요청 |
| Bij voorbaat dank. | 미리 감사드립니다. | 이메일 마무리 |
| Met vriendelijke groet | 친절한 인사와 함께 | 이메일 서명 |
| Hoogachtend | 존경을 담아 | 매우 격식적 이메일 서명 |
| Geachte heer/mevrouw | 존경하는 선생님/여사님 | 이메일 시작 |
| Ter informatie | 참고로 | 정보 공유 시 |
| In overleg met... | ~와 협의하여 | 결정 설명 시 |
