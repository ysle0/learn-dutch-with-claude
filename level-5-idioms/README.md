# Level 5: 관용구와 표현

> 일상에서 자주 쓰이는 관용구, 속담, 구어체 표현을 배웁니다.

---

## 5-1. 일상 관용구 (Dagelijkse uitdrukkingen)

### 감정과 상태

| 관용구 | 직역 | 의미 |
|--------|------|------|
| Ik heb er zin in. | 나는 거기에 맛/의지가 있다. | 기대돼! / 하고 싶어! |
| Ik heb er geen zin in. | 나는 거기에 맛이 없다. | 하기 싫어. |
| Ik ben er klaar mee. | 나는 거기에 대해 끝났다. | 진절머리가 나. / 지겨워. |
| Het valt mee. | 그것은 (기대보다) 덜 나쁘다. | 생각보다 괜찮아. |
| Het valt tegen. | 그것은 (기대보다) 실망이다. | 기대에 못 미쳐. / 실망이야. |
| Ik heb er schoon genoeg van. | 나는 거기에 깨끗하게 충분하다. | 정말 진절머리가 나. |
| Het zit me dwars. | 그것이 나를 가로로 앉아있다. | 마음에 걸려. / 신경 쓰여. |
| Ik voel me niet lekker. | 나는 맛있지 않게 느낀다. | 몸이 안 좋아. |
| Het kan me niet schelen. | 그것이 나를 구분짓지 못한다. | 상관없어. |
| Ik heb er geen kaas van gegeten. | 나는 거기서 치즈를 안 먹었다. | 전혀 모르겠어. (아는 게 하나도 없다.) |

### 동의와 의견

| 관용구 | 의미 |
|--------|------|
| Dat klopt. | 맞아요. (That's correct.) |
| Precies! | 정확해! |
| Inderdaad. | 정말/확실히. (Indeed.) |
| Dat vind ik ook. | 나도 그렇게 생각해. |
| Dat ben ik met je eens. | 네 말에 동의해. |
| Dat ben ik niet met je eens. | 네 말에 동의하지 않아. |
| Dat is niet waar. | 그건 사실이 아니야. |
| Nou ja... | 음, 글쎄... |
| Eerlijk gezegd... | 솔직히 말하면... |
| Aan de ene kant... aan de andere kant... | 한편으로는... 다른 한편으로는... |

### 일상 반응

| 관용구 | 의미 |
|--------|------|
| Gezellig! | 아늑해! / 좋다! / 분위기 좋다! |
| Lekker! | 맛있다! / 좋다! / 기분 좋다! |
| Jammer! | 안타깝다! / 아쉽다! |
| Wat leuk! | 얼마나 재미있어! / 좋다! |
| Wat jammer! | 정말 아쉽다! |
| Geen probleem. | 문제없어. |
| Maakt niet uit. | 상관없어. |
| Laat maar. | 됐어. / 신경 쓰지 마. |
| Dat meen je niet! | 설마! / 진짜야?! |
| Echt waar? | 정말이야? |

> **gezellig**: 네덜란드어에서 가장 중요한 단어 중 하나! 정확히 번역하기 어렵지만, 따뜻하고 아늑하고 함께하는 즐거운 분위기를 뜻합니다. 친구와의 모임, 카페의 분위기, 크리스마스 마켓 등을 묘사할 때 씁니다.

## 5-2. 동물 관련 관용구

네덜란드어에는 동물을 사용한 관용구가 특히 많습니다.

| 관용구 | 직역 | 의미 |
|--------|------|------|
| De kat uit de boom kijken | 나무에서 고양이를 내려다보다 | 상황을 관망하다 |
| Met de kippen op stok gaan | 닭들과 함께 횃대에 올라가다 | 일찍 잠자리에 들다 |
| Als een olifant in een porseleinkast | 도자기 가게의 코끼리처럼 | 매우 어색하고 서투르게 |
| Twee vliegen in één klap slaan | 한 번에 두 마리 파리를 잡다 | 일석이조 |
| De hond in de pot vinden | 냄비에서 개를 발견하다 | 너무 늦게 도착하다 (음식이 다 없어진 후) |
| Het paard achter de wagen spannen | 마차 뒤에 말을 묶다 | 순서를 바꿔 하다 (본말전도) |
| Een ezel stoot zich geen twee keer aan dezelfde steen | 당나귀도 같은 돌에 두 번 걸려 넘어지지 않는다 | 같은 실수를 반복하지 마라 |
| Zo ziek als een hond | 개처럼 아프다 | 매우 아프다 |
| Mieren neuken | 개미와 성교하다 | 사소한 것에 지나치게 집착하다 |
| Een kat in het donker knijpen | 어둠 속에서 고양이를 꼬집다 | 몰래 나쁜 짓을 하다 |

## 5-3. 날씨 관련 표현

네덜란드에서 날씨는 매우 중요한 대화 주제입니다!

| 표현 | 의미 |
|------|------|
| Het regent. | 비가 온다. |
| Het waait. | 바람이 분다. |
| Het is bewolkt. | 흐리다. |
| Het is zonnig. | 햇볕이 난다. |
| Het vriest. | 얼 정도로 춥다. |
| Het sneeuwt. | 눈이 온다. |
| Wat een hondenweer! | 정말 개 같은 날씨! (악천후) |
| Het regent pijpenstelen. | 파이프 자루가 비처럼 온다. (폭우가 쏟아진다.) |
| Een lekker zonnetje. | 좋은 햇살. (화창한 날씨를 즐길 때) |
| Na regen komt zonneschijn. | 비 뒤에 햇빛이 온다. (고진감래) |

## 5-4. 네덜란드어 속담 (Spreekwoorden)

| 속담 | 직역 | 한국어 대응 |
|------|------|-----------|
| Oost west, thuis best. | 동쪽이든 서쪽이든, 집이 최고. | 집만 한 곳이 없다. |
| Wie het laatst lacht, lacht het best. | 마지막에 웃는 자가 가장 잘 웃는다. | 최후에 웃는 자가 승자다. |
| De appel valt niet ver van de boom. | 사과는 나무에서 멀리 안 떨어진다. | 부전자전. |
| Oefening baart kunst. | 연습이 기술을 낳는다. | 연습이 완벽을 만든다. |
| Wie niet waagt, die niet wint. | 모험하지 않는 자는 이기지 못한다. | 호랑이 굴에 들어가야 호랑이를 잡는다. |
| Beter laat dan nooit. | 안 하는 것보다 늦는 게 낫다. | 늦었다고 생각할 때가 가장 빠르다. |
| Spreken is zilver, zwijgen is goud. | 말하는 것은 은이요, 침묵은 금이다. | 침묵은 금이다. |
| Wie een kuil graaft voor een ander, valt er zelf in. | 남을 위해 구덩이를 파는 자, 스스로 빠진다. | 남을 해치려다 자기가 당한다. |
| Wie A zegt, moet ook B zeggen. | A를 말한 자는 B도 말해야 한다. | 시작했으면 끝을 맺어라. |
| Hoge bomen vangen veel wind. | 높은 나무는 바람을 많이 맞는다. | 높은 자리에는 책임이 따른다. |
| Het is niet al goud wat er blinkt. | 반짝인다고 다 금은 아니다. | 빛 좋은 개살구. |
| Boontje komt om zijn loontje. | 콩이 자기 삯을 받으러 온다. | 인과응보. / 뿌린 대로 거둔다. |
| Zuinigheid met vlijt bouwt huizen als kastelen. | 절약과 근면은 성 같은 집을 짓는다. | 티끌 모아 태산. |
| Haastige spoed is zelden goed. | 서두르는 것은 좀처럼 좋지 않다. | 급할수록 돌아가라. |
| Wie boter op zijn hoofd heeft, moet uit de zon blijven. | 머리에 버터를 올린 자는 햇볕을 피해야 한다. | 자기 결점을 알면 조심해라. |

## 5-5. 구어체 & 비격식 표현

### 축약과 비격식 표현

| 비격식 | 원래 형태 | 뜻 |
|--------|----------|-----|
| d'r | haar | 그녀의 |
| z'n | zijn | 그의 |
| 't | het | 그것 |
| 'k | ik | 나 |
| ie | hij | 그 |
| ff | even | 잠깐 |
| Hè hè! | - | 휴! (안도) |
| Tja... | - | 글쎄... (어쩔 수 없지) |
| Zeg maar... | - | 음... / 그러니까... |
| Weet je? | - | 있잖아? / 알지? |
| Snap je? | - | 이해해? |
| Echt? / Serieus? | - | 진짜? |
| Geen idee. | - | 모르겠어. |
| Doe maar. | - | 그렇게 해. / 그걸로 할게. |
| Doe maar normaal. | - | 보통으로 해. / 유난떨지 마. |

> **"Doe maar normaal, dan doe je al gek genoeg"**
> "평범하게 행동해, 그러면 이미 충분히 미친 거야."
> — 네덜란드 문화를 대표하는 격언. 겸손하고 평범하게 행동하는 것을 중시합니다.

## 5-6. 비즈니스/격식 표현

| 표현 | 의미 | 사용 상황 |
|------|------|----------|
| Met alle respect... | 모든 존경을 담아... | 반대 의견 제시 전 |
| Naar mijn mening... | 제 의견으로는... | 의견 제시 |
| Ik zou graag willen... | ~을 하고 싶습니다. | 정중한 요청 |
| Zou u zo vriendelijk willen zijn om...? | ~해 주실 수 있을까요? | 매우 정중한 요청 |
| Bij voorbaat dank. | 미리 감사드립니다. | 이메일 마무리 |
| Met vriendelijke groet | 친절한 인사와 함께 | 이메일 서명 |
| Hoogachtend | 존경을 담아 | 매우 격식적 이메일 서명 |
| Geachte heer/mevrouw | 존경하는 선생님/여사님 | 이메일 시작 |
| Ter informatie | 참고로 | 정보 공유 시 |
| In overleg met... | ~와 협의하여 | 결정 설명 시 |
