# 21~99: 일의 자리를 먼저 말함

> **핵심 규칙**: 독일어처럼 **일의 자리 + en + 십의 자리** 순서로 읽습니다!
> 예: 25 = vijf**en**twintig (5 그리고 20)

#### 20대

| 숫자 | 네덜란드어 | 분해 |
|------|-----------|------|
| 21 | eenentwintig | een-en-twintig (1과 20) |
| 22 | tweeëntwintig | twee-en-twintig (2와 20) |
| 23 | drieëntwintig | drie-en-twintig (3과 20) |
| 24 | vierentwintig | vier-en-twintig (4와 20) |
| 25 | vijfentwintig | vijf-en-twintig (5와 20) |
| 26 | zesentwintig | zes-en-twintig (6과 20) |
| 27 | zevenentwintig | zeven-en-twintig (7과 20) |
| 28 | achtentwintig | acht-en-twintig (8과 20) |
| 29 | negenentwintig | negen-en-twintig (9와 20) |
