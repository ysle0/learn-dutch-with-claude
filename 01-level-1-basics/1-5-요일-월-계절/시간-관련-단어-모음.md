#### 기간 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| een dag | 하루 |
| een week | 일주일 |
| twee weken | 2주 |
| een maand | 한 달 |
| een jaar | 1년 |
| een uur | 한 시간 |
| een minuut | 1분 |
| een seconde | 1초 |
| een halfjaar | 반년 |
| een kwartier | 15분 |
| een half uur | 30분 |
| anderhalf uur | 1시간 반 |

```
Ik woon hier al drie jaar.          (여기 산 지 벌써 3년이에요.)
Ik ga twee weken op vakantie.        (2주간 휴가를 갑니다.)
Het duurt een kwartier.              (15분 걸려요.)
We zijn er over een half uur.        (30분 후에 도착해요.)
Ik heb een uur gewacht.              (한 시간 기다렸어요.)
Ik ben hier sinds vorig jaar.        (작년부터 여기 있어요.)
```
