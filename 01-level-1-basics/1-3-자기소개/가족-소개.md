# 가족 소개

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik ben getrouwd. | 저는 결혼했어요. |
| Ik ben vrijgezel / single. | 저는 미혼이에요. |
| Ik heb een vriend(in). | 남자/여자 친구가 있어요. |
| Ik heb [숫자] kinderen. | 아이가 [숫자]명 있어요. |
| Ik heb geen kinderen. | 아이가 없어요. |
| Ik heb een zoon en een dochter. | 아들 하나, 딸 하나 있어요. |
