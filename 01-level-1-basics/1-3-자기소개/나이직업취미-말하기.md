# 나이/직업/취미 말하기

#### 나이

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik ben [숫자] jaar oud. | 저는 [숫자]살입니다. |
| Ik ben vijfentwintig (jaar oud). | 저는 25살이에요. |
| Hoe oud ben je? | 몇 살이야? (비격식) |
| Hoe oud bent u? | 연세가 어떻게 되세요? (격식) |

#### 직업

| 네덜란드어 | 한국어 |
|-----------|-------|
| Wat doe je? / Wat voor werk doe je? | 무슨 일 해? |
| Wat doet u voor werk? | 무슨 일을 하세요? (격식) |
| Ik ben [직업]. | 저는 [직업]입니다. |
| Ik werk bij [회사]. | 저는 [회사]에서 일해요. |
| Ik werk als [직업]. | 저는 [직업]으로 일해요. |
| Ik studeer [전공]. | 저는 [전공]을 공부해요. |
| Ik ben student. | 저는 학생입니다. |
| Ik ben gepensioneerd. | 저는 은퇴했습니다. |
| Ik zoek werk. | 저는 일을 찾고 있어요. |
