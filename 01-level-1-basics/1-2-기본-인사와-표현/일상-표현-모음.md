# 일상 표현 모음

#### 이해/소통 관련

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik begrijp het. | 이해해요. |
| Ik begrijp het niet. | 이해하지 못하겠어요. |
| Kunt u dat herhalen? | 다시 말씀해 주시겠어요? (격식) |
| Kun je dat herhalen? | 다시 말해 줄래? (비격식) |
| Kunt u langzamer spreken? | 더 천천히 말씀해 주시겠어요? |
| Wat bedoel je? | 무슨 뜻이야? |
| Wat betekent dat? | 그게 무슨 뜻이에요? |
| Hoe zeg je ___ in het Nederlands? | ___를 네덜란드어로 뭐라고 해요? |
| Spreekt u Engels? | 영어 하시나요? (격식) |
| Ik spreek een beetje Nederlands. | 네덜란드어를 조금 해요. |
| Mijn Nederlands is niet zo goed. | 제 네덜란드어가 별로 좋지 않아요. |

#### 감정 표현

#### 동의/반대

| 네덜란드어 | 한국어 |
|-----------|-------|
| Dat klopt. | 맞아요. |
| Dat is waar. | 그건 사실이에요. |
| Precies! | 정확해! |
| Inderdaad. | 그렇죠. / 확실히. |
| Ik ben het ermee eens. | 동의해요. |

| Misschien. | 아마도. |
