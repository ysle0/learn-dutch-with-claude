# 전화 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| Met [이름]. | [이름]입니다. (전화 받을 때) |
| U spreekt met [이름]. | [이름]입니다. (격식) |
| Mag ik [이름] spreken? | [이름]과 통화할 수 있나요? |
| Ik bel over... | ~에 대해 전화드렸어요. |
| Kunt u een boodschap achterlaten? | 메시지를 남기시겠어요? |
| Ik bel later terug. | 나중에 다시 전화할게요. |
| Sorry, verkeerd verbonden. | 죄송합니다, 잘못 걸었어요. |
