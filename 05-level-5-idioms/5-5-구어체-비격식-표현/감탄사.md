# 축약과 비격식 표현

| 비격식 | 원래 형태 | 뜻 |
|--------|----------|-----|
| d'r | haar | 그녀의 |
| z'n | zijn | 그의 |
| 't | het | 그것 |
| 'k | ik | 나 |
| ie | hij | 그 |
| ff | even | 잠깐 |
| Hè hè! | - | 휴! (안도) |
| Tja... | - | 글쎄... (어쩔 수 없지) |
| Zeg maar... | - | 음... / 그러니까... |
| Weet je? | - | 있잖아? / 알지? |
| Snap je? | - | 이해해? |
| Echt? / Serieus? | - | 진짜? |
| Geen idee. | - | 모르겠어. |
| Doe maar. | - | 그렇게 해. / 그걸로 할게. |
| Doe maar normaal. | - | 보통으로 해. / 유난떨지 마. |
