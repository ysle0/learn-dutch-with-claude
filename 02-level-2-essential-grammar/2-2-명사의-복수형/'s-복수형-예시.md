# -'s 복수형 예시

주로 모음(a, i, o, u, y)으로 끝나는 외래어에 사용합니다. 어포스트로피(')를 넣는 이유는 발음의 혼동을 방지하기 위해서입니다.

| 단수 | 복수 | 뜻 |
|------|------|-----|
| auto | auto's | 자동차들 |
| foto | foto's | 사진들 |
| menu | menu's | 메뉴들 |
| café | café's | 카페들 |
| taxi | taxi's | 택시들 |
| baby | baby's | 아기들 |
| hobby | hobby's | 취미들 |
| paraplu | paraplu's | 우산들 |
| oma | oma's | 할머니들 |
| opa | opa's | 할아버지들 |
