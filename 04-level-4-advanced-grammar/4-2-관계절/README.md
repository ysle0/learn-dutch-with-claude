# 4-2. 관계절 (Betrekkelijke bijzinnen)

관계절은 명사를 수식하는 종속절입니다. 한국어에서는 "~하는 사람", "~한 책"처럼 관형절로 명사 앞에 놓이지만, 네덜란드어에서는 영어처럼 명사 뒤에 관계절이 옵니다.
