# 제3조건문: 과거의 비현실 (als + 과거완료)

과거에 실제로 일어나지 않은 일을 가정합니다. 한국어의 "~했더라면 ~했을 텐데"에 해당합니다. 후회를 표현할 때 자주 사용됩니다.

**구조: Als + 과거완료, zou(den) + 과거분사 + hebben/zijn**

```
Als ik harder had gestudeerd, zou ik geslaagd zijn.
(더 열심히 공부했더라면, 합격했을 텐데.)

Als ik het had geweten, was ik gekomen.
(알았더라면, 왔을 텐데.)

Als we eerder waren vertrokken, hadden we de trein gehaald.
(더 일찍 출발했더라면, 기차를 탔을 텐데.)

Als ik de waarschuwing had gelezen, zou ik voorzichtiger zijn geweest.
(경고를 읽었더라면, 더 조심했을 텐데.)

Als zij niet had gebeld, zou ik het vergeten zijn.
(그녀가 전화하지 않았더라면, 잊어버렸을 텐데.)

Als het niet had geregend, zouden we zijn gaan wandelen.
(비가 오지 않았더라면, 산책하러 갔을 텐데.)

Als ik meer geld had gespaard, had ik een huis kunnen kopen.
(돈을 더 모았더라면, 집을 살 수 있었을 텐데.)

Als hij op tijd was gekomen, had hij de presentatie niet gemist.
(그가 제시간에 왔더라면, 발표를 놓치지 않았을 텐데.)
```

> **참고:** 구어체에서는 zou(den) 대신 과거완료를 결과절에도 쓸 수 있습니다:
> ```
> 격식: Als ik het had geweten, zou ik zijn gekomen.
> 구어: Als ik het had geweten, was ik gekomen.
> ```
