# voor het geval (dat) (in case / 혹시 ~할 경우를 대비해)

```
Neem een paraplu mee, voor het geval dat het regent.
(비가 올 경우를 대비해 우산을 가져가.)

Ik geef je mijn nummer, voor het geval je me nodig hebt.
(혹시 나한테 연락해야 할 경우를 대비해 번호 줄게.)

Ik heb extra eten meegenomen, voor het geval we langer blijven.
(더 오래 있을 경우를 대비해 음식을 더 가져왔어.)

Zet de wekker, voor het geval je je verslaapt.
(늦잠 잘 경우를 대비해 알람을 맞춰.)

Schrijf het op, voor het geval je het vergeet.
(잊어버릴 경우를 대비해 적어 놔.)
```
