# 분리 vs 비분리 — 동일 접두사가 두 가지로 쓰이는 양용 전철

일부 접두사(**door, over, onder, om, voor**)는 **강세 위치**에 따라 분리/비분리가 달라지며, 의미도 완전히 바뀝니다. 이것은 네덜란드어에서 가장 주의해야 할 부분입니다.

> **핵심 규칙:**
> - **접두사에 강세** → 분리동사 (보통 구체적/물리적 의미)
> - **어근에 강세** → 비분리동사 (보통 추상적/비유적 의미)

---

**1. door- (분리: 계속/관통 vs 비분리: 통과/관철)**

| | 분리동사 (DOORgaan) | 비분리동사 (doorZOEken) |
|---|---|---|
| **강세** | 접두사 **door**에 강세 | 어근에 강세 |
| **의미** | 구체적: 계속하다, 물리적으로 통과 | 추상적: 전체를 관통, 완전히 수행 |
| **과거분사** | door**ge**gaan | doorzocht (ge- 없음) |

| 분리동사 | 뜻 | 비분리동사 | 뜻 |
|---------|-----|---------|-----|
| **door**gaan | 계속하다 | door**brengen** | (시간을) 보내다 |
| | We gaan door met de les. | | Hij doorbrengt de vakantie in Italië. |
| **door**lezen | 끝까지 읽다 | door**zoeken** | 수색하다 |
| | Lees het artikel **door**. | | De politie doorzoekt het huis. |
| **door**geven | 전달하다 | door**kruisen** | 가로지르다 |
| | Geef dit bericht **door**. | | De trein doorkruist het land. |
| **door**lopen | 계속 걷다 | door**lopen** | (과정을) 거치다 |
| | Loop gewoon **door**. | | Hij doorloopt een moeilijke fase. |
| **door**rijden | 계속 운전하다 | door**staan** | 견디다 |
| | Rij maar **door**. | | Zij doorstaat de beproeving. |
| **door**werken | 계속 일하다 | door**breken** | 돌파하다 |
| | Hij werkt tot laat **door**. | | De zon doorbreekt de wolken. |
| **door**komen | 통과하다/살아남다 | door**gronden** | 꿰뚫어보다 |
| | Ik kom de winter wel **door**. | | Zij doorgrondt het probleem. |
| **door**strepen | 줄을 긋다 | door**zien** | 간파하다 |
| | Streep het verkeerde antwoord **door**. | | Ik doorziet zijn bedoelingen. |

---

**2. over- (분리: 다시/넘어 vs 비분리: 전체에 걸쳐/초과)**

| | 분리동사 (OVERstappen) | 비분리동사 (overLEGgen) |
|---|---|---|
| **강세** | 접두사 **over**에 강세 | 어근에 강세 |
| **의미** | 구체적: 물리적 이동, 다시 함 | 추상적: 전체에 걸침, 초과, 숙고 |
| **과거분사** | over**ge**stapt | overlegd (ge- 없음) |

| 분리동사 | 뜻 | 비분리동사 | 뜻 |
|---------|-----|---------|-----|
| **over**stappen | 환승하다 | over**leggen** | 상의하다 |
| | Stap bij Utrecht **over**. | | We overleggen morgen. |
| **over**komen | 건너오다 | over**komen** | (인상을) 주다/발생하다 |
| | Kom je morgen **over**? | | Hij overkomt als aardig. |
| **over**gaan | 넘어가다/울리다 | over**leven** | 살아남다 |
| | De telefoon gaat **over**. | | Zij overleeft de ziekte. |
| **over**doen | 다시 하다 | over**drijven** | 과장하다 |
| | Ik doe het examen **over**. | | Hij overdrijft altijd. |
| **over**nemen | 인수하다 | over**tuigen** | 설득하다 |
| | Hij neemt het bedrijf **over**. | | Zij overtuigt de jury. |
| **over**houden | 남기다/남다 | over**wegen** | 고려하다 |
| | Ik houd geld **over**. | | Ik overweeg een andere baan. |
| **over**lopen | (물이) 넘치다 | over**winnen** | 극복하다/이기다 |
| | Het bad loopt **over**. | | Zij overwint haar angst. |
| **over**steken | 건너다 | over**treden** | 위반하다 |
| | Steek de straat voorzichtig **over**. | | Hij overtreedt de wet. |
| **over**schrijven | 옮겨 쓰다 | over**vallen** | 습격하다 |
| | Schrijf de tekst **over**. | | De dief overvalt de bank. |
| **over**maken | 송금하다 | over**handigen** | 전달하다(격식) |
| | Ik maak het geld **over**. | | Zij overhandigt de prijs. |
