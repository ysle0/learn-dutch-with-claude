# 비분리동사 (Onscheidbare werkwoorden)

**be-, er-, ge-, her-, mis-, ont-, ver-** 접두사는 절대 분리되지 않으며, 과거분사에서도 **ge-를 붙이지 않습니다**.

> **핵심 규칙:** 비분리 전철은 강세가 없고(동사 어근에 강세), 과거분사에 ge-가 붙지 않습니다.
> ```
> betalen → betaald (✅)    (×gebetaald)
> vergeten → vergeten (✅)  (×gevergeten)
> ```

#### 비분리 전철 완전 목록

---

**1. be- (타동사화, 대상에 영향을 미침)**

> **의미 변화:** 가장 중요한 기능은 **자동사를 타동사로** 만드는 것입니다. 또한 동작이 대상 전체에 영향을 미친다는 뉘앙스를 줍니다.
> - 타동사화: 전치사가 필요했던 동사를 직접목적어를 취하는 타동사로 변환
>   ```
>   kijken naar → bekijken    (보다 → 살펴보다)
>   spreken over → bespreken  (말하다 → 논의하다)
>   stijgen op → bestijgen    (오르다 → 등반하다)
>   ```
> - 대상에 영향: 동작이 대상을 완전히 감싸거나 영향을 미침
>   ```
>   schrijven (쓰다) → beschrijven (묘사하다 — 대상을 글로 감쌈)
>   ```

| 원형 | 뜻 | 현재시제 | 과거분사 |
|------|-----|---------|---------|
| **be**talen | 지불하다 | Ik betaal de rekening. | betaald |
| **be**grijpen | 이해하다 | Ik begrijp het. | begrepen |
| **be**zoeken | 방문하다 | We bezoeken het museum. | bezocht |
| **be**schrijven | 묘사하다 | Hij beschrijft de situatie. | beschreven |
| **be**slissen | 결정하다 | Zij beslist morgen. | beslist |
| **be**danken | 감사하다 | Ik bedank je voor je hulp. | bedankt |
| **be**denken | 고안하다/생각해내다 | Wie heeft dit bedacht? | bedacht |
| **be**ginnen | 시작하다 | De les begint om 9 uur. | begonnen |
| **be**horen | ~에 속하다 | Dat behoort tot het verleden. | behoord |
| **be**loven | 약속하다 | Ik beloof het je. | beloofd |
| **be**oordelen | 평가하다 | De jury beoordeelt de kandidaten. | beoordeeld |
| **be**reiken | 도달하다 | We bereiken de top. | bereikt |
| **be**schermen | 보호하다 | De muur beschermt de stad. | beschermd |
| **be**staan | 존재하다 | Dat woord bestaat niet. | bestaan |

**2. ver- (변화, 강화, 잘못됨, 이동)**

> **의미 변화:** 매우 다양한 의미를 가진 전철입니다. 크게 네 가지 기능이 있습니다:
> - **변화:** 상태의 변화 (anderen → **ver**anderen = 바꾸다)
> - **강화:** 의미를 강하게 함 (groot → **ver**groten = 확대하다)
> - **잘못됨:** 실수·부정적 행위 (kijken → **ver**kijken = 잘못 보다)
>   ```
>   spreken → verspreken (zich) = 말실수하다
>   slapen → verslapen (zich) = 늦잠 자다
>   schrijven → verschrijven (zich) = 잘못 쓰다
>   ```
> - **이동·멀어짐:** 떠나거나 사라짐 (trekken → **ver**trekken = 출발하다)

| 원형 | 뜻 | 현재시제 | 과거분사 |
