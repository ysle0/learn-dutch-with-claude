# 수동태를 사용하는 경우와 피하는 경우

**수동태를 사용하는 경우:**
- 행위자가 불명확하거나 중요하지 않을 때: *De auto is gestolen.* (차가 도둑맞았다.)
- 공식적/학술적 글쓰기: *Het onderzoek werd uitgevoerd in 2023.* (연구는 2023년에 수행되었다.)
- 규칙/지시: *Roken is hier niet toegestaan.* (여기서 흡연은 허용되지 않습니다.)
- 뉴스/보도: *De minister werd ondervraagd.* (장관이 심문받았다.)

**수동태를 피하는 경우 (능동태가 자연스러운 경우):**
- 일상 대화: *Mijn vriend heeft me gebeld.* (수동태 X)
- 행위자가 명확할 때: *Ik heb het boek gelezen.* (수동태 X)
- 간결한 표현이 필요할 때

# 수동태 + 조동사 (Modal verbs)

조동사와 함께 수동태를 사용할 수 있습니다: **조동사 + 과거분사 + worden**

```
Het huis moet worden schoongemaakt.   (집이 청소되어야 한다.)
De auto kan worden gerepareerd.       (차가 수리될 수 있다.)
Dit formulier moet worden ingevuld.   (이 양식이 작성되어야 한다.)
De patiënt mag niet worden gestoord.  (환자가 방해받아서는 안 된다.)
De brief hoeft niet te worden verstuurd. (편지가 발송될 필요는 없다.)
Het probleem zal snel worden opgelost.   (문제가 빨리 해결될 것이다.)
```

> **주의:** 조동사 수동태에서 worden은 문장 끝에 옵니다.
