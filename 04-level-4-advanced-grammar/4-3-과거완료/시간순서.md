# 시간순서 다이어그램

과거완료는 두 과거 사건의 순서를 명확히 합니다:

```
시간 흐름 →

[과거완료]                    [과거시제]                 [현재]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
먼저 일어난 일                  나중에 일어난 일            지금
(had gegeten)                 (hij kwam)
(was vertrokken)              (ik aankwam)
```

---

# 과거완료 연습 대화

```
A: Waarom was je gisteren zo moe?
   (어제 왜 그렇게 피곤했어?)
B: Ik had de hele nacht niet geslapen.
   (밤새 잠을 못 잤거든.)
A: Hoe kwam dat?
   (왜 그랬는데?)
B: Ik had te veel koffie gedronken.
   (커피를 너무 많이 마셨어.)
A: Had je niet eerder moeten stoppen?
   (더 일찍 그만뒀어야 하지 않았어?)
B: Ja, maar ik had een deadline. Nadat ik mijn rapport
   had afgemaakt, was het al ochtend.
   (응, 하지만 마감이 있었어. 보고서를 끝낸 후에 이미 아침이었어.)
```

# 자주 함께 쓰이는 접속사/부사

| 접속사/부사 | 뜻 | 예시 |
|-----------|-----|------|
| **voordat** | ~하기 전에 | Ik had al gegeten **voordat** hij kwam. |
| **nadat** | ~한 후에 | **Nadat** ik had gegeten, ging ik naar bed. |
| **toen** | ~했을 때 | **Toen** ik aankwam, was zij al vertrokken. |
| **al** | 이미 | Hij **had** het **al** gedaan. |
| **nog niet** | 아직 ~않다 | Ik had het **nog niet** afgemaakt. |
| **net** | 방금 | Ik had het **net** gedaan. |
| **nooit (eerder)** | (이전에) 한번도 ~않다 | Ik had dat **nooit eerder** meegemaakt. |

# 활용표

| 인칭 | hebben 동사 (예: werken) | zijn 동사 (예: gaan) |
|------|------------------------|---------------------|
| ik | **had** gewerkt | **was** gegaan |
| jij | **had** gewerkt | **was** gegaan |
| hij/zij/het | **had** gewerkt | **was** gegaan |
| wij | **hadden** gewerkt | **waren** gegaan |
| jullie | **hadden** gewerkt | **waren** gegaan |
| zij (복수) | **hadden** gewerkt | **waren** gegaan |
