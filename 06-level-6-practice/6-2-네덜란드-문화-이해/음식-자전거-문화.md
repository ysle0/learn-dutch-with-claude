# 네덜란드 음식 문화

| 음식 | 설명 |
|------|------|
| **Stamppot** | 감자와 채소를 으깬 전통 요리 (rookworst와 함께) |
| **Erwtensoep** (snert) | 걸쭉한 완두콩 수프 (겨울 대표 음식) |
| **Haring** | 생청어 — 머리를 뒤로 젖히고 꼬리를 잡고 먹음 |
| **Stroopwafel** | 시럽이 든 와플 쿠키 |
| **Bitterballen** | 둥근 크로켓 (맥주 안주) |
| **Kaas** | 치즈! 하우다(Gouda), 에이담(Edam) 등 |
| **Oliebollen** | 기름에 튀긴 도넛 (새해 전통) |
| **Pannenkoeken** | 네덜란드식 팬케이크 (크고 얇음) |
| **Broodje kroket** | 크로켓 샌드위치 |
| **Drop** | 짠맛 감초 사탕 (외국인은 호불호가 갈림!) |

# 자전거 문화 (Fietscultuur)

네덜란드는 세계적인 자전거 나라입니다.

- **fiets**: 자전거
- **fietsen**: 자전거를 타다
- **fietspad**: 자전거 도로
- **fietsenrek**: 자전거 거치대
- **OV-fiets**: 공공 임대 자전거

```
Ik ga met de fiets naar mijn werk.
(나는 자전거로 출근한다.)

Kijk uit! Er komt een fietser aan!
(조심해! 자전거가 온다!)
```

> 네덜란드에는 약 2,300만 대의 자전거가 있습니다 — 인구(약 1,800만)보다 많습니다!
