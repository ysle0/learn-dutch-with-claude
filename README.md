# 네덜란드어 학습 커리큘럼 (Learn Dutch with Claude)

> 기초부터 문법, 관용구까지 — 체계적인 네덜란드어 완전 정복 과정

---

## 커리큘럼 개요

| 레벨 | 주제 | 목표 |
|------|------|------|
| **Level 1** | 기초 입문 | 알파벳, 발음, 인사, 자기소개 |
| **Level 2** | 필수 문법 기초 | 관사, 명사, 기본 어순, 현재시제 |
| **Level 3** | 중급 문법 | 과거시제, 접속사, 전치사, 비교급 |
| **Level 4** | 고급 문법 | 수동태, 관계절, 가정법, 조건문 |
| **Level 5** | 관용구와 표현 | 일상 관용구, 속담, 구어체 표현 |
| **Level 6** | 실전과 문화 | 실생활 회화, 문화적 맥락, 종합 연습 |

---

## 학습 순서

```
Level 1 → Level 2 → Level 3 → Level 4 → Level 5 → Level 6
(기초)    (문법기초)  (중급문법)  (고급문법)  (관용구)    (실전)
```
