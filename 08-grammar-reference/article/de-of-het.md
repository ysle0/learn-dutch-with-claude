### De of Het — 네덜란드어 관사 선택 가이드

네덜란드어에는 두 가지 정관사가 있습니다: **de** (통성)와 **het** (중성).
어떤 명사에 어떤 관사를 쓰는지는 네덜란드어 학습에서 가장 어려운 부분 중 하나입니다.

---

### DE 관사 (통성)

**de**를 사용하는 경우:

| 규칙 | 예시 |
|------|------|
| 복수형은 모두 de | de boeken, de huizen, de kinderen |
| 사람/직업 명사 | de man, de vrouw, de leraar |
| 채소와 과일 | de aardappel, de appel, de wortel |
| 교통수단 | de auto, de fiets, de trein |
| 영어 차용어 | de computer, de printer, de file |
| 산맥 이름 | de Alpen, de Andes |
| 강 이름 | de Rijn, de Maas |
| -ing으로 끝나는 단어 | de woning, de vergadering |

> **예외:** het vliegtuig (비행기), het weekend (주말)

---

### HET 관사 (중성)

**het**을 사용하는 경우:

| 규칙 | 예시 |
|------|------|
| 축소사 (-je 접미사) | het meisje, het hondje, het boekje |
| 동사의 명사화 | het eten, het drinken, het lezen |
| 스포츠와 게임 | het voetbal, het tennis, het schaak |
| 언어 이름 | het Nederlands, het Engels |
| 금속 이름 | het goud, het zilver, het ijzer |
| -isme으로 끝나는 단어 | het kapitalisme, het toerisme |

---

### 확인 방법

관사가 불확실할 때는 [woordenlijst.org](https://woordenlijst.org)에서 검색하여 확인할 수 있습니다.

> **팁:** 새로운 명사를 배울 때 항상 관사와 함께 외우세요!
> 예: "boek"이 아니라 "het boek"으로 기억하세요.
