# Komen en Gaan

"오다"와 "가다"를 뜻하는 기본 이동 동사입니다.

## 활용표

### Komen (오다)

| 시제 | 형태 |
|------|------|
| 현재형 | ik kom / jij komt / hij komt / wij komen |
| 과거형 | ik kwam / wij kwamen |
| 완료형 | ik **ben** gekomen |

### Gaan (가다)

| 시제 | 형태 |
|------|------|
| 현재형 | ik ga / jij gaat / hij gaat / wij gaan |
| 과거형 | ik ging / wij gingen |
| 완료형 | ik **ben** gegaan |

> **참고:** 두 동사 모두 완료형에서 **zijn**을 사용합니다.

---

## 관점 규칙

네덜란드어에서는 **목적지에 있는 사람의 관점**으로 동사를 선택합니다.

### Komen 사용 (상대방이 목적지에 있을 때)

- Ik **kom** naar je toe. (나 너한테 갈게.) — 상대방이 있는 곳으로 이동
- Kom je morgen? (내일 올 거야?) — 말하는 사람이 목적지에 있음
- Ze **komen** bij ons eten. (그들이 우리 집에 먹으러 온다.)

### Gaan 사용 (제3의 장소로 이동할 때)

- Ik **ga** naar de supermarkt. (나 슈퍼마켓에 간다.) — 제3의 장소
- We **gaan** naar het strand. (우리 해변에 간다.)
- Hij **gaat** naar school. (그는 학교에 간다.)

---

## Naar huis vs. Thuis

| 표현 | 의미 | 사용 |
|------|------|------|
| naar huis | 집으로 (이동) | Ik ga **naar huis**. (집에 간다.) |
| thuis | 집에 (위치) | Ik ben **thuis**. (집에 있다.) |

- Ik ga naar huis. (나는 집에 간다.) — 이동 중
- Ik kom thuis om 6 uur. (나는 6시에 집에 도착한다.) — 도착
- Ik ben thuis. (나는 집에 있다.) — 상태
