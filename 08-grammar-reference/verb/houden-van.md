# Houden van

"좋아하다"를 뜻하는 네덜란드어 핵심 동사입니다.

## 활용표

| 시제 | 형태 |
|------|------|
| 현재형 | ik hou(d) / jij houdt / hij houdt / wij houden |
| 과거형 | ik hield / wij hielden |
| 완료형 | ik heb gehouden |

> **참고:** 1인칭 단수에서 **hou**와 **houd** 모두 사용 가능합니다.

---

## 문장 구조

**van**의 위치는 항상 고정되어 있고, 동사만 이동합니다.

- Ik **hou van** chocolade. (나는 초콜릿을 좋아한다.)
- **Hou** jij **van** koffie? (너는 커피를 좋아하니?)
- Ik heb altijd **van** muziek **gehouden**. (나는 항상 음악을 좋아했다.)

---

## 강도 표현

약한 것에서 강한 것 순서입니다.

| 표현 | 예문 |
|------|------|
| veel | Ik hou veel van je. |
| erg | Ik hou erg van je. |
| heel veel | Ik hou heel veel van je. |
| enorm veel | Ik hou enorm veel van je. |

---

## "hou ervan om..." 구문

동사나 긴 표현과 함께 사용할 때 **ervan om**을 씁니다.

- Ik **hou ervan om** te koken. (나는 요리하는 것을 좋아한다.)
- Ik **hou ervan om** 's ochtends te wandelen. (나는 아침에 산책하는 것을 좋아한다.)
- Hij **houdt ervan om** laat op te blijven. (그는 늦게까지 깨어 있는 것을 좋아한다.)

---

## "houden" 단독 의미

**van** 없이 쓰이면 다른 뜻을 가집니다.

| 의미 | 예문 |
|------|------|
| 유지하다 | Hou dit vast. (이것을 잡고 있어.) |
| 잡다 | Hij houdt de deur open. (그가 문을 잡고 있다.) |
| 개최하다 | We houden een feest. (우리는 파티를 연다.) |
| 동물 기르다 | Ze houdt kippen. (그녀는 닭을 기른다.) |
