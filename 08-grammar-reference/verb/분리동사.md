# 분리동사 (Scheidbare werkwoorden)

기본 동사에 전치사가 붙어 새로운 의미를 만드는 동사입니다.

## 기본 원리

**기본 동사 + 전치사 = 새로운 의미**

- leggen (놓다) + uit = **uitleggen** (설명하다)
- maken (만들다) + op = **opmaken** (화장하다 / 다 쓰다)

---

## Geven 예시 표

| 분리동사 | 의미 |
|----------|------|
| opgeven | 포기하다 |
| uitgeven | 지출하다 / 출판하다 |
| afgeven | 전달하다 / 내뿜다 |
| overgeven | 토하다 / 항복하다 |
| meegeven | 함께 주다 |
| teruggeven | 돌려주다 |
| weggeven | 나눠 주다 |
| doorgeven | 전달하다 |

---

## 활용 규칙

### 1. 부정사 (조동사와 함께) — 붙여 쓴다

- Kan je dit **uitleggen**?

### 2. 현재형 / 과거형 — 전치사가 문장 끝으로

- Ik **leg** het **uit**. (나는 그것을 설명한다.)
- Ik **legde** het **uit**. (나는 그것을 설명했다.)

### 3. 완료형 — ge-가 중간에

- Ik heb het **uitgelegd**. (나는 그것을 설명했다.)

### 4. 종속절 — 다시 붙여 쓴다

- ...omdat ik het **uitleg**. (...내가 그것을 설명하기 때문에.)

---

## 분리 확률 (전치사별)

| 전치사 | 분리 확률 |
|--------|-----------|
| op / uit / af / mee / weg / terug | 100% |
| aan | 97% |
| door | 85% |
| voor | 77% |

---

## 강세 패턴

분리동사와 비분리동사는 **강세 위치**로 구별합니다.

| 유형 | 강세 | 예시 |
|------|------|------|
| 분리동사 | **전치사**에 강세 | **VOOR**stellen (소개하다) |
| 비분리동사 | **기본 동사**에 강세 | voor**STEL**len (상상하다) |
