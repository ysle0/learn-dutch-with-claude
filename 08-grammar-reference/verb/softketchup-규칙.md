# Softketchup 규칙 (t/d)

과거형에서 **-te/-ten** 또는 **-de/-den**을 쓸지 결정하는 규칙입니다.

## 규칙

1. 동사의 1인칭 단수 현재형을 구합니다 (ik-vorm).
2. **-en** 앞의 마지막 글자를 확인합니다.
3. 그 글자가 **softketchup**에 있으면 → **-te/-ten**
4. 없으면 → **-de/-den**

### Softketchup 글자

> **s, o(f), t, k, e(ch), t(ch), u(p)** → **s, f, t, k, ch, p**

외우는 방법: **soft ketchup** 안에 있는 자음 = **s, f, t, k, ch, p**

---

## 적용 예시

### -te/-ten (softketchup에 해당)

| 부정사 | ik-vorm | 마지막 글자 | 과거형 |
|--------|---------|-------------|--------|
| fietsen | fiets | **s** ✓ | fiet**ste** / fiet**sten** |
| werken | werk | **k** ✓ | werk**te** / werk**ten** |
| stoppen | stop | **p** ✓ | stop**te** / stop**ten** |
| lachen | lach | **ch** ✓ | lach**te** / lach**ten** |
| straffen | straf | **f** ✓ | straf**te** / straf**ten** |

### -de/-den (softketchup에 해당 안 됨)

| 부정사 | ik-vorm | 마지막 글자 | 과거형 |
|--------|---------|-------------|--------|
| leven | leef | **v→f**... | leef**de** / leef**den** |
| bouwen | bouw | **w** ✗ | bouw**de** / bouw**den** |
| reizen | reis | **z→s**... | reis**de** / reis**den** |
| spelen | speel | **l** ✗ | speel**de** / speel**den** |

---

## 예외

모든 **불규칙 동사**는 이 규칙이 적용되지 않습니다. 별도로 외워야 합니다.

- gaan → ging (✘ gaade)
- komen → kwam (✘ komde)
- zijn → was (✘ zijnde)

---

## 추가: X도 포함

**x**로 끝나는 동사도 **-te/-ten**을 사용합니다.

| 동사 | 과거형 |
|------|--------|
| detoxen | detox**te** |
| faxen | fax**te** |
| fixen | fix**te** |
| mixen | mix**te** |
| relaxen | relax**te** |
| waxen | wax**te** |

> **기억법:** softketchup + **x** = **-te/-ten**
