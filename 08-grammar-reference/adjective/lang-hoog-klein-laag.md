# Lang, Hoog, Klein, Laag

네덜란드어에서 크기와 높이를 나타내는 형용사는 **사람**에게 쓸 때와 **사물**에게 쓸 때 다르게 사용됩니다. 한국어와 다른 부분이 많으므로 주의가 필요합니다.

> 출처: [Zichtbaar Nederlands - Adjective](https://zichtbaarnederlands.nl/en/adjective/)

---

### 사람에게 사용하는 형용사

| 네덜란드어 | 한국어 | 설명 |
|---|---|---|
| klein | 키가 작은 | 사람의 키가 작을 때 |
| lang | 키가 큰 | 사람의 키가 클 때 |
| groot | 키가 큰 / 큰 | lang과 비슷하게 사용 가능 |

### 예문 (사람)

- Mijn broer is **lang**. (내 형은 키가 크다.)
- Zij is **klein**. (그녀는 키가 작다.)
- Hij is een **grote** man. (그는 큰 남자이다.)

---

### 사물에게 사용하는 형용사

| 네덜란드어 | 한국어 | 설명 |
|---|---|---|
| laag | 낮은 | 사물의 높이가 낮을 때 |
| hoog | 높은 | 사물의 높이가 높을 때 |
| lang | 긴 | 사물의 길이가 길 때 |
| kort | 짧은 | 사물의 길이가 짧을 때 |

### 예문 (사물)

- De tafel is **laag**. (테이블이 낮다.)
- Het gebouw is **hoog**. (건물이 높다.)
- De rivier is **lang**. (강이 길다.)
- De straat is **kort**. (거리가 짧다.)

---

### 주의: lang의 두 가지 의미

**lang**은 사람과 사물에 모두 쓰이지만 의미가 다릅니다.

| 대상 | 의미 | 예문 |
|---|---|---|
| 사람 | 키가 큰 | Hij is **lang**. (그는 키가 크다.) |
| 사물 | 긴 | De weg is **lang**. (길이 길다.) |

---

### 요약 비교표

| 한국어 | 사람 | 사물 |
|---|---|---|
| 작은 / 낮은 | klein | laag |
| 큰 / 높은 | lang / groot | hoog |
| 긴 | - | lang |
| 짧은 | - | kort |
