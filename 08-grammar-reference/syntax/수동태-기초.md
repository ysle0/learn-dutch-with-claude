# 수동태 기초 (Passive Voice)

네덜란드어 수동태의 기본 구조를 알아봅니다.

## 능동태 vs 수동태

능동태는 **행위자**가 주어이고, 수동태는 **행위의 대상**이 주어입니다.

| 구분 | 예시 |
|------|------|
| 능동 | *De hond bijt de jongen.* (개가 소년을 문다) |
| 수동 | *De jongen wordt gebeten.* (소년이 물린다) |

## 수동태 만들기: 구조 변환

능동태를 수동태로 바꾸는 단계:

1. 능동태의 **목적어** → 수동태의 **주어**
2. 능동태의 **동사** → **worden/zijn + 과거분사**
3. 능동태의 **주어** → **door + 행위자** (선택적)

## 현재 수동태: worden + 과거분사

**worden**의 현재형과 과거분사를 결합합니다.

| 인칭 | worden 현재형 |
|------|--------------|
| ik | word |
| jij | wordt |
| hij/zij | wordt |
| wij/jullie/zij | worden |

- *De jongen **wordt** gebeten (door de hond).* — 소년이 (개에게) 물립니다.
- *De brieven **worden** geschreven.* — 편지들이 쓰여집니다.

## 과거 수동태: werd(en) + 과거분사

**worden**의 과거형을 사용합니다.

| 인칭 | worden 과거형 |
|------|--------------|
| ik/hij/zij | werd |
| wij/jullie/zij | werden |

- *De jongen **werd** gebeten.* — 소년이 물렸습니다.
- *De brieven **werden** geschreven.* — 편지들이 쓰여졌습니다.

## 완료 수동태: zijn + 과거분사

완료형에서는 **worden** 대신 **zijn**을 사용합니다.

| 인칭 | zijn 현재형 |
|------|------------|
| ik | ben |
| jij | bent |
| hij/zij | is |
| wij/jullie/zij | zijn |

- *De jongen **is** gebeten.* — 소년이 물렸습니다 (현재완료).
- *De brieven **zijn** geschreven.* — 편지들이 쓰여졌습니다 (현재완료).

> **주의:** 완료형에서 ~~*is geworden gebeten*~~이 아니라 *is gebeten*입니다. **worden**이 사라집니다.

## Door로 행위자 표시

행위자를 밝히고 싶을 때 **door**를 사용합니다. 생략 가능합니다.

- *Het boek wordt gelezen **door de studenten**.* — 그 책은 학생들에 의해 읽힙니다.
- *De taart is gebakken **door mijn moeder**.* — 그 케이크는 어머니에 의해 구워졌습니다.

> 대부분의 수동태 문장에서 행위자는 생략됩니다. 행위자가 불분명하거나 중요하지 않을 때 수동태를 사용합니다.
