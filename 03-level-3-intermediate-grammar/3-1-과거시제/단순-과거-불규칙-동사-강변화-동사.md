# 단순 과거: 불규칙 동사 (강변화 동사)

불규칙 동사는 모음이 변합니다. 암기가 필요합니다.

> **패턴 인식 팁**: 불규칙 동사도 어느 정도 패턴이 있습니다. 비슷한 모음 변화를 하는 동사들을 묶어서 외우면 효율적입니다.

**기본 필수 불규칙 동사:**

**모음 변화 패턴별 분류:**

| 패턴 | 원형 → 과거 | 동사 예시 |
|------|-----------|----------|
| **ij → ee** | schrijven → schreef | schrijven, rijden, blijven, kijken |
| **i → o** | drinken → dronk | drinken, zingen, springen, winnen |
| **e → a** | spreken → sprak | spreken, breken, nemen, stelen |
| **ie → oo** | kiezen → koos | kiezen, sluiten, bieden, vliegen |
| **a → ie** | slapen → sliep | slapen, laten, vallen, houden |
| **oe → a** | doen → deed | doen (특수) |

**불규칙 동사 과거시제 예문:**

```
Ik was gisteren ziek.                          (나는 어제 아팠다.)
Zij had geen geld bij zich.                    (그녀는 돈을 가지고 있지 않았다.)
Wij gingen vorig jaar naar Spanje.             (우리는 작년에 스페인에 갔다.)
De kinderen kwamen laat thuis.                 (아이들이 늦게 집에 왔다.)
Hij zag een mooie vogel in de tuin.            (그는 정원에서 아름다운 새를 보았다.)
Zij gaf mij een cadeau voor mijn verjaardag.   (그녀는 내 생일에 선물을 줬다.)
Ik nam de trein naar Den Haag.                 (나는 덴하흐로 기차를 탔다.)
We spraken Nederlands tijdens de les.          (우리는 수업 중에 네덜란드어를 말했다.)
Zij schreef een lange brief aan haar moeder.   (그녀는 어머니께 긴 편지를 썼다.)
Ik las het hele boek in één dag.               (나는 그 책을 하루 만에 다 읽었다.)
De hond liep snel door het park.               (개가 공원을 빨리 뛰어갔다.)
Het kind viel van de fiets.                    (아이가 자전거에서 떨어졌다.)
Wij zwommen in de Noordzee.                    (우리는 북해에서 수영했다.)
Hij vergat zijn sleutels thuis.                (그는 열쇠를 집에 두고 왔다.)
De winkel sloot om zes uur.                    (가게가 6시에 문을 닫았다.)
```
