#### 분리동사의 과거분사

ge-가 분리 접두사와 어간 사이에 들어갑니다:

| 원형 | 과거분사 | 뜻 |
|------|---------|-----|
| opbellen | **op-ge-beld** | 전화하다 |
| aankomen | **aan-ge-komen** | 도착하다 |
| meenemen | **mee-ge-nomen** | 가져가다 |
| opruimen | **op-ge-ruimd** | 정리하다 |
| uitnodigen | **uit-ge-nodigd** | 초대하다 |
| thuiskomen | **thuis-ge-komen** | 귀가하다 |
| binnenkomen | **binnen-ge-komen** | 들어오다 |
| afwassen | **af-ge-wassen** | 설거지하다 |
| opstaan | **op-ge-staan** | 일어나다 |
| meebrengen | **mee-ge-bracht** | 가져오다 |
| uitzetten | **uit-ge-zet** | 끄다 |
| aantrekken | **aan-ge-trokken** | 입다(옷) |
| uitdoen | **uit-ge-daan** | 끄다/벗다 |

```
Ik heb mijn moeder opgebeld.           (나는 어머니께 전화했다.)
De trein is om drie uur aangekomen.    (기차가 3시에 도착했다.)
Heb je je jas aangetrokken?            (외투 입었어?)
Ik heb de keuken opgeruimd.            (나는 부엌을 정리했다.)
We hebben onze vrienden uitgenodigd.   (우리는 친구들을 초대했다.)
```
