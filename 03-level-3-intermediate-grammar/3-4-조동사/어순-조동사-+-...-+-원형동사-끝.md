# 어순: 조동사 + ... + 원형동사 (끝)

```
Ik   kan   goed   Nederlands   spreken.
주어  조동사  부사   목적어       원형동사
나는  할 수 있다  잘  네덜란드어를  말하다.
```

**다양한 어순 예시:**

```
평서문:   Ik moet morgen vroeg opstaan.
          (나는 내일 일찍 일어나야 한다.)

의문문:   Moet je morgen vroeg opstaan?
          (내일 일찍 일어나야 해?)

부정문:   Ik kan vanavond niet komen.
          (오늘 저녁에 올 수 없어.)

종속절:   ...omdat ik morgen vroeg moet opstaan.
          (...내일 일찍 일어나야 하기 때문에.)
          → 종속절에서는 조동사도 끝으로!
```
