# 조동사 활용표

#### kunnen (~할 수 있다: 능력, 가능성)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | kan | kon |
| jij | kunt/kan | kon |
| u | kunt/kan | kon |
| hij/zij/het | kan | kon |
| wij | kunnen | konden |
| jullie | kunnen | konden |
| zij | kunnen | konden |

```
Ik kan goed zwemmen.                     (나는 수영을 잘 할 수 있다.)
Kun je me helpen?                         (나 좀 도와줄 수 있어?)
Hij kan heel goed koken.                  (그는 요리를 아주 잘 한다.)
Zij konden niet komen vanwege het weer.   (날씨 때문에 올 수 없었다.)
Kun je een beetje Nederlands spreken?     (네덜란드어 조금 할 수 있어?)
Dat kan niet waar zijn!                   (그게 사실일 리 없어!)
Ik kon gisteren niet slapen.              (어제 잠을 잘 수 없었다.)
Je kunt hier parkeren.                    (여기 주차할 수 있어요.)
```

> **뉘앙스**: kunnen은 능력("~할 줄 알다")과 가능성("~할 수 있다") 두 가지 의미가 있습니다.
> - 능력: Ik kan pianospelen. (피아노를 칠 줄 안다.)
> - 가능성: Het kan morgen regenen. (내일 비가 올 수 있다.)

#### mogen (~해도 된다: 허가, 허락)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | mag | mocht |
| jij | mag | mocht |
| u | mag | mocht |
| hij/zij/het | mag | mocht |
| wij | mogen | mochten |
| jullie | mogen | mochten |
| zij | mogen | mochten |

```
Mag ik hier zitten?                       (여기 앉아도 될까요?)
Je mag hier niet roken.                   (여기서 흡연하면 안 됩니다.)
Mag ik iets vragen?                       (뭐 좀 물어봐도 될까요?)
De kinderen mogen niet laat opblijven.    (아이들은 늦게까지 안 자면 안 된다.)
Mocht ik van mijn ouders naar het feest?  (부모님이 파티에 가도 된다고 했어?)
U mag hier niet parkeren.                 (여기 주차하시면 안 됩니다.)
Mag ik even langs?                        (잠깐 지나가도 될까요?)
Je mag best trots zijn op jezelf.         (스스로 자랑스러워해도 돼.)
```

> **mogen vs kunnen**: mogen은 **허가** (규칙상/사회적으로 괜찮은지), kunnen은 **능력/가능성** (물리적으로/능력상 가능한지)
> - Mag ik hier zwemmen? (여기서 수영해도 돼요?) — 허가를 묻는 것
> - Kan ik hier zwemmen? (여기서 수영할 수 있어요?) — 물리적으로 가능한지

#### moeten (~해야 한다: 의무, 필요)

| 인칭 | 현재형 | 과거형 |
|------|--------|--------|
| ik | moet | moest |
| jij | moet | moest |
| u | moet | moest |
| hij/zij/het | moet | moest |
| wij | moeten | moesten |
| jullie | moeten | moesten |
| zij | moeten | moesten |

```
Ik moet morgen vroeg opstaan.             (나는 내일 일찍 일어나야 한다.)
Je moet je huiswerk maken.                (숙제를 해야 해.)
We moeten nu vertrekken.                  (지금 출발해야 해.)
Hij moest gisteren overwerken.            (그는 어제 야근해야 했다.)
Moet ik een afspraak maken?               (예약을 해야 하나요?)
Je moet niet zo veel suiker eten.         (설탕을 그렇게 많이 먹으면 안 돼.)
Zij moest naar de dokter.                 (그녀는 병원에 가야 했다.)
Moet je vanavond werken?                  (오늘 저녁에 일해야 해?)
U moet rechtsaf slaan bij het stoplicht.  (신호등에서 우회전하셔야 합니다.)
```

> **moeten의 다양한 강도**:
> - 강한 의무: Je **moet** je belasting betalen. (세금을 내야 한다.) — 법적 의무
> - 강한 권고: Je **moet** dit boek lezen! (이 책 꼭 읽어봐!) — 강력 추천
> - 추측: Het **moet** koud zijn buiten. (밖이 추울 것이다.) — ~임에 틀림없다
