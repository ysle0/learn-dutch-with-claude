# 종속절에서 동사가 여러 개인 경우

조동사나 완료시제가 포함된 종속절에서는 모든 동사가 절의 끝으로 갑니다:

```
① 조동사 + 원형동사:
   Ik weet dat hij goed Nederlands kan spreken.
   (나는 그가 네덜란드어를 잘 할 수 있다는 것을 안다.)
   → kan spreken (조동사 + 원형) 둘 다 끝으로

② 현재 완료:
   Ik weet dat hij een boek heeft gelezen.
   (나는 그가 책을 읽었다는 것을 안다.)
   → heeft gelezen (조동사 + 과거분사) 둘 다 끝으로

③ 조동사 + 완료:
   Ik denk dat hij het boek moet hebben gelezen.
   (나는 그가 그 책을 읽었어야 한다고 생각한다.)
   → moet hebben gelezen 모두 끝으로
```

> **한국어 화자에게 좋은 소식**: 종속절에서 동사가 끝으로 가는 것은 한국어의 "~한다고/~하기 때문에" 등과 비슷한 구조입니다. 한국어도 동사가 절의 끝에 오는 SOV 어순이므로, 이 부분은 한국어 화자에게 상대적으로 자연스럽습니다.
