# 종속절이 먼저 오는 경우

종속절이 문장 앞에 오면, 주절에서 **도치**가 일어납니다:

```
Omdat ik ziek ben,  blijf  ik  thuis.
종속절              동사   주어  나머지
왜냐하면 아프기 때문에, 있다  나는  집에.
```

> **도치 원칙**: 네덜란드어에서 동사는 항상 **주절의 두 번째 위치**에 옵니다. 종속절 전체가 첫 번째 위치를 차지하므로, 그 다음에 바로 동사가 와야 합니다.

**다양한 종속절 도치 예문:**

```
① Omdat het regent, neem ik een paraplu mee.
   (비가 오기 때문에, 우산을 가져간다.)
   → 종속절이 먼저 → 도치: neem + ik

② Als ik klaar ben, ga ik naar huis.
   (끝나면, 집에 갈 거야.)
   → 종속절이 먼저 → 도치: ga + ik

③ Toen ik in Nederland aankwam, sprak ik geen woord Nederlands.
   (네덜란드에 도착했을 때, 네덜란드어를 한 마디도 못 했다.)
   → 종속절이 먼저 → 도치: sprak + ik

④ Hoewel hij moe was, werkte hij de hele nacht door.
   (피곤했음에도 불구하고, 밤새 일했다.)
   → 종속절이 먼저 → 도치: werkte + hij

⑤ Voordat je vertrekt, moet je je kamer opruimen.
   (떠나기 전에, 방을 정리해야 해.)
   → 종속절이 먼저 → 도치: moet + je

⑥ Zodra de winkel opengaat, ga ik boodschappen doen.
   (가게가 열자마자, 장을 보러 갈 거야.)
   → 종속절이 먼저 → 도치: ga + ik

⑦ Sinds ik hier woon, heb ik veel vrienden gemaakt.
   (여기 산 이후로, 많은 친구를 사겼다.)
   → 종속절이 먼저 → 도치: heb + ik
```
