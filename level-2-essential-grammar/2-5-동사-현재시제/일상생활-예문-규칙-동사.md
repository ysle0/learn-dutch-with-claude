# 일상생활 예문 (규칙 동사)

| 예문 | 번역 |
|------|------|
| Ik **woon** in Amsterdam. | 나는 암스테르담에 살아. |
| Hij **werkt** bij een ziekenhuis. | 그는 병원에서 일해. |
| Wij **leren** Nederlands. | 우리는 네덜란드어를 배워. |
| Ze **leest** elke dag een boek. | 그녀는 매일 책을 읽어. |
| **Fiets** jij naar school? | 너는 자전거로 학교에 가니? |
| Jullie **studeren** medicijnen. | 너희는 의학을 공부해. |
| Ik **koop** boodschappen. | 나는 장을 봐. |
| Hij **schrijft** een brief. | 그는 편지를 써. |
| Wat **vind** jij ervan? | 너는 어떻게 생각해? |
| We **eten** om zes uur. | 우리는 6시에 먹어. |
| Zij **drinkt** graag koffie. | 그녀는 커피를 즐겨 마셔. |
| De kinderen **spelen** buiten. | 아이들이 밖에서 놀아. |
