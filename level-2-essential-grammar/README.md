## 2-1. 관사 (Lidwoorden)

네덜란드어에는 두 종류의 정관사가 있습니다. 한국어에는 관사 개념이 없기 때문에 한국어 화자에게 가장 어려운 부분 중 하나입니다. 영어의 the/a와 비슷하지만, 네덜란드어는 정관사가 **de**와 **het** 두 가지로 나뉘어 명사의 성(性)에 따라 구분해야 합니다.

### 정관사

| 관사 | 사용 | 예시 |
|------|------|------|
| **de** | 공통성 명사 (남성/여성) | de man (남자), de vrouw (여자), de tafel (테이블) |
| **het** | 중성 명사 | het kind (아이), het boek (책), het huis (집) |

### 부정관사

| 관사 | 사용 | 예시 |
|------|------|------|
| **een** | 모든 단수 명사 | een man (한 남자), een boek (한 책) |

부정관사 **een**은 발음할 때 보통 약하게 "은"처럼 발음됩니다. 숫자 "하나"를 강조할 때는 **één**으로 쓰고 강하게 발음합니다.

- Ik wil **een** kopje koffie. (커피 한 잔 원해요.) — 부정관사, 약하게 발음
- Ik wil **één** kopje koffie, niet twee. (커피 두 잔이 아니라 한 잔 원해요.) — 숫자, 강하게 발음

### 관사를 쓰지 않는 경우

영어와 마찬가지로 관사를 생략하는 경우가 있습니다:

- 직업을 말할 때: Ik ben **student**. (나는 학생이다.) — 단, 형용사가 붙으면 관사 필요: Ik ben **een goede** student.
- 국적을 말할 때: Hij is **Nederlander**. (그는 네덜란드인이다.)
- 셀 수 없는 명사 일반론: Ik drink **koffie**. (나는 커피를 마신다.)
- 복수 명사 일반론: **Kinderen** spelen graag. (아이들은 놀기를 좋아한다.)

> **핵심 포인트**:
> - 복수형은 항상 **de**를 사용합니다: de boeken (책들), de huizen (집들)
> - de 명사가 약 2/3, het 명사가 약 1/3입니다
> - 어떤 관사를 쓰는지 외우는 것이 중요합니다 — 규칙이 완벽하지 않으므로 단어와 함께 관사를 외우세요
> - 사전에서 새 명사를 찾을 때 항상 관사도 함께 확인하세요

### het 명사가 되는 패턴 (참고용)

- 축소형 (-je): het hondje (강아지), het meisje (소녀), het kopje (잔), het bloempje (꽃)
- 대부분의 ge- 접두사 명사: het gebouw (건물), het gevoel (느낌), het geluid (소리), het gezicht (얼굴), het gesprek (대화), het gerecht (요리)
- 대부분의 2음절 이상 명사로 be-, ge-, ver-, ont-로 시작하는 것: het begin (시작), het bezoek (방문), het verkeer (교통), het ontbijt (아침식사)
- 언어 이름: het Nederlands (네덜란드어), het Engels (영어), het Koreaans (한국어)
- 스포츠/게임: het voetbal (축구), het tennis (테니스), het schaak (체스)
- 금속: het goud (금), het zilver (은), het ijzer (철)
- 방위: het noorden (북쪽), het zuiden (남쪽), het oosten (동쪽), het westen (서쪽)

### de 명사가 되는 패턴 (참고용)

- 사람을 나타내는 명사 대부분: de leraar (선생님), de dokter (의사), de student (학생)
- -ing으로 끝나는 명사: de woning (주거), de vergadering (회의), de oefening (연습)
- -heid/-teit로 끝나는 명사: de vrijheid (자유), de universiteit (대학)
- -ie로 끝나는 명사: de politie (경찰), de fantasie (환상)
- -ij로 끝나는 명사: de bakkerij (빵집), de bibliotheek (도서관)

### 한국인이 자주 하는 실수

1. **관사를 생략하는 실수**: 한국어에는 관사가 없으므로 관사를 빼먹기 쉽습니다.
   - (X) Ik lees boek. → (O) Ik lees **een** boek. / Ik lees **het** boek.

2. **de와 het을 혼동하는 실수**: 단어마다 외워야 합니다.
   - (X) **de** huis → (O) **het** huis
   - (X) **het** tafel → (O) **de** tafel

3. **복수형에서 het을 쓰는 실수**: 복수는 항상 **de**입니다.
   - (X) **het** boeken → (O) **de** boeken
   - (X) **het** kinderen → (O) **de** kinderen

4. **직업 앞에 관사를 넣는 실수**: 네덜란드어에서 직업을 말할 때는 관사를 생략합니다.
   - (X) Ik ben **een** student. → (O) Ik ben student. (단, 형용사가 있으면 관사 사용)

### 연습 대화

```
A: Wat is dat?                      (그것이 뭐야?)
B: Dat is een boek.                 (그것은 책이야.)
A: Is het boek nieuw?               (그 책은 새 거야?)
B: Ja, het boek is nieuw.           (응, 그 책은 새 거야.)
A: En wat is dit?                   (그리고 이것은 뭐야?)
B: Dit is de krant van vandaag.     (이것은 오늘의 신문이야.)
A: Waar is de telefoon?             (전화기 어디 있어?)
B: De telefoon ligt op de tafel.    (전화기는 테이블 위에 있어.)
```

## 2-2. 명사의 복수형 (Meervoud)

네덜란드어 복수형은 크게 **-en**, **-s**, **-'s** 세 가지 패턴이 있습니다. 한국어의 "-들"처럼 간단하지 않고, 영어처럼 철자 변화가 따르기도 합니다. 네덜란드어 특유의 철자 규칙을 이해하면 복수형을 올바르게 쓸 수 있습니다.

### 주요 복수형 규칙

| 규칙 | 적용 조건 | 예시 |
|------|----------|------|
| **-en 추가** | 가장 일반적인 패턴 | boek → boek**en** (책들) |
| **-s 추가** | -el, -em, -en, -er, -je, -ie로 끝나는 단어 | tafel → tafel**s** (테이블들) |
| **-'s 추가** | -a, -i, -o, -u, -y로 끝나는 외래어 | auto → auto**'s** (자동차들) |

### -en 복수형 예시 (가장 일반적)

| 단수 | 복수 | 뜻 |
|------|------|-----|
| boek | boeken | 책들 |
| dag | dagen | 날들 |
| vraag | vragen | 질문들 |
| taal | talen | 언어들 |
| jaar | jaren | 년들 |
| dier | dieren | 동물들 |
| woord | woorden | 단어들 |
| stad | steden | 도시들 (불규칙!) |
| kind | kinderen | 아이들 (불규칙!) |
| land | landen | 나라들 |
| punt | punten | 점들 |
| brief | brieven | 편지들 (f→v) |

### -s 복수형 예시

| 단수 | 복수 | 뜻 |
|------|------|-----|
| tafel | tafels | 테이블들 |
| sleutel | sleutels | 열쇠들 |
| bodem | bodems | 바닥들 |

| tante | tantes | 이모/고모들 |

| 단수 | 복수 | 뜻 |
|------|------|-----|

| hobby | hobby's | 취미들 |
| paraplu | paraplu's | 우산들 |
| oma | oma's | 할머니들 |
| opa | opa's | 할아버지들 |

### 철자 변화에 주의

네덜란드어의 핵심 철자 규칙입니다. 이 규칙을 이해하면 복수형뿐 아니라 동사 활용에서도 큰 도움이 됩니다.

#### 규칙 1: 단모음 보호 — 자음 겹침

단모음(a, e, i, o, u를 한 번만 쓴 것) + 자음 하나인 경우, 복수형 -en을 붙이면 음절이 열리면서 장모음으로 발음될 위험이 있습니다. 이를 방지하기 위해 **자음을 겹쳐 씁니다**.

| 단수 | 복수 | 설명 |
|------|------|------|
| man | ma**nn**en | 단모음 a 유지 → n을 겹침 |
| bos | bo**ss**en | 단모음 o 유지 → s를 겹침 |
| kat | ka**tt**en | 단모음 a 유지 → t를 겹침 |
| bed | be**dd**en | 단모음 e 유지 → d를 겹침 |
| les | le**ss**en | 단모음 e 유지 → s를 겹침 |
| bus | bu**ss**en | 단모음 u 유지 → s를 겹침 |
| fles | fle**ss**en | 단모음 e 유지 → s를 겹침 |
| pot | po**tt**en | 단모음 o 유지 → t를 겹침 |
| zon | zo**nn**en | 단모음 o 유지 → n을 겹침 |
| bal | ba**ll**en | 단모음 a 유지 → l을 겹침 |

#### 규칙 2: 장모음 유지 — 겹자음 줄임

장모음(aa, oo, uu, ee 등 겹모음)은 음절이 열리면 모음 하나로도 장모음을 나타낼 수 있으므로, **겹자음을 하나로 줄입니다**.

| 단수 | 복수 | 설명 |
|------|------|------|
| maan | ma**n**en | 장모음 aa → 열린 음절에서 a 하나로 충분 |
| boot | bo**t**en | 장모음 oo → 열린 음절에서 o 하나로 충분 |
| naam | na**m**en | 장모음 aa → a |
| straat | stra**t**en | 장모음 aa → a |
| school | scho**l**en | 장모음 oo → o |
| duur | du**r**en | 장모음 uu → u |
| been | be**n**en | 장모음 ee → e |
| week | we**k**en | 장모음 ee → e |
| buur | bu**r**en | 장모음 uu → u |

#### 규칙 3: f → v, s → z 변화

일부 단어에서 어말의 f와 s가 복수형에서 각각 v와 z로 바뀝니다.

| 단수 | 복수 | 변화 |
|------|------|------|
| huis | hu**iz**en | s → z |
| brief | br**iev**en | f → v |
| kaas | ka**z**en | s → z |
| prijs | pr**ijz**en | s → z |
| neus | ne**uz**en | s → z |
| golf | go**lv**en | f → v |
| dief | d**iev**en | f → v |
| druif | dr**uiv**en | f → v |
| reis | re**iz**en | s → z |
| graaf | gra**v**en | f → v |

> **참고**: 모든 -f/-s 단어가 이렇게 변하는 것은 아닙니다. 예를 들어 brief → brieven이지만, dak → daken (변화 없음)입니다. 자주 쓰는 단어의 복수형은 개별적으로 외우는 것이 좋습니다.

### 불규칙 복수형

일부 명사는 일반 규칙을 따르지 않습니다. 이들은 개별적으로 암기해야 합니다.

| 단수 | 복수 | 뜻 |
|------|------|-----|
| kind | **kinder**en | 아이들 (-eren 복수) |
| ei | **eier**en | 달걀들 (-eren 복수) |
| lied | **lieder**en | 노래들 (-eren 복수) |
| volk | **volker**en | 민족들 (-eren 복수) |
| stad | **sted**en | 도시들 (모음 변화) |
| lid | **led**en | 회원들 (모음 변화) |
| schip | **schep**en | 배들 (모음 변화) |
| museum | muse**a** | 박물관들 (라틴어식) |
| centrum | centr**a** | 센터들 (라틴어식) |

### 복합 명사의 복수형

네덜란드어는 복합 명사(합성어)가 매우 많습니다. 복합 명사의 복수형은 **마지막 단어**만 복수형으로 바꿉니다.

| 단수 | 복수 | 뜻 |
|------|------|-----|
| schoolboek | schoolboek**en** | 교과서들 |
| voetbalwedstrijd | voetbalwedstrijd**en** | 축구 경기들 |
| huisdier | huisdier**en** | 애완동물들 |
| slaapkamer | slaapkamer**s** | 침실들 |
| postkantoor | postkanto**r**en | 우체국들 (oo→o) |
| boekenkast | boekenkast**en** | 책장들 |
| zonnebril | zonnebril**len** | 선글라스들 |

### 복수형이 없거나 특별한 명사

- **항상 복수인 명사**: de kleren (옷), de kosten (비용), de ouders (부모)
- **셀 수 없는 명사** (복수형 없음): het geld (돈), de melk (우유), het weer (날씨), de rijst (쌀)
- **단복수 동형**: het schaap → de schapen (양), 하지만 een vis → vissen (물고기), een mens → mensen (사람)

### 한국인이 자주 하는 실수

1. **철자 변화를 무시하는 실수**:
   - (X) man → manen → (O) man → ma**nn**en
   - (X) boot → booten → (O) boot → bo**t**en

2. **f/s → v/z 변화를 잊는 실수**:
   - (X) huis → huisen → (O) huis → hu**iz**en
   - (X) brief → briefen → (O) brief → br**iev**en

3. **-'s와 -s를 혼동하는 실수**:
   - (X) auto → autos → (O) auto → auto**'s** (모음으로 끝나는 외래어는 어포스트로피 필요)
   - (O) tafel → tafels (자음으로 끝나는 -el 단어는 그냥 -s)

4. **불규칙 복수형을 규칙적으로 만드는 실수**:
   - (X) kind → kinden → (O) kind → **kinderen**
   - (X) stad → stadden → (O) stad → **steden**

> **규칙 정리**:
> - 단모음 + 자음 하나 → 복수형에서 자음을 겹쳐 단모음을 유지
> - 장모음(겹모음) + 자음 겹침 → 복수형에서 자음 하나로 줄임
> - 일부 단어에서 f → v, s → z 변화
> - 불규칙 복수형은 개별 암기 필요

## 2-3. 인칭대명사 (Persoonlijke voornaamwoorden)

네덜란드어 인칭대명사는 **강조형**과 **비강조형**이 있습니다. 일상 회화에서는 대부분 비강조형을 사용하고, 대조하거나 강조할 때만 강조형을 씁니다. 한국어에서 "나"와 "내가"의 차이, 또는 "너"와 "네가"의 차이와 비슷하다고 생각할 수 있습니다.

### 주격

| 인칭 | 강조형 | 비강조형 | 한국어 |
|------|--------|---------|-------|
| 1인칭 단수 | **ik** | 'k | 나 |
| 2인칭 단수 (비격식) | **jij** | je | 너 |
| 2인칭 단수 (격식) | **u** | u | 당신 |
| 3인칭 단수 남성 | **hij** | -ie | 그 |
| 3인칭 단수 여성 | **zij** | ze | 그녀 |
| 3인칭 단수 중성 | **het** | 't | 그것 |
| 1인칭 복수 | **wij** | we | 우리 |
| 2인칭 복수 | **jullie** | - | 너희 |
| 3인칭 복수 | **zij** | ze | 그들 |

### 강조형 vs 비강조형 사용법

**비강조형** — 일반적인 상황 (더 자연스럽고 일상적):
- **Je** bent laat. (너 늦었어.)
- **Ze** woont in Amsterdam. (그녀는 암스테르담에 살아.)
- **We** gaan morgen. (우리 내일 가.)
- Hoe gaat het met **je**? (어떻게 지내?)
- Ik heb **'t** gezien. (나는 그것을 봤어.)

**강조형** — 대조, 강조, 문장 시작 시:
- **Jij** bent laat, niet ik! (너가 늦은 거야, 내가 아니라!)
- **Zij** woont in Amsterdam, maar hij woont in Rotterdam. (그녀는 암스테르담에 살고, 그는 로테르담에 살아.)
- **Wij** gaan wel, maar **zij** niet. (우리는 가지만, 그들은 안 가.)
- Wie is dat? — Dat is **hij**. (누구야? — 그야.)
- Ik bedoel **jou**, niet haar. (나는 너를 말하는 거야, 그녀가 아니라.)

### 목적격

| 주격 | 목적격 (강조) | 목적격 (비강조) | 예시 |
|------|-------------|---------------|------|
| ik | **mij** | **me** | Hij ziet mij/me. (그가 나를 봐.) |
| jij | **jou** | **je** | Ik ken jou/je. (나는 너를 알아.) |
| u | **u** | **u** | Ik help u. (제가 도와드려요.) |
| hij | **hem** | **'m** | Ik ken hem/'m. (나는 그를 알아.) |
| zij (그녀) | **haar** | **d'r / ze** | Ik ken haar/ze. (나는 그녀를 알아.) |
| het | **het** | **'t** | Ik zie het/'t. (나는 그것을 봐.) |
| wij | **ons** | **ons** | Hij kent ons. (그가 우리를 알아.) |
| jullie | **jullie** | **jullie** | Ik ken jullie. (나는 너희를 알아.) |
| zij (그들) | **hen / hun** | **ze** | Ik ken hen/ze. (나는 그들을 알아.) |

### hen vs hun 구분

**hen**과 **hun**의 공식적인 구분은 다음과 같습니다 (실제 회화에서는 거의 구분하지 않고 **ze**를 많이 씁니다):

| 형태 | 사용 | 예시 |
|------|------|------|
| **hen** | 직접 목적어 (전치사 뒤) | Ik zie **hen**. (나는 그들을 본다.) / voor **hen** (그들을 위해) |
| **hun** | 간접 목적어 | Ik geef **hun** het boek. (나는 그들에게 책을 준다.) |
| **ze** | 일상 회화에서 둘 다 대체 | Ik zie **ze**. / Ik geef **ze** het boek. |

> **실용 팁**: 일상 대화에서는 hen/hun 대신 거의 항상 **ze**를 씁니다. 글에서만 hen/hun을 정확히 구분하세요.

### 2인칭 사용법: jij / u / jullie

네덜란드어의 2인칭 선택은 사회적 맥락에 따라 달라집니다:

| 대명사 | 사용 상황 | 한국어 비교 |
|--------|---------|-----------|
| **jij/je** | 친구, 가족, 동료, 비슷한 나이의 사람 | 반말 "너" |
| **u** | 모르는 어른, 고객 서비스, 공식 상황 | 존댓말 "당신/선생님" |
| **jullie** | 복수의 비격식 대상 | "너희들" |

- 네덜란드에서는 **jij**를 매우 자유롭게 씁니다. 대부분의 일상 상황에서 jij를 사용합니다.
- **u**는 주로 고객 서비스, 공식 편지, 노인에게, 또는 벨기에(플란더스)에서 더 많이 사용합니다.
- 벨기에에서는 네덜란드보다 **u**를 훨씬 더 자주 씁니다.

### 비인칭 주어 het과 er

네덜란드어에서 **het**은 비인칭 주어로도 사용됩니다 (영어의 "it"과 같은 역할):

- **Het** regent. (비가 온다.)
- **Het** sneeuwt. (눈이 온다.)
- **Het** is koud. (추워.)
- **Het** is vijf uur. (5시야.)
- Hoe laat is **het**? (몇 시야?)
- **Het** gaat goed. (잘 되고 있어.)

**er**은 영어의 "there"에 해당하는 비인칭 표현에 사용됩니다:

- **Er** is een probleem. (문제가 있다.)
- **Er** zijn veel mensen. (사람이 많다.)
- **Er** staat een auto voor het huis. (집 앞에 차가 서 있다.)

### 인칭대명사 활용 예문

| 문장 | 번역 |
|------|------|
| **Ik** spreek Nederlands. | 나는 네덜란드어를 한다. |
| **Je** hebt een mooie tas. | 너는 예쁜 가방을 가지고 있어. |
| **Hij** werkt bij een bank. | 그는 은행에서 일해. |
| **Ze** studeert medicijnen. | 그녀는 의학을 공부해. |
| Ik geef **het** aan **haar**. | 나는 그것을 그녀에게 줘. |
| Kom **je** met **ons** mee? | 우리와 함께 갈래? |
| **Wij** wonen hier, maar **zij** wonen daar. | 우리는 여기 살고, 그들은 저기 살아. |
| Mag ik **u** iets vragen? | 제가 질문 하나 해도 될까요? |
| Kan ik **je** helpen? | 내가 도와줄까? |
| **Het** is lekker weer vandaag. | 오늘 날씨가 좋다. |

### 연습 대화

```
A: Hoi! Hoe gaat het met je?            (안녕! 어떻게 지내?)
B: Goed, en met jou?                     (잘 지내, 너는?)
A: Ook goed. Ken je hem?                 (나도 좋아. 그를 아니?)
B: Ja, hij is mijn collega.              (응, 그는 내 동료야.)
A: En zij? Ken je haar ook?              (그리고 그녀? 그녀도 아니?)
B: Nee, ik ken haar niet.               (아니, 그녀를 몰라.)
A: Zij is mijn zus. Ik stel jullie      (그녀는 내 여동생이야. 너희를
   aan elkaar voor.                       서로 소개해 줄게.)
B: Leuk! Wij gaan straks koffie          (좋아! 우리 이따 커피 마시러
   drinken. Ga je mee?                    갈 건데. 같이 갈래?)
A: Ja, ik ga met jullie mee.            (응, 너희와 같이 갈게.)
```

### 한국인이 자주 하는 실수

1. **주격과 목적격 혼동**: 한국어는 조사로 구분하므로 대명사 형태가 바뀌는 것을 놓치기 쉽습니다.
   - (X) Ik ken **hij**. → (O) Ik ken **hem**. (나는 그를 알아.)
   - (X) Zij ziet **wij**. → (O) Zij ziet **ons**. (그녀가 우리를 봐.)

2. **항상 강조형을 쓰는 실수**: 일상 대화에서는 비강조형이 더 자연스럽습니다.
   - (딱딱함) **Zij** gaat naar de winkel. → (자연스러움) **Ze** gaat naar de winkel.
   - (딱딱함) Ik zie **hem**. → (자연스러움) Ik zie **'m**. (구어체)

3. **zij(그녀)와 zij(그들) 혼동**: 문맥과 동사 활용으로 구분합니다.
   - Zij **is** mooi. (그녀는 예뻐.) — 단수 동사 is
   - Zij **zijn** mooi. (그들은 멋져.) — 복수 동사 zijn

4. **u를 너무 자주/적게 쓰는 실수**: 네덜란드에서는 대부분 jij를 쓰고, u는 공식적 상황에서만 씁니다.

## 2-4. 소유 대명사 (Bezittelijke voornaamwoorden)

소유 대명사는 "~의"에 해당하며, 명사 앞에 놓여 소유를 나타냅니다. 한국어의 "나의", "너의"와 기능이 같지만, 네덜란드어에서는 인칭대명사처럼 강조형과 비강조형이 있습니다.

### 소유 대명사 (명사 앞에 사용)

| 인칭 | 소유 대명사 (강조) | 비강조형 | 예시 |
|------|-----------------|---------|------|
| ik | **mijn** | m'n | mijn huis (나의 집) |
| jij | **jouw** | je | jouw boek (너의 책) |
| u | **uw** | - | uw naam (당신의 이름) |
| hij | **zijn** | z'n | zijn auto (그의 차) |
| zij | **haar** | d'r | haar tas (그녀의 가방) |
| het | **zijn** | z'n | zijn kleur (그것의 색) |
| wij | **ons / onze** | - | ons huis (het), onze auto (de) |
| jullie | **jullie** | - | jullie kamer (너희 방) |
| zij (복수) | **hun** | - | hun kinderen (그들의 아이들) |

### 각 소유 대명사 상세 예문

#### mijn / m'n (나의)

| 예문 | 번역 |
|------|------|
| **Mijn** naam is Kim. | 내 이름은 Kim이야. |
| Dit is **mijn** huis. | 이것은 내 집이야. |
| **Mijn** ouders wonen in Seoul. | 우리 부모님은 서울에 사셔. |
| Waar is **m'n** telefoon? | 내 전화기 어디 있어? (구어체) |
| **Mijn** broer is arts. | 내 형/오빠는 의사야. |

#### jouw / je (너의)

| 예문 | 번역 |
|------|------|
| Is dit **jouw** fiets? | 이거 네 자전거야? |
| Wat is **je** adres? | 네 주소가 뭐야? |
| **Jouw** idee is goed! | 네 아이디어가 좋아! |
| Ik vind **je** huis mooi. | 나는 네 집이 예쁘다고 생각해. |
| Hoe heet **je** kat? | 네 고양이 이름이 뭐야? |

#### zijn / z'n (그의, 그것의)

| 예문 | 번역 |
|------|------|
| **Zijn** vader werkt bij een bank. | 그의 아버지는 은행에서 일해. |
| Dit is **z'n** auto. | 이것은 그의 차야. (구어체) |
| **Zijn** hobby is voetbal. | 그의 취미는 축구야. |
| De kat likt **zijn** poot. | 고양이가 자기 발을 핥아. |
| Het boek en **zijn** auteur. | 그 책과 그것의 저자. |

#### haar / d'r (그녀의)

| 예문 | 번역 |
|------|------|
| **Haar** moeder is lerares. | 그녀의 어머니는 선생님이야. |
| Ik heb **haar** nummer niet. | 나는 그녀의 번호가 없어. |
| Dit is **d'r** kamer. | 이것은 그녀의 방이야. (구어체) |
| **Haar** Nederlandse is heel goed. | 그녀의 네덜란드어는 아주 좋아. |

#### uw (당신의 — 격식)

| 예문 | 번역 |
|------|------|
| Wat is **uw** naam? | 성함이 어떻게 되세요? |
| Hier is **uw** koffie. | 여기 커피 나왔습니다. |
| Mag ik **uw** paspoort zien? | 여권을 보여주시겠습니까? |
| **Uw** bestelling is klaar. | 주문하신 것이 준비됐습니다. |
| Wat is **uw** telefoonnummer? | 전화번호가 어떻게 되세요? |

### ons vs onze 상세 설명

"우리의"에 해당하는 소유 대명사는 뒤에 오는 명사의 관사에 따라 형태가 달라집니다. 이것은 다른 소유 대명사에는 없는 특별한 규칙입니다.

| 규칙 | 설명 | 예시 |
|------|------|------|
| **ons** + het 명사 | het 명사 앞에서 | **ons** huis (우리 집), **ons** kind (우리 아이), **ons** land (우리 나라) |
| **onze** + de 명사 | de 명사 앞에서 | **onze** auto (우리 차), **onze** school (우리 학교), **onze** leraar (우리 선생님) |
| **onze** + 복수 명사 | 복수 명사 앞에서 | **onze** kinderen (우리 아이들), **onze** boeken (우리 책들) |

더 많은 예시:

| ons (+ het 명사) | onze (+ de 명사) |
|-----------------|-----------------|
| ons bed (우리 침대) | onze keuken (우리 부엌) |
| ons geld (우리 돈) | onze tuin (우리 정원) |
| ons kantoor (우리 사무실) | onze stad (우리 도시) |
| ons probleem (우리 문제) | onze buurman (우리 이웃) |
| ons werk (우리 일) | onze vrienden (우리 친구들) |

> **팁**: ons/onze를 헷갈리면, 해당 명사가 **de** 명사인지 **het** 명사인지 먼저 확인하세요. het 명사면 **ons**, de 명사면 **onze**입니다. 복수형은 항상 de이므로 **onze**입니다.

### 독립 소유 대명사 (de mijne, de jouwe 등)

명사 없이 "~의 것"으로 쓸 때 사용하는 독립 소유 대명사도 있습니다. 영어의 "mine, yours, his" 등에 해당합니다.

| 인칭 | de 명사용 | het 명사용 | 복수용 |
|------|----------|----------|--------|
| ik | de **mijne** | het **mijne** | de **mijne** |
| jij | de **jouwe** | het **jouwe** | de **jouwe** |
| u | de **uwe** | het **uwe** | de **uwe** |
| hij | de **zijne** | het **zijne** | de **zijne** |
| zij | de **hare** | het **hare** | de **hare** |
| wij | de **onze** | het **onze** | de **onze** |
| jullie | - (die van jullie) | - (dat van jullie) | - (die van jullie) |
| zij (복수) | - (die van hen) | - (dat van hen) | - (die van hen) |

예문:

- Dit boek is **het mijne** (= mijn boek). (이 책은 내 것이야.)
- Is deze auto **de jouwe**? (이 차가 네 것이야?)
- **De onze** is groter. (우리 것이 더 커.)
- Zijn huis is klein, maar **het hare** is groot. (그의 집은 작지만, 그녀의 것은 커.)

> **실용 팁**: 일상 회화에서는 독립 소유 대명사 대신 **"van mij", "van jou"** 등을 더 자주 씁니다:
> - Dit boek is **van mij**. (이 책은 내 거야.) — 가장 자연스러운 표현
> - Is deze auto **van jou**? (이 차 네 거야?)
> - Die schoenen zijn **van haar**. (그 신발은 그녀 거야.)

### 소유 표현 "van"

한국어의 "~의"에 해당하는 다른 표현으로 **van**이 있습니다. 사람 이름이나 명사의 소유를 나타낼 때 사용합니다:

- het boek **van** de leraar (선생님의 책)
- de auto **van** mijn vader (아버지의 차)
- de kleur **van** het huis (집의 색깔)
- het huis **van** Jan (Jan의 집)
- de deur **van** de kamer (방의 문)

> **한국어와 비교**: 한국어는 "아버지**의** 차"처럼 소유격 조사 "의"가 소유자 뒤에 오지만, 네덜란드어는 "de auto **van** mijn vader"처럼 소유 대상이 먼저 나오고 van + 소유자가 뒤에 옵니다. 어순이 반대입니다!

### 연습 대화

```
A: Is dit jouw tas?                      (이거 네 가방이야?)
B: Nee, dat is niet de mijne.            (아니, 내 것이 아니야.)
   Dat is haar tas.                      (그것은 그녀의 가방이야.)
A: En deze sleutels? Van wie zijn ze?    (그리고 이 열쇠들? 누구 거야?)
B: Die zijn van mij! Dank je wel.        (내 거야! 고마워.)
A: Waar is ons boek?                     (우리 책 어디 있어?)
B: Ik heb het niet. Vraag het aan        (나한테 없어. 그들한테 물어봐.
   hen. Misschien is het hun boek.        아마 그들의 책일 거야.)
```

### 한국인이 자주 하는 실수

1. **ons/onze 혼동**:
   - (X) **onze** huis → (O) **ons** huis (het huis이므로 ons)
   - (X) **ons** auto → (O) **onze** auto (de auto이므로 onze)

2. **zijn을 "그녀의"로 쓰는 실수**: zij(그녀)의 소유 대명사는 **haar**이고, hij(그)의 소유 대명사가 **zijn**입니다.
   - (X) **Zijn** moeder (그녀의 어머니) → (O) **Haar** moeder

3. **소유 대명사와 인칭대명사 혼동**:
   - **haar** = 그녀의 (소유) / 그녀를 (목적격)
   - **hun** = 그들의 (소유) / 그들에게 (간접 목적격)
   - 문맥으로 구분해야 합니다: Ik ken **haar**. (그녀를 알아.) vs **Haar** boek is mooi. (그녀의 책이 예뻐.)

4. **"van" 어순 실수**: 한국어 어순의 영향으로 소유자를 먼저 쓰는 실수
   - (X) mijn vader van de auto → (O) de auto **van** mijn vader

## 2-5. 동사 현재시제 (Tegenwoordige tijd)

네덜란드어 동사 활용은 영어보다 복잡하지만 한국어만큼 복잡하지는 않습니다. 기본 규칙을 익히면 대부분의 동사를 올바르게 활용할 수 있습니다.

### 규칙 동사 활용 패턴

동사 원형(infinitief)에서 **-en**을 제거하면 **어간(stam)**이 됩니다.

예: **werken** (일하다) → 어간: **werk**

| 인칭 | 활용 | 예시 (werken) |
|------|------|-------------|
| ik | 어간 | ik **werk** |
| jij/je | 어간 + **t** | jij **werkt** |
| u | 어간 + **t** | u **werkt** |
| hij/zij/het | 어간 + **t** | hij **werkt** |
| wij/we | 원형 | wij **werken** |
| jullie | 원형 | jullie **werken** |
| zij (복수) | 원형 | zij **werken** |

### 어간(stam) 만드는 법 상세

원형에서 -en을 빼면 어간이 되지만, **철자 규칙**에 주의해야 합니다. 복수형 규칙과 동일한 원리입니다.

#### 기본 어간

| 원형 | 어간 | 뜻 |
|------|------|-----|
| werken | werk | 일하다 |
| leren | leer | 배우다 |
| wonen | woon | 살다 |
| koken | kook | 요리하다 |
| praten | praat | 이야기하다 |
| spelen | speel | 놀다/연주하다 |
| horen | hoor | 듣다 |
| vragen | vraag | 질문하다 |
| wachten | wacht | 기다리다 |
| antwoorden | antwoord | 대답하다 |

#### 철자 변화가 있는 어간

**장모음 유지**: 원형에서 모음이 하나인데 장모음이면, 어간에서 모음을 겹쳐 씁니다.

| 원형 | 어간 | 설명 |
|------|------|------|
| lev**en** | lee**f** | "살다" — 열린 음절의 e → 닫힌 음절에서 ee로, v → f |
| schrijv**en** | schrij**f** | "쓰다" — v → f |
| gev**en** | gee**f** | "주다" — 열린 e → ee, v → f |
| lezen | lee**s** | "읽다" — 열린 e → ee, z → s |
| reizen | rei**s** | "여행하다" — z → s |
| verhuiz**en** | verhui**s** | "이사하다" — z → s |
| mak**en** | maak | "만들다" — 열린 a → aa |
| lek**en** | leek | 닫힌 음절에서 장모음 유지 |

> **핵심 규칙**: 어간이 단음절이 되면서 닫힌 음절이 되므로:
> - 장모음은 겹모음으로 써야 합니다: leven → leef (e→ee), maken → maak (a→aa)
> - 어말의 v → f, z → s로 바뀝니다 (네덜란드어에서 단어 끝에 v, z가 올 수 없음)

#### 어간이 -t로 끝나는 동사

어간이 이미 -t로 끝나면, jij/hij 형태에 -t를 더 추가하지 않습니다.

| 원형 | 어간 | jij 형태 | 설명 |
|------|------|---------|------|
| zitten | zit | jij **zit** | -t를 추가하지 않음 (X: zitt) |
| praten | praat | jij **praat** | 이미 -t로 끝남 |
| wachten | wacht | jij **wacht** | 이미 -t로 끝남 |
| chatten | chat | jij **chat** | 이미 -t로 끝남 |
| eten | eet | jij **eet** | 이미 -t로 끝남 |

#### 어간이 -d로 끝나는 동사

어간이 -d로 끝나면 jij/hij 형태에 -t를 추가합니다. 발음은 /t/로 합니다.

| 원형 | 어간 | jij 형태 | 발음 |
|------|------|---------|------|
| vinden | vind | jij **vindt** | /vint/ |
| worden | word | jij **wordt** | /wort/ |
| houden | houd | jij **houdt** | /hout/ |
| rijden | rijd | jij **rijdt** | /rijt/ |

### 도치(역순)에서 jij의 특별 규칙

**jij**가 동사 뒤에 오면 **-t**가 사라집니다. 이것은 네덜란드어의 매우 중요한 규칙입니다!

- Jij werk**t** hard. (너는 열심히 일해.)
- Werk jij hard? (너 열심히 일해?) ← **t 탈락!**

더 많은 예시:

| 정상 어순 | 도치 (의문문) |
|----------|-------------|
| Jij **leest** een boek. | **Lees** jij een boek? |
| Jij **woont** in Amsterdam. | **Woon** jij in Amsterdam? |
| Jij **spreekt** Nederlands. | **Spreek** jij Nederlands? |
| Jij **hebt** een auto. | **Heb** jij een auto? |
| Jij **vindt** het leuk. | **Vind** jij het leuk? |
| Jij **eet** veel. | **Eet** jij veel? (어간이 -t로 끝나면 변화 없음) |

> **주의**: 이 규칙은 **jij/je**에만 적용됩니다. u나 hij에게는 적용되지 않습니다.
> - Werkt u hard? (t 유지)
> - Werkt hij hard? (t 유지)

### 자주 쓰는 규칙 동사 활용표

| 원형 | 뜻 | ik | jij | hij/zij | wij |
|------|-----|-----|------|---------|------|
| werken | 일하다 | werk | werkt | werkt | werken |
| wonen | 살다 | woon | woont | woont | wonen |

| fietsen | 자전거 타다 | fiets | fietst | fietst | fietsen |

### 일상생활 예문 (규칙 동사)

| 예문 | 번역 |
|------|------|
| Ik **woon** in Amsterdam. | 나는 암스테르담에 살아. |
| Hij **werkt** bij een ziekenhuis. | 그는 병원에서 일해. |
| Wij **leren** Nederlands. | 우리는 네덜란드어를 배워. |
| Ze **leest** elke dag een boek. | 그녀는 매일 책을 읽어. |

| Ik **koop** boodschappen. | 나는 장을 봐. |
| Hij **schrijft** een brief. | 그는 편지를 써. |

| Zij **drinkt** graag koffie. | 그녀는 커피를 즐겨 마셔. |
| De kinderen **spelen** buiten. | 아이들이 밖에서 놀아. |

### 주요 불규칙 동사

#### zijn (이다) — 가장 중요한 동사

| 인칭 | 활용 | 예문 |
|------|------|------|
| ik | **ben** | Ik **ben** student. (나는 학생이야.) |
| jij/je | **bent** | Jij **bent** laat. (너 늦었어.) |
| u | **bent** / **is** | U **bent** welkom. (환영합니다.) |
| hij/zij/het | **is** | Het **is** mooi weer. (날씨가 좋아.) |
| wij/we | **zijn** | Wij **zijn** vrienden. (우리는 친구야.) |
| jullie | **zijn** | Jullie **zijn** klaar. (너희 다 됐어.) |
| zij (복수) | **zijn** | Zij **zijn** uit Korea. (그들은 한국에서 왔어.) |

zijn의 일상 표현:
- Hoe **is** het? (어때?)
- Waar **ben** je? (너 어디야?)
- **Ben** jij Koreaans? (너 한국인이야?) — 도치에서도 t 탈락 없음 (불규칙이므로)
- Het **is** half drie. (2시 반이야.)
- Er **zijn** veel mensen. (사람이 많아.)

#### hebben (가지다)

| 인칭 | 활용 | 예문 |
|------|------|------|
| ik | **heb** | Ik **heb** een vraag. (질문이 있어요.) |
| jij/je | **hebt** | Jij **hebt** een mooie fiets. (너 예쁜 자전거 가지고 있네.) |
| u | **hebt** / **heeft** | **Heeft** u een moment? (시간 좀 있으세요?) |
| hij/zij/het | **heeft** | Hij **heeft** twee kinderen. (그는 아이가 둘이야.) |
| wij/we | **hebben** | Wij **hebben** honger. (우리 배고파.) |
| jullie | **hebben** | Jullie **hebben** geluk. (너희 운이 좋아.) |
| zij (복수) | **hebben** | Zij **hebben** geen geld. (그들은 돈이 없어.) |

hebben의 일상 표현:
- Ik **heb** honger. (나 배고파.) — 한국어: "배고프다" (형용사), 네덜란드어: "배고픔을 가지다" (동사)
- Ik **heb** dorst. (나 목말라.)
- Ik **heb** het koud. (나 추워.)
- Ik **heb** het warm. (나 더워.)
- **Heb** je zin in koffie? (커피 마실래?)

#### 기타 자주 쓰는 불규칙 동사

| 원형 | 뜻 | ik | jij | hij/zij | wij |
|------|-----|-----|------|---------|------|
| gaan | 가다 | ga | gaat | gaat | gaan |
| staan | 서다 | sta | staat | staat | staan |
| doen | 하다 | doe | doet | doet | doen |
| komen | 오다 | kom | komt | komt | komen |
| zien | 보다 | zie | ziet | ziet | zien |
| slaan | 치다 | sla | slaat | slaat | slaan |
| weten | 알다 | weet | weet | weet | weten |
| zeggen | 말하다 | zeg | zegt | zegt | zeggen |
| nemen | 가져가다 | neem | neemt | neemt | nemen |
| houden | 가지고 있다/좋아하다 | houd | houdt | houdt | houden |
| lopen | 걷다 | loop | loopt | loopt | lopen |
| rijden | 운전하다/타다 | rijd | rijdt | rijdt | rijden |

#### 조동사 (Hulpwerkwoorden)

조동사는 다른 동사와 함께 쓰여 의미를 보충합니다. 조동사 + 원형동사 구조로 사용됩니다.

| 원형 | 뜻 | ik | jij | hij/zij | wij |
|------|-----|-----|------|---------|------|
| zullen | ~할 것이다 | zal | zult/zal | zal | zullen |
| kunnen | ~할 수 있다 | kan | kunt/kan | kan | kunnen |
| mogen | ~해도 된다 | mag | mag | mag | mogen |
| moeten | ~해야 한다 | moet | moet | moet | moeten |
| willen | ~하고 싶다 | wil | wilt/wil | wil | willen |

조동사 예문:

| 예문 | 번역 |
|------|------|
| Ik **kan** Nederlands spreken. | 나는 네덜란드어를 할 수 있어. |
| **Mag** ik hier zitten? | 여기 앉아도 되나요? |
| Je **moet** hard studeren. | 너는 열심히 공부해야 해. |
| Wij **willen** naar Amsterdam gaan. | 우리는 암스테르담에 가고 싶어. |
| Het **zal** morgen regenen. | 내일 비가 올 거야. |
| **Kun** jij zwemmen? | 너 수영할 수 있어? (도치 t 탈락) |
| Je **mag** hier niet roken. | 여기서 담배 피면 안 돼. |
| Ik **moet** om 8 uur beginnen. | 나는 8시에 시작해야 해. |
| **Wil** je koffie of thee? | 커피 마실래, 차 마실래? |
| Ze **kunnen** goed voetballen. | 그들은 축구를 잘 할 수 있어. |

> **조동사 + 원형 어순**: 조동사가 두 번째 위치에 오고, 원형 동사는 문장 **끝**에 갑니다.
> - Ik **wil** een boek **lezen**. (나는 책을 읽고 싶어.)
> - Hij **moet** morgen vroeg **opstaan**. (그는 내일 일찍 일어나야 해.)
> - Wij **kunnen** goed Nederlands **spreken**. (우리는 네덜란드어를 잘 할 수 있어.)

### 연습 대화

```
A: Waar werk je?                         (어디서 일해?)
B: Ik werk bij een IT-bedrijf.           (IT 회사에서 일해.)
   En jij? Wat doe jij?                  (너는? 뭐 해?)
A: Ik studeer nog. Ik leer Nederlands.   (나는 아직 학생이야. 네덜란드어를 배워.)
B: Spreek je al een beetje?              (벌써 좀 할 줄 아니?)
A: Ja, ik kan een beetje spreken.        (응, 좀 할 수 있어.)
   Maar ik moet meer oefenen.            (하지만 더 연습해야 해.)
B: Wil je samen oefenen?                 (같이 연습할래?)
A: Ja, graag! Wanneer heb je tijd?       (응, 좋아! 언제 시간 있어?)
B: Ik heb morgenavond tijd.              (내일 저녁에 시간 있어.)
```

### 한국인이 자주 하는 실수

1. **도치에서 -t 탈락을 잊는 실수**:
   - (X) Werk**t** jij morgen? → (O) **Werk** jij morgen?
   - (X) Heb**t** jij een auto? → (O) **Heb** jij een auto?

2. **어간 철자 실수**:
   - (X) Ik **lef** → (O) Ik **leef** (leven의 어간은 leef)
   - (X) Ik **schrijv** → (O) Ik **schrijf** (v→f)
   - (X) Ik **lez** → (O) Ik **lees** (z→s, 열린 e→ee)

3. **조동사 뒤에 활용형을 쓰는 실수**: 조동사 뒤의 동사는 항상 원형!
   - (X) Ik wil **werk**. → (O) Ik wil **werken**.
   - (X) Hij kan **spreekt**. → (O) Hij kan **spreken**.

4. **어간이 -t로 끝나는 동사에 -t를 더 붙이는 실수**:
   - (X) Jij **zitt** → (O) Jij **zit**
   - (X) Hij **eet**t → (O) Hij **eet**

5. **현재시제의 의미 차이**: 네덜란드어 현재시제는 영어의 현재진행형도 포함합니다.
   - Ik **lees** een boek. = I read a book. / I am reading a book.
   - 네덜란드어에는 별도의 진행형이 없습니다!

## 2-6. 기본 어순 (Woordvolgorde)

네덜란드어 어순은 한국어 화자에게 매우 중요한 주제입니다. 한국어는 SOV(주어-목적어-동사)이고, 네덜란드어는 기본적으로 SVO(주어-동사-목적어)이며, 특히 **V2(동사 두 번째 위치) 규칙**이 핵심입니다. 독일어와 매우 유사한 규칙입니다.

### 평서문: SVO (주어 + 동사 + 목적어)

네덜란드어의 기본 어순은 영어와 비슷합니다.

```
Ik     lees    een boek.
주어    동사    목적어
나는   읽는다   책을.
```

더 많은 예시:

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik **drink** koffie. | 나는 커피를 마신다. |
| Hij **spreekt** Nederlands. | 그는 네덜란드어를 한다. |
| Wij **wonen** in Amsterdam. | 우리는 암스테르담에 산다. |
| De kinderen **spelen** in het park. | 아이들이 공원에서 논다. |
| Mijn moeder **kookt** het eten. | 우리 어머니가 음식을 요리한다. |
| Zij **studeert** aan de universiteit. | 그녀는 대학에서 공부한다. |
| De leraar **geeft** les. | 선생님이 수업을 한다. |

### 의문문: VSO (동사 + 주어 + 목적어)

예/아니오 질문은 동사가 첫 번째 위치에 옵니다.

```
Lees    jij    een boek?
동사    주어    목적어
읽니    너는    책을?
```

더 많은 예시:

| 의문문 | 번역 |
|--------|------|
| **Spreek** jij Nederlands? | 너 네덜란드어 하니? |
| **Heb** je een pen? | 펜 있어? |
| **Woon** jij in Seoul? | 너 서울에 살아? |
| **Is** zij je zus? | 그녀가 네 여동생이야? |
| **Gaan** jullie morgen naar school? | 너희 내일 학교 가니? |
| **Komt** hij vanavond? | 그가 오늘 저녁에 오니? |
| **Mag** ik hier zitten? | 여기 앉아도 되나요? |

### 의문사가 있는 의문문

의문사(wie, wat, waar, wanneer, hoe, waarom 등)가 첫 번째 위치에 오고, 동사가 두 번째 위치에 옵니다.

| 의문문 | 번역 |
|--------|------|
| **Wat** lees jij? | 너 뭐 읽어? |
| **Waar** woon je? | 어디 살아? |
| **Wanneer** kom je? | 언제 와? |
| **Hoe** gaat het? | 어떻게 지내? |
| **Waarom** leer je Nederlands? | 왜 네덜란드어를 배우니? |
| **Wie** is dat? | 누구야? |
| **Hoeveel** kost het? | 얼마예요? |
| **Hoe laat** begint de les? | 수업 몇 시에 시작해? |
| **Welk** boek lees je? | 어떤 책 읽어? |
| **Hoelang** woon je hier? | 여기 얼마나 오래 살았어? |

### V2 규칙 (핵심!)

네덜란드어에서 **활용된 동사는 항상 두 번째 위치**에 옵니다. 이것이 네덜란드어 어순의 가장 중요한 규칙입니다.

```
Vandaag    lees    ik    een boek.
부사       동사    주어   목적어
오늘       읽는다  나는   책을.
```

> 부사가 문장 앞에 오면 주어와 동사가 뒤바뀝니다 (도치/inversie).

다양한 요소가 첫 번째 위치에 올 수 있습니다:

| 첫 번째 위치 | 문장 | 번역 |
|-------------|------|------|
| 시간 부사 | **Morgen** ga ik naar Amsterdam. | 내일 나는 암스테르담에 간다. |
| 장소 부사 | **Hier** woon ik. | 여기 내가 산다. |
| 목적어 | **Dat boek** lees ik graag. | 그 책을 나는 즐겨 읽는다. |
| 전치사구 | **In Nederland** regent het vaak. | 네덜란드에서는 비가 자주 온다. |
| 부사절 | **Daarom** leer ik Nederlands. | 그래서 나는 네덜란드어를 배운다. |
| 접속부사 | **Eerst** eet ik, **dan** ga ik werken. | 먼저 먹고, 그 다음에 일하러 간다. |
| 빈도부사 | **Soms** lees ik 's avonds. | 가끔 나는 저녁에 읽는다. |

> **핵심**: 무엇이 첫 번째 위치에 오든, 활용된 동사는 **항상 두 번째 위치**입니다!

#### V2 규칙이 적용되지 않는 접속사

**등위접속사**(en, of, maar, want, dus)는 위치를 차지하지 않으므로 그 뒤에 정상 SVO 어순이 옵니다:

| 문장 | 번역 |
|------|------|
| Ik leer Nederlands **en** ik woon in Amsterdam. | 나는 네덜란드어를 배우고 암스테르담에 산다. |
| Hij werkt hard, **maar** hij is moe. | 그는 열심히 일하지만, 피곤하다. |
| Ik ga niet, **want** ik ben ziek. | 나는 안 가, 왜냐하면 아프니까. |
| Wil je koffie **of** wil je thee? | 커피 마실래 아니면 차 마실래? |

### 시간-방법-장소 규칙 (Tijd-Wijze-Plaats)

영어는 Place-Manner-Time 순서이지만, 네덜란드어는 **Tijd(시간)-Wijze(방법)-Plaats(장소)** 순서입니다:

```
Ik    ga    morgen    met de trein    naar Amsterdam.
주어   동사   시간       방법           장소
나는   간다   내일     기차로         암스테르담으로.
```

| 문장 | 시간 | 방법 | 장소 |
|------|------|------|------|
| Ik ga **morgen** **met de fiets** **naar school**. | morgen | met de fiets | naar school |
| Wij gaan **vanavond** **samen** **naar het restaurant**. | vanavond | samen | naar het restaurant |
| Hij rijdt **elke dag** **met de auto** **naar het werk**. | elke dag | met de auto | naar het werk |
| Ze vliegt **volgende week** **alleen** **naar Korea**. | volgende week | alleen | naar Korea |

> **영어와 비교**: 영어 "I go to school by bike tomorrow" (장소-방법-시간) vs 네덜란드어 "Ik ga morgen met de fiets naar school" (시간-방법-장소)
>
> **한국어와 비교**: 한국어 "나는 내일 자전거로 학교에 간다" — 흥미롭게도 한국어의 시간-방법-장소 순서는 네덜란드어와 비슷합니다!

### 부정문: niet / geen

| 부정어 | 사용 | 예시 |
|--------|------|------|
| **niet** | 동사, 형용사, 부사 부정 | Ik werk **niet**. (나는 일하지 않는다.) |
| **geen** | 부정관사(een) 또는 무관사 명사 부정 | Ik heb **geen** auto. (나는 차가 없다.) |

#### geen 사용 예시

**een**이 있는 문장이나 무관사 명사를 부정할 때 **geen**을 씁니다:

| 긍정문 | 부정문 |
|--------|--------|
| Ik heb **een** auto. | Ik heb **geen** auto. (차가 없다.) |
| Ik heb **een** hond. | Ik heb **geen** hond. (개가 없다.) |
| Ik drink koffie. | Ik drink **geen** koffie. (커피를 마시지 않는다.) |
| Hij heeft geld. | Hij heeft **geen** geld. (돈이 없다.) |
| Er is tijd. | Er is **geen** tijd. (시간이 없다.) |
| Ik heb honger. | Ik heb **geen** honger. (배고프지 않다.) |
| Wij hebben kinderen. | Wij hebben **geen** kinderen. (아이가 없다.) |

#### niet의 위치 규칙

**niet**의 위치는 복잡하지만, 기본 규칙을 따르면 대부분의 상황을 처리할 수 있습니다:

**1) 문장 끝에 오는 경우 (가장 기본)**

| 예문 | 번역 |
|------|------|
| Ik werk **niet**. | 나는 일하지 않는다. |
| Hij komt **niet**. | 그는 오지 않는다. |
| Wij gaan **niet**. | 우리는 가지 않는다. |

**2) 형용사/부사 앞에 오는 경우**

| 예문 | 번역 |
|------|------|
| Het boek is **niet** goed. | 그 책은 좋지 않다. |
| Hij werkt **niet** hard. | 그는 열심히 일하지 않는다. |
| Dat is **niet** mooi. | 그것은 예쁘지 않다. |
| Ik ben **niet** moe. | 나는 피곤하지 않다. |

**3) 전치사구 앞에 오는 경우**

| 예문 | 번역 |
|------|------|
| Ik ga **niet** naar school. | 나는 학교에 가지 않는다. |
| Hij woont **niet** in Amsterdam. | 그는 암스테르담에 살지 않는다. |
| We gaan **niet** met de trein. | 우리는 기차로 가지 않는다. |

**4) 원형동사(조동사 구문에서) 앞에 오는 경우**

| 예문 | 번역 |
|------|------|
| Ik kan **niet** zwemmen. | 나는 수영할 수 없다. |
| Hij wil **niet** werken. | 그는 일하고 싶지 않다. |
| Je hoeft **niet** te komen. | 너는 올 필요 없다. |

**5) 정관사(de/het) 명사 앞에 오는 경우** (특정 대상 부정)

| 예문 | 번역 |
|------|------|
| Ik lees **niet** het boek. (Ik lees het boek **niet**.) | 나는 그 책을 읽지 않는다. |
| Hij kent **niet** de weg. | 그는 그 길을 모른다. |

> **niet vs geen 정리**:
> - **geen** = een/무관사 명사를 부정 ("없다"의 느낌)
> - **niet** = 그 외 모든 것을 부정 ("아니다"의 느낌)
> - 의심스러우면: "없다"로 번역되면 **geen**, "안 하다/아니다"로 번역되면 **niet**

### 조동사가 있는 문장의 어순

조동사가 있으면 조동사가 두 번째 위치에 오고, 본동사(원형)가 문장 끝에 갑니다:

```
Ik    wil    morgen    naar Amsterdam    gaan.
주어   조동사  시간       장소             본동사(원형)
나는  ~싶다   내일     암스테르담으로       가다.
```

| 문장 | 번역 |
|------|------|
| Ik **moet** morgen vroeg **opstaan**. | 나는 내일 일찍 일어나야 해. |
| Hij **kan** goed Nederlands **spreken**. | 그는 네덜란드어를 잘 할 수 있어. |
| Wij **willen** vanavond samen **eten**. | 우리는 오늘 저녁 같이 먹고 싶어. |
| **Kun** je me morgen **helpen**? | 내일 나를 도와줄 수 있어? |
| Ze **mag** hier niet **parkeren**. | 그녀는 여기 주차하면 안 돼. |

### 연습 대화

```
A: Ga je morgen naar het werk?           (내일 출근해?)
B: Nee, morgen werk ik niet.             (아니, 내일은 일 안 해.)
   Ik ga met de trein naar Utrecht.      (기차로 위트레흐트에 가.)
A: Waarom ga je naar Utrecht?            (왜 위트레흐트에 가?)
B: Ik moet mijn vriend daar bezoeken.    (거기 있는 친구를 방문해야 해.)
A: Hoe laat ga je?                       (몇 시에 가?)
B: Ik ga om negen uur 's ochtends.       (아침 9시에 가.)
A: Wil je ook iets voor mij kopen?       (나한테도 뭔가 사 줄래?)
B: Ja, wat wil je hebben?                (응, 뭘 갖고 싶어?)
A: Ik heb geen nieuwe schoenen.          (새 신발이 없거든.)
   Kun je schoenen voor mij meenemen?    (신발을 가져다줄 수 있어?)
```

### 한국인이 자주 하는 실수

1. **동사를 문장 끝에 놓는 실수**: 한국어 어순(SOV)의 영향
   - (X) Ik een boek **lees**. → (O) Ik **lees** een boek. (동사는 두 번째 위치!)

2. **V2 규칙을 무시하는 실수**: 부사가 앞에 오면 도치해야 합니다.
   - (X) Morgen **ik ga** naar school. → (O) Morgen **ga ik** naar school.
   - (X) Soms **ik lees** een boek. → (O) Soms **lees ik** een boek.

3. **niet과 geen 혼동**:
   - (X) Ik heb **niet** auto. → (O) Ik heb **geen** auto. (een 명사 부정은 geen)
   - (X) Ik ben **geen** moe. → (O) Ik ben **niet** moe. (형용사 부정은 niet)

4. **시간-방법-장소 순서를 영어식으로 하는 실수**:
   - (X) Ik ga naar school met de fiets morgen. → (O) Ik ga morgen met de fiets naar school.

5. **조동사 구문에서 본동사를 활용하는 실수**: 본동사는 항상 원형!
   - (X) Ik wil naar huis **ga**. → (O) Ik wil naar huis **gaan**.

## 2-7. 자주 쓰는 전치사 (Voorzetsels)

전치사는 문장에서 명사/대명사와 결합하여 장소, 시간, 방향, 수단 등을 나타냅니다. 한국어의 조사(-에, -에서, -으로, -와 등)에 해당하지만, 네덜란드어 전치사는 명사 **앞에** 옵니다 (한국어 조사는 명사 뒤에 옴).

### 장소 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **in** | ~안에 | in de kamer (방 안에), in het water (물 안에) |
| **op** | ~위에 | op de tafel (테이블 위에), op straat (거리에서) |
| **aan** | ~에 (접촉/붙어) | aan de muur (벽에), aan tafel (식탁에 앉아) |
| **bij** | ~옆에, ~근처에 | bij het station (역 근처에), bij mij (내 옆에) |
| **voor** | ~앞에 | voor het huis (집 앞에), voor de deur (문 앞에) |
| **achter** | ~뒤에 | achter het huis (집 뒤에), achter de auto (차 뒤에) |
| **naast** | ~옆에 | naast de school (학교 옆에), naast mij (내 옆에) |
| **boven** | ~위에 (떨어져) | boven de deur (문 위에), boven de wolken (구름 위에) |
| **onder** | ~아래에 | onder de tafel (테이블 아래에), onder het bed (침대 아래에) |
| **tussen** | ~사이에 | tussen de huizen (집들 사이에), tussen ons (우리 사이에) |
| **tegenover** | ~맞은편에 | tegenover het station (역 맞은편에) |
| **binnen** | ~안에 (건물 등) | binnen het gebouw (건물 안에) |
| **buiten** | ~밖에 | buiten de stad (도시 밖에) |

#### in vs op 구분

| in (내부, 둘러싸인 공간) | op (표면, 위) |
|------------------------|-------------|
| in de kamer (방 안에) | op de tafel (테이블 위에) |
| in het water (물 속에) | op het dak (지붕 위에) |
| in de stad (도시 안에) | op straat (거리에서) |
| in de trein (기차 안에) | op de fiets (자전거를 타고) |
| in bed (침대에 누워) | op school (학교에서) |
| in het ziekenhuis (병원에 입원) | op kantoor (사무실에서) |

> **주의**: in/op의 사용은 항상 논리적이지 않습니다. "op school" (학교에서), "op kantoor" (사무실에서)처럼 표면 위가 아니라 장소를 나타내는 경우도 있습니다. 관용적 표현으로 외워야 합니다.

### 방향/이동 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **naar** | ~으로 (방향) | naar school (학교로), naar Amsterdam (암스테르담으로) |
| **van** | ~에서 (출발점) | van Amsterdam (암스테르담에서) |
| **uit** | ~에서 (출신/나옴) | uit Korea (한국에서 온), uit de kast (옷장에서 꺼내) |
| **door** | ~을 통해 | door de stad (도시를 통해), door het park (공원을 지나) |
| **langs** | ~을 따라/지나 | langs de rivier (강을 따라), langs het huis (집을 지나) |
| **tot** | ~까지 | tot Amsterdam (암스테르담까지), tot morgen (내일까지) |
| **richting** | ~방향으로 | richting het centrum (시내 방향으로) |

#### naar vs aan vs in (장소 이동)

| naar (이동 방향) | aan (도착/접촉) | in (안에 들어감) |
|----------------|----------------|----------------|
| Ik ga **naar** school. | Ik zit **aan** tafel. | Ik ben **in** de kamer. |
| Ik ga **naar** Amsterdam. | Ik woon **aan** zee. | Ik stap **in** de trein. |
| Ik ga **naar** huis. | Hij komt **aan** (도착하다). | Ik zwem **in** het water. |

### 시간 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **om** | ~시에 (시각) | om drie uur (3시에), om half vijf (4시 반에) |
| **in** | ~에 (월/계절/년) | in januari (1월에), in de zomer (여름에), in 2024 (2024년에) |
| **op** | ~에 (요일/날짜) | op maandag (월요일에), op 5 mei (5월 5일에) |
| **'s** | ~에 (하루 시간대) | 's ochtends (아침에), 's middags (오후에), 's avonds (저녁에), 's nachts (밤에) |
| **na** | ~후에 | na het eten (식사 후에), na de les (수업 후에) |
| **voor** | ~전에 | voor het eten (식사 전에), voor vijf uur (5시 전에) |
| **tijdens** | ~동안 | tijdens de les (수업 동안), tijdens het eten (식사 동안) |
| **sinds** | ~이래로 | sinds gisteren (어제 이래로), sinds 2020 (2020년 이래로) |
| **tot** | ~까지 | tot vijf uur (5시까지), tot morgen (내일까지) |
| **van...tot** | ~부터 ~까지 | van negen tot vijf (9시부터 5시까지) |
| **tegen** | ~즈음에 | tegen de avond (저녁 즈음에) |

시간 전치사 사용 예문:

| 예문 | 번역 |
|------|------|
| De les begint **om** negen uur. | 수업은 9시에 시작해. |
| Ik ga **in** juli **naar** Korea. | 나는 7월에 한국에 가. |
| **Op** zaterdag werk ik niet. | 토요일에는 일하지 않아. |
| Ik drink **'s ochtends** altijd koffie. | 아침에는 항상 커피를 마셔. |
| **Na** het werk ga ik sporten. | 일 끝나고 운동하러 가. |
| Ik woon hier **sinds** twee jaar. | 나는 여기 2년 전부터 살아. |
| De winkel is open **van** 9 **tot** 5. | 가게는 9시부터 5시까지 열어. |

### 기타 중요한 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **met** | ~와 함께, ~로 (수단) | met mijn vriend (친구와 함께), met de trein (기차로) |
| **voor** | ~을 위해 | voor jou (너를 위해), voor het werk (일을 위해) |
| **van** | ~의 | het boek van Jan (Jan의 책) |
| **over** | ~에 대해 | over het weer (날씨에 대해), over politiek (정치에 대해) |
| **zonder** | ~없이 | zonder geld (돈 없이), zonder jou (너 없이) |
| **tegen** | ~에게 (말), ~에 반대 | tegen mij (나에게), tegen de muur (벽에 기대어) |
| **volgens** | ~에 따르면 | volgens mij (내 생각에), volgens de krant (신문에 따르면) |
| **ondanks** | ~에도 불구하고 | ondanks het weer (날씨에도 불구하고) |
| **dankzij** | ~덕분에 | dankzij jou (너 덕분에) |
| **vanwege** | ~때문에 | vanwege de regen (비 때문에) |
| **behalve** | ~을 제외하고 | behalve mij (나를 제외하고) |
| **per** | ~당, ~로 (수단) | per e-mail (이메일로), per dag (하루에) |

### 전치사 + 동사 고정 표현

네덜란드어에서 많은 동사가 특정 전치사와 결합합니다. 이것은 외워야 합니다.

| 동사 + 전치사 | 뜻 | 예문 |
|-------------|-----|------|
| denken **aan** | ~을 생각하다 | Ik denk **aan** je. (너를 생각해.) |
| wachten **op** | ~을 기다리다 | Ik wacht **op** de bus. (버스를 기다려.) |
| luisteren **naar** | ~을 듣다 | Ik luister **naar** muziek. (음악을 들어.) |
| kijken **naar** | ~을 보다 | Ik kijk **naar** een film. (영화를 봐.) |
| zoeken **naar** | ~을 찾다 | Ik zoek **naar** mijn sleutels. (열쇠를 찾아.) |
| houden **van** | ~을 좋아하다 | Ik houd **van** je. (너를 사랑해.) |
| praten **over** | ~에 대해 이야기하다 | We praten **over** het werk. (일에 대해 이야기해.) |
| beginnen **met** | ~을 시작하다 | Ik begin **met** de les. (수업을 시작해.) |
| stoppen **met** | ~을 그만두다 | Hij stopt **met** roken. (그는 담배를 끊어.) |
| dromen **van** | ~을 꿈꾸다 | Ik droom **van** vakantie. (휴가를 꿈꿔.) |
| lijken **op** | ~을 닮다 | Hij lijkt **op** zijn vader. (그는 아버지를 닮았어.) |
| vragen **om** | ~을 요청하다 | Ik vraag **om** hulp. (도움을 요청해.) |
| feliciteren **met** | ~을 축하하다 | Ik feliciteer je **met** je verjaardag! (생일 축하해!) |
| lachen **om** | ~을 보고 웃다 | We lachen **om** de grap. (농담에 웃어.) |
| rekenen **op** | ~을 기대하다/믿다 | Je kunt **op** mij rekenen. (나를 믿어도 돼.) |

### 전치사 축약형 (er + 전치사)

사물을 가리킬 때 "전치사 + het"을 쓰지 않고 **er + 전치사**를 씁니다:

| 원래 형태 (사용 X) | 축약형 (사용 O) | 뜻 |
|-------------------|----------------|-----|
| op het | **erop** | 그 위에 / 그것에 대해 |
| aan het | **eraan** | 그것에 |
| in het | **erin** | 그 안에 |
| over het | **erover** | 그것에 대해 |
| van het | **ervan** | 그것의 / 그것으로부터 |
| mee het | **ermee** | 그것으로 / 그것과 함께 |
| naar het | **ernaar** | 그것을 향해 |

예문:

| 예문 | 번역 |
|------|------|
| Ik denk **eraan**. | 나는 그것을 생각해. (ik denk aan het (X)) |
| Ik kijk **ernaar**. | 나는 그것을 봐. |
| Hij praat **erover**. | 그는 그것에 대해 이야기해. |
| Ik wacht **erop**. | 나는 그것을 기다려. |
| Ik houd **ervan**. | 나는 그것을 좋아해. |
| Wat doe je **ermee**? | 그것으로 뭐 해? |

> **규칙**: 사물에 대해 말할 때 → er + 전치사 (eraan, erop, erover...)
> 사람에 대해 말할 때 → 전치사 + 대명사 (aan hem, op haar, over hen...)
> - Ik denk **aan hem**. (나는 그를 생각해.) — 사람
> - Ik denk **eraan**. (나는 그것을 생각해.) — 사물

### 한국인이 자주 하는 실수

1. **전치사 위치를 한국어 조사처럼 뒤에 놓는 실수**:
   - (X) school **naar** → (O) **naar** school (전치사는 명사 앞에!)

2. **naar와 in 혼동**:
   - Ik ga **naar** Amsterdam. (암스테르담으로 간다.) — 이동/방향
   - Ik woon **in** Amsterdam. (암스테르담에 산다.) — 위치/상태

3. **시간 전치사 혼동 (om/in/op)**:
   - **om** drie uur (3시에) — 정확한 시각
   - **in** januari (1월에) — 월/계절/년
   - **op** maandag (월요일에) — 요일/날짜

4. **동사 + 전치사 고정 표현을 모르는 실수**:
   - (X) Ik denk **over** je → (O) Ik denk **aan** je (denken aan = ~을 생각하다)
   - (X) Ik luister muziek → (O) Ik luister **naar** muziek (luisteren naar = ~을 듣다)

5. **er + 전치사를 사용하지 않는 실수**:
   - (X) Ik denk aan **het**. → (O) Ik denk **eraan**. (사물은 er + 전치사 사용)
