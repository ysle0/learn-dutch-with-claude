# Level 2: 필수 문법 기초

> 관사, 명사, 인칭대명사, 기본 동사 활용, 어순을 배웁니다.

---

## 2-1. 관사 (Lidwoorden)

네덜란드어에는 두 종류의 정관사가 있습니다.

### 정관사

| 관사 | 사용 | 예시 |
|------|------|------|
| **de** | 공통성 명사 (남성/여성) | de man (남자), de vrouw (여자), de tafel (테이블) |
| **het** | 중성 명사 | het kind (아이), het boek (책), het huis (집) |

### 부정관사

| 관사 | 사용 | 예시 |
|------|------|------|
| **een** | 모든 단수 명사 | een man (한 남자), een boek (한 책) |

> **핵심 포인트**:
> - 복수형은 항상 **de**를 사용합니다: de boeken (책들), de huizen (집들)
> - de 명사가 약 2/3, het 명사가 약 1/3입니다
> - 어떤 관사를 쓰는지 외우는 것이 중요합니다 — 규칙이 완벽하지 않으므로 단어와 함께 관사를 외우세요

### het 명사가 되는 패턴 (참고용)

- 축소형 (-je): het hondje (강아지), het meisje (소녀)
- 대부분의 ge- 접두사 명사: het gebouw (건물), het gevoel (느낌)
- 대부분의 2음절 이상 명사로 be-, ge-, ver-, ont-로 시작하는 것

## 2-2. 명사의 복수형 (Meervoud)

### 주요 복수형 규칙

| 규칙 | 단수 → 복수 | 예시 |
|------|------------|------|
| **-en 추가** | 가장 일반적 | boek → boek**en** (책들) |
| **-s 추가** | -el, -em, -en, -er, -je로 끝나는 단어 | tafel → tafel**s** (테이블들) |
| **-'s 추가** | 모음으로 끝나는 외래어 | auto → auto**'s** (자동차들) |

### 철자 변화에 주의

단모음이 열린 음절이 되면 철자가 바뀝니다:

| 단수 | 복수 | 설명 |
|------|------|------|
| man | ma**nn**en | 단모음 유지를 위해 자음 겹침 |
| maan | ma**n**en | 장모음이므로 자음 하나로 |
| bos | bo**ss**en | 단모음 유지 |
| boot | bo**t**en | 장모음이므로 자음 하나로 |
| huis | hu**iz**en | s → z 변화 |
| brief | br**iev**en | f → v 변화 |

> **규칙**: 단모음 + 자음 하나 → 복수형에서 자음을 겹쳐 단모음을 유지
> 장모음(겹모음) + 자음 겹침 → 복수형에서 자음 하나로 줄임

## 2-3. 인칭대명사 (Persoonlijke voornaamwoorden)

### 주격

| 인칭 | 강조형 | 비강조형 | 한국어 |
|------|--------|---------|-------|
| 1인칭 단수 | **ik** | 'k | 나 |
| 2인칭 단수 (비격식) | **jij** | je | 너 |
| 2인칭 단수 (격식) | **u** | u | 당신 |
| 3인칭 단수 남성 | **hij** | -ie | 그 |
| 3인칭 단수 여성 | **zij** | ze | 그녀 |
| 3인칭 단수 중성 | **het** | 't | 그것 |
| 1인칭 복수 | **wij** | we | 우리 |
| 2인칭 복수 | **jullie** | - | 너희 |
| 3인칭 복수 | **zij** | ze | 그들 |

### 목적격

| 주격 | 목적격 | 예시 |
|------|--------|------|
| ik | **mij / me** | Hij ziet mij. (그가 나를 봐.) |
| jij | **jou / je** | Ik ken jou. (나는 너를 알아.) |
| u | **u** | Ik help u. (제가 도와드려요.) |
| hij | **hem** | Ik ken hem. (나는 그를 알아.) |
| zij (그녀) | **haar** | Ik ken haar. (나는 그녀를 알아.) |
| het | **het** | Ik zie het. (나는 그것을 봐.) |
| wij | **ons** | Hij kent ons. (그가 우리를 알아.) |
| jullie | **jullie** | Ik ken jullie. (나는 너희를 알아.) |
| zij (그들) | **hen / hun** | Ik ken hen. (나는 그들을 알아.) |

## 2-4. 소유 대명사 (Bezittelijke voornaamwoorden)

| 인칭 | 소유 대명사 | 예시 |
|------|-----------|------|
| ik | **mijn** (m'n) | mijn huis (나의 집) |
| jij | **jouw** (je) | jouw boek (너의 책) |
| u | **uw** | uw naam (당신의 이름) |
| hij | **zijn** (z'n) | zijn auto (그의 차) |
| zij | **haar** (d'r) | haar tas (그녀의 가방) |
| het | **zijn** | zijn kleur (그것의 색) |
| wij | **ons / onze** | ons huis (het), onze auto (de) |
| jullie | **jullie** | jullie kamer (너희 방) |
| zij (복수) | **hun** | hun kinderen (그들의 아이들) |

> **ons vs onze**: het 명사 앞에서는 **ons**, de 명사 앞에서는 **onze**
> - ons huis (우리 집, het huis)
> - onze auto (우리 차, de auto)

## 2-5. 동사 현재시제 (Tegenwoordige tijd)

### 규칙 동사 활용 패턴

동사 원형(infinitief)에서 **-en**을 제거하면 **어간(stam)**이 됩니다.

예: **werken** (일하다) → 어간: **werk**

| 인칭 | 활용 | 예시 (werken) |
|------|------|-------------|
| ik | 어간 | ik **werk** |
| jij/je | 어간 + **t** | jij **werkt** |
| u | 어간 + **t** | u **werkt** |
| hij/zij/het | 어간 + **t** | hij **werkt** |
| wij/we | 원형 | wij **werken** |
| jullie | 원형 | jullie **werken** |
| zij (복수) | 원형 | zij **werken** |

### 도치(역순)에서 jij의 특별 규칙

**jij**가 동사 뒤에 오면 **-t**가 사라집니다:

- Jij werk**t** hard. (너는 열심히 일해.)
- Werk jij hard? (너 열심히 일해?) ← **t 탈락!**

### 주요 불규칙 동사

#### zijn (이다) — 가장 중요한 동사

| 인칭 | 활용 |
|------|------|
| ik | **ben** |
| jij/je | **bent** |
| u | **bent** / **is** |
| hij/zij/het | **is** |
| wij/we | **zijn** |
| jullie | **zijn** |
| zij (복수) | **zijn** |

#### hebben (가지다)

| 인칭 | 활용 |
|------|------|
| ik | **heb** |
| jij/je | **hebt** |
| u | **hebt** / **heeft** |
| hij/zij/het | **heeft** |
| wij/we | **hebben** |
| jullie | **hebben** |
| zij (복수) | **hebben** |

#### 기타 자주 쓰는 불규칙 동사

| 원형 | 뜻 | ik | jij | hij/zij | wij |
|------|-----|-----|------|---------|------|
| gaan | 가다 | ga | gaat | gaat | gaan |
| doen | 하다 | doe | doet | doet | doen |
| komen | 오다 | kom | komt | komt | komen |
| zullen | ~할 것이다 | zal | zult/zal | zal | zullen |
| kunnen | ~할 수 있다 | kan | kunt/kan | kan | kunnen |
| mogen | ~해도 된다 | mag | mag | mag | mogen |
| moeten | ~해야 한다 | moet | moet | moet | moeten |
| willen | ~하고 싶다 | wil | wilt/wil | wil | willen |

## 2-6. 기본 어순 (Woordvolgorde)

### 평서문: SVO (주어 + 동사 + 목적어)

네덜란드어의 기본 어순은 영어와 비슷합니다.

```
Ik     lees    een boek.
주어    동사    목적어
나는   읽는다   책을.
```

### 의문문: VSO (동사 + 주어 + 목적어)

```
Lees    jij    een boek?
동사    주어    목적어
읽니    너는    책을?
```

### V2 규칙 (핵심!)

네덜란드어에서 **활용된 동사는 항상 두 번째 위치**에 옵니다.

```
Vandaag    lees    ik    een boek.
부사       동사    주어   목적어
오늘       읽는다  나는   책을.
```

> 부사가 문장 앞에 오면 주어와 동사가 뒤바뀝니다 (도치/inversie).

### 부정문: niet / geen

| 부정어 | 사용 | 예시 |
|--------|------|------|
| **niet** | 동사, 형용사, 부사 부정 | Ik werk **niet**. (나는 일하지 않는다.) |
| **geen** | 부정관사(een) 또는 무관사 명사 부정 | Ik heb **geen** auto. (나는 차가 없다.) |

- Ik heb een auto. → Ik heb **geen** auto. (een → geen)
- Ik drink koffie. → Ik drink **geen** koffie. (무관사 → geen)
- Ik werk hard. → Ik werk **niet** hard. (동사/부사 부정)
- Het boek is goed. → Het boek is **niet** goed. (형용사 부정)

## 2-7. 자주 쓰는 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| in | ~안에 | in de kamer (방 안에) |
| op | ~위에 | op de tafel (테이블 위에) |
| aan | ~에 (접촉) | aan de muur (벽에) |
| met | ~와 함께 | met mijn vriend (친구와 함께) |
| voor | ~을 위해, ~앞에 | voor jou (너를 위해) |
| van | ~의, ~에서 | van Amsterdam (암스테르담에서) |
| naar | ~으로 (방향) | naar school (학교로) |
| uit | ~에서 (출처) | uit Korea (한국에서) |
| bij | ~옆에, ~에서 | bij het station (역 옆에) |
| over | ~에 대해, ~위로 | over het weer (날씨에 대해) |
| door | ~을 통해 | door de stad (도시를 통해) |
| zonder | ~없이 | zonder geld (돈 없이) |
