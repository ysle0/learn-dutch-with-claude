# 2-6. 기본 어순 (Woordvolgorde)

네덜란드어 어순은 한국어 화자에게 매우 중요한 주제입니다. 한국어는 SOV(주어-목적어-동사)이고, 네덜란드어는 기본적으로 SVO(주어-동사-목적어)이며, 특히 **V2(동사 두 번째 위치) 규칙**이 핵심입니다. 독일어와 매우 유사한 규칙입니다.

### 평서문: SVO (주어 + 동사 + 목적어)

네덜란드어의 기본 어순은 영어와 비슷합니다.

```
Ik     lees    een boek.
주어    동사    목적어
나는   읽는다   책을.
```

더 많은 예시:

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik **drink** koffie. | 나는 커피를 마신다. |
| Hij **spreekt** Nederlands. | 그는 네덜란드어를 한다. |
| Wij **wonen** in Amsterdam. | 우리는 암스테르담에 산다. |
| De kinderen **spelen** in het park. | 아이들이 공원에서 논다. |
| Mijn moeder **kookt** het eten. | 우리 어머니가 음식을 요리한다. |
| Zij **studeert** aan de universiteit. | 그녀는 대학에서 공부한다. |
| De leraar **geeft** les. | 선생님이 수업을 한다. |

### 의문문: VSO (동사 + 주어 + 목적어)

예/아니오 질문은 동사가 첫 번째 위치에 옵니다.

```
Lees    jij    een boek?
동사    주어    목적어
읽니    너는    책을?
```

더 많은 예시:

| 의문문 | 번역 |
|--------|------|
| **Spreek** jij Nederlands? | 너 네덜란드어 하니? |
| **Heb** je een pen? | 펜 있어? |
| **Woon** jij in Seoul? | 너 서울에 살아? |
| **Is** zij je zus? | 그녀가 네 여동생이야? |
| **Gaan** jullie morgen naar school? | 너희 내일 학교 가니? |
| **Komt** hij vanavond? | 그가 오늘 저녁에 오니? |
| **Mag** ik hier zitten? | 여기 앉아도 되나요? |

### 의문사가 있는 의문문

의문사(wie, wat, waar, wanneer, hoe, waarom 등)가 첫 번째 위치에 오고, 동사가 두 번째 위치에 옵니다.

| 의문문 | 번역 |
|--------|------|
| **Wat** lees jij? | 너 뭐 읽어? |
| **Waar** woon je? | 어디 살아? |
| **Wanneer** kom je? | 언제 와? |
| **Hoe** gaat het? | 어떻게 지내? |
| **Waarom** leer je Nederlands? | 왜 네덜란드어를 배우니? |
| **Wie** is dat? | 누구야? |
| **Hoeveel** kost het? | 얼마예요? |
| **Hoe laat** begint de les? | 수업 몇 시에 시작해? |
| **Welk** boek lees je? | 어떤 책 읽어? |
| **Hoelang** woon je hier? | 여기 얼마나 오래 살았어? |

### V2 규칙 (핵심!)

네덜란드어에서 **활용된 동사는 항상 두 번째 위치**에 옵니다. 이것이 네덜란드어 어순의 가장 중요한 규칙입니다.

```
Vandaag    lees    ik    een boek.
부사       동사    주어   목적어
오늘       읽는다  나는   책을.
```

> 부사가 문장 앞에 오면 주어와 동사가 뒤바뀝니다 (도치/inversie).

다양한 요소가 첫 번째 위치에 올 수 있습니다:

| 첫 번째 위치 | 문장 | 번역 |
|-------------|------|------|
| 시간 부사 | **Morgen** ga ik naar Amsterdam. | 내일 나는 암스테르담에 간다. |
| 장소 부사 | **Hier** woon ik. | 여기 내가 산다. |
| 목적어 | **Dat boek** lees ik graag. | 그 책을 나는 즐겨 읽는다. |
| 전치사구 | **In Nederland** regent het vaak. | 네덜란드에서는 비가 자주 온다. |
| 부사절 | **Daarom** leer ik Nederlands. | 그래서 나는 네덜란드어를 배운다. |
| 접속부사 | **Eerst** eet ik, **dan** ga ik werken. | 먼저 먹고, 그 다음에 일하러 간다. |
| 빈도부사 | **Soms** lees ik 's avonds. | 가끔 나는 저녁에 읽는다. |

> **핵심**: 무엇이 첫 번째 위치에 오든, 활용된 동사는 **항상 두 번째 위치**입니다!

#### V2 규칙이 적용되지 않는 접속사

**등위접속사**(en, of, maar, want, dus)는 위치를 차지하지 않으므로 그 뒤에 정상 SVO 어순이 옵니다:

| 문장 | 번역 |
|------|------|
| Ik leer Nederlands **en** ik woon in Amsterdam. | 나는 네덜란드어를 배우고 암스테르담에 산다. |
| Hij werkt hard, **maar** hij is moe. | 그는 열심히 일하지만, 피곤하다. |
| Ik ga niet, **want** ik ben ziek. | 나는 안 가, 왜냐하면 아프니까. |
| Wil je koffie **of** wil je thee? | 커피 마실래 아니면 차 마실래? |

### 시간-방법-장소 규칙 (Tijd-Wijze-Plaats)

영어는 Place-Manner-Time 순서이지만, 네덜란드어는 **Tijd(시간)-Wijze(방법)-Plaats(장소)** 순서입니다:

```
Ik    ga    morgen    met de trein    naar Amsterdam.
주어   동사   시간       방법           장소
나는   간다   내일     기차로         암스테르담으로.
```

| 문장 | 시간 | 방법 | 장소 |
|------|------|------|------|
| Ik ga **morgen** **met de fiets** **naar school**. | morgen | met de fiets | naar school |
| Wij gaan **vanavond** **samen** **naar het restaurant**. | vanavond | samen | naar het restaurant |
| Hij rijdt **elke dag** **met de auto** **naar het werk**. | elke dag | met de auto | naar het werk |
| Ze vliegt **volgende week** **alleen** **naar Korea**. | volgende week | alleen | naar Korea |

> **영어와 비교**: 영어 "I go to school by bike tomorrow" (장소-방법-시간) vs 네덜란드어 "Ik ga morgen met de fiets naar school" (시간-방법-장소)
>
> **한국어와 비교**: 한국어 "나는 내일 자전거로 학교에 간다" — 흥미롭게도 한국어의 시간-방법-장소 순서는 네덜란드어와 비슷합니다!

### 부정문: niet / geen

| 부정어 | 사용 | 예시 |
|--------|------|------|
| **niet** | 동사, 형용사, 부사 부정 | Ik werk **niet**. (나는 일하지 않는다.) |
| **geen** | 부정관사(een) 또는 무관사 명사 부정 | Ik heb **geen** auto. (나는 차가 없다.) |

#### geen 사용 예시

**een**이 있는 문장이나 무관사 명사를 부정할 때 **geen**을 씁니다:

| 긍정문 | 부정문 |
|--------|--------|
| Ik heb **een** auto. | Ik heb **geen** auto. (차가 없다.) |
| Ik heb **een** hond. | Ik heb **geen** hond. (개가 없다.) |
| Ik drink koffie. | Ik drink **geen** koffie. (커피를 마시지 않는다.) |
| Hij heeft geld. | Hij heeft **geen** geld. (돈이 없다.) |
| Er is tijd. | Er is **geen** tijd. (시간이 없다.) |
| Ik heb honger. | Ik heb **geen** honger. (배고프지 않다.) |
| Wij hebben kinderen. | Wij hebben **geen** kinderen. (아이가 없다.) |

#### niet의 위치 규칙

**niet**의 위치는 복잡하지만, 기본 규칙을 따르면 대부분의 상황을 처리할 수 있습니다:

**1) 문장 끝에 오는 경우 (가장 기본)**

| 예문 | 번역 |
|------|------|
| Ik werk **niet**. | 나는 일하지 않는다. |
| Hij komt **niet**. | 그는 오지 않는다. |
| Wij gaan **niet**. | 우리는 가지 않는다. |

**2) 형용사/부사 앞에 오는 경우**

| 예문 | 번역 |
|------|------|
| Het boek is **niet** goed. | 그 책은 좋지 않다. |
| Hij werkt **niet** hard. | 그는 열심히 일하지 않는다. |
| Dat is **niet** mooi. | 그것은 예쁘지 않다. |
| Ik ben **niet** moe. | 나는 피곤하지 않다. |

**3) 전치사구 앞에 오는 경우**

| 예문 | 번역 |
|------|------|
| Ik ga **niet** naar school. | 나는 학교에 가지 않는다. |
| Hij woont **niet** in Amsterdam. | 그는 암스테르담에 살지 않는다. |
| We gaan **niet** met de trein. | 우리는 기차로 가지 않는다. |

**4) 원형동사(조동사 구문에서) 앞에 오는 경우**

| 예문 | 번역 |
|------|------|
| Ik kan **niet** zwemmen. | 나는 수영할 수 없다. |
| Hij wil **niet** werken. | 그는 일하고 싶지 않다. |
| Je hoeft **niet** te komen. | 너는 올 필요 없다. |

**5) 정관사(de/het) 명사 앞에 오는 경우** (특정 대상 부정)

| 예문 | 번역 |
|------|------|
| Ik lees **niet** het boek. (Ik lees het boek **niet**.) | 나는 그 책을 읽지 않는다. |
| Hij kent **niet** de weg. | 그는 그 길을 모른다. |

> **niet vs geen 정리**:
> - **geen** = een/무관사 명사를 부정 ("없다"의 느낌)
> - **niet** = 그 외 모든 것을 부정 ("아니다"의 느낌)
> - 의심스러우면: "없다"로 번역되면 **geen**, "안 하다/아니다"로 번역되면 **niet**

### 조동사가 있는 문장의 어순

조동사가 있으면 조동사가 두 번째 위치에 오고, 본동사(원형)가 문장 끝에 갑니다:

```
Ik    wil    morgen    naar Amsterdam    gaan.
주어   조동사  시간       장소             본동사(원형)
나는  ~싶다   내일     암스테르담으로       가다.
```

| 문장 | 번역 |
|------|------|
| Ik **moet** morgen vroeg **opstaan**. | 나는 내일 일찍 일어나야 해. |
| Hij **kan** goed Nederlands **spreken**. | 그는 네덜란드어를 잘 할 수 있어. |
| Wij **willen** vanavond samen **eten**. | 우리는 오늘 저녁 같이 먹고 싶어. |
| **Kun** je me morgen **helpen**? | 내일 나를 도와줄 수 있어? |
| Ze **mag** hier niet **parkeren**. | 그녀는 여기 주차하면 안 돼. |

### 연습 대화

```
A: Ga je morgen naar het werk?           (내일 출근해?)
B: Nee, morgen werk ik niet.             (아니, 내일은 일 안 해.)
   Ik ga met de trein naar Utrecht.      (기차로 위트레흐트에 가.)
A: Waarom ga je naar Utrecht?            (왜 위트레흐트에 가?)
B: Ik moet mijn vriend daar bezoeken.    (거기 있는 친구를 방문해야 해.)
A: Hoe laat ga je?                       (몇 시에 가?)
B: Ik ga om negen uur 's ochtends.       (아침 9시에 가.)
A: Wil je ook iets voor mij kopen?       (나한테도 뭔가 사 줄래?)
B: Ja, wat wil je hebben?                (응, 뭘 갖고 싶어?)
A: Ik heb geen nieuwe schoenen.          (새 신발이 없거든.)
   Kun je schoenen voor mij meenemen?    (신발을 가져다줄 수 있어?)
```

### 한국인이 자주 하는 실수

1. **동사를 문장 끝에 놓는 실수**: 한국어 어순(SOV)의 영향
   - (X) Ik een boek **lees**. → (O) Ik **lees** een boek. (동사는 두 번째 위치!)

2. **V2 규칙을 무시하는 실수**: 부사가 앞에 오면 도치해야 합니다.
   - (X) Morgen **ik ga** naar school. → (O) Morgen **ga ik** naar school.
   - (X) Soms **ik lees** een boek. → (O) Soms **lees ik** een boek.

3. **niet과 geen 혼동**:
   - (X) Ik heb **niet** auto. → (O) Ik heb **geen** auto. (een 명사 부정은 geen)
   - (X) Ik ben **geen** moe. → (O) Ik ben **niet** moe. (형용사 부정은 niet)

4. **시간-방법-장소 순서를 영어식으로 하는 실수**:
   - (X) Ik ga naar school met de fiets morgen. → (O) Ik ga morgen met de fiets naar school.

5. **조동사 구문에서 본동사를 활용하는 실수**: 본동사는 항상 원형!
   - (X) Ik wil naar huis **ga**. → (O) Ik wil naar huis **gaan**.
