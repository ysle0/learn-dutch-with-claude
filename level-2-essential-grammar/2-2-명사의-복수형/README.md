# 2-2. 명사의 복수형 (Meervoud)

네덜란드어 복수형은 크게 **-en**, **-s**, **-'s** 세 가지 패턴이 있습니다. 한국어의 "-들"처럼 간단하지 않고, 영어처럼 철자 변화가 따르기도 합니다. 네덜란드어 특유의 철자 규칙을 이해하면 복수형을 올바르게 쓸 수 있습니다.
