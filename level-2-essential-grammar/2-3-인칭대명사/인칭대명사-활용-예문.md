# 인칭대명사 활용 예문

| 문장 | 번역 |
|------|------|
| **Ik** spreek Nederlands. | 나는 네덜란드어를 한다. |
| **Je** hebt een mooie tas. | 너는 예쁜 가방을 가지고 있어. |
| **Hij** werkt bij een bank. | 그는 은행에서 일해. |
| **Ze** studeert medicijnen. | 그녀는 의학을 공부해. |
| Ik geef **het** aan **haar**. | 나는 그것을 그녀에게 줘. |
| Kom **je** met **ons** mee? | 우리와 함께 갈래? |
| **Wij** wonen hier, maar **zij** wonen daar. | 우리는 여기 살고, 그들은 저기 살아. |
| Mag ik **u** iets vragen? | 제가 질문 하나 해도 될까요? |
| Kan ik **je** helpen? | 내가 도와줄까? |
| **Het** is lekker weer vandaag. | 오늘 날씨가 좋다. |
