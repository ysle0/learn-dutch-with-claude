# 비인칭 주어 het과 er

네덜란드어에서 **het**은 비인칭 주어로도 사용됩니다 (영어의 "it"과 같은 역할):

- **Het** regent. (비가 온다.)
- **Het** sneeuwt. (눈이 온다.)
- **Het** is koud. (추워.)
- **Het** is vijf uur. (5시야.)
- Hoe laat is **het**? (몇 시야?)
- **Het** gaat goed. (잘 되고 있어.)

**er**은 영어의 "there"에 해당하는 비인칭 표현에 사용됩니다:

- **Er** is een probleem. (문제가 있다.)
- **Er** zijn veel mensen. (사람이 많다.)
- **Er** staat een auto voor het huis. (집 앞에 차가 서 있다.)
