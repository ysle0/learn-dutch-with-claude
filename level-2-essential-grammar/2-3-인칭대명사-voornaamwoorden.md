# 2-3. 인칭대명사 (Persoonlijke voornaamwoorden)

네덜란드어 인칭대명사는 **강조형**과 **비강조형**이 있습니다. 일상 회화에서는 대부분 비강조형을 사용하고, 대조하거나 강조할 때만 강조형을 씁니다. 한국어에서 "나"와 "내가"의 차이, 또는 "너"와 "네가"의 차이와 비슷하다고 생각할 수 있습니다.

### 주격

| 인칭 | 강조형 | 비강조형 | 한국어 |
|------|--------|---------|-------|
| 1인칭 단수 | **ik** | 'k | 나 |
| 2인칭 단수 (비격식) | **jij** | je | 너 |
| 2인칭 단수 (격식) | **u** | u | 당신 |
| 3인칭 단수 남성 | **hij** | -ie | 그 |
| 3인칭 단수 여성 | **zij** | ze | 그녀 |
| 3인칭 단수 중성 | **het** | 't | 그것 |
| 1인칭 복수 | **wij** | we | 우리 |
| 2인칭 복수 | **jullie** | - | 너희 |
| 3인칭 복수 | **zij** | ze | 그들 |

### 강조형 vs 비강조형 사용법

**비강조형** — 일반적인 상황 (더 자연스럽고 일상적):
- **Je** bent laat. (너 늦었어.)
- **Ze** woont in Amsterdam. (그녀는 암스테르담에 살아.)
- **We** gaan morgen. (우리 내일 가.)
- Hoe gaat het met **je**? (어떻게 지내?)
- Ik heb **'t** gezien. (나는 그것을 봤어.)

**강조형** — 대조, 강조, 문장 시작 시:
- **Jij** bent laat, niet ik! (너가 늦은 거야, 내가 아니라!)
- **Zij** woont in Amsterdam, maar hij woont in Rotterdam. (그녀는 암스테르담에 살고, 그는 로테르담에 살아.)
- **Wij** gaan wel, maar **zij** niet. (우리는 가지만, 그들은 안 가.)
- Wie is dat? — Dat is **hij**. (누구야? — 그야.)
- Ik bedoel **jou**, niet haar. (나는 너를 말하는 거야, 그녀가 아니라.)

### 목적격

| 주격 | 목적격 (강조) | 목적격 (비강조) | 예시 |
|------|-------------|---------------|------|
| ik | **mij** | **me** | Hij ziet mij/me. (그가 나를 봐.) |
| jij | **jou** | **je** | Ik ken jou/je. (나는 너를 알아.) |
| u | **u** | **u** | Ik help u. (제가 도와드려요.) |
| hij | **hem** | **'m** | Ik ken hem/'m. (나는 그를 알아.) |
| zij (그녀) | **haar** | **d'r / ze** | Ik ken haar/ze. (나는 그녀를 알아.) |
| het | **het** | **'t** | Ik zie het/'t. (나는 그것을 봐.) |
| wij | **ons** | **ons** | Hij kent ons. (그가 우리를 알아.) |
| jullie | **jullie** | **jullie** | Ik ken jullie. (나는 너희를 알아.) |
| zij (그들) | **hen / hun** | **ze** | Ik ken hen/ze. (나는 그들을 알아.) |

### hen vs hun 구분

**hen**과 **hun**의 공식적인 구분은 다음과 같습니다 (실제 회화에서는 거의 구분하지 않고 **ze**를 많이 씁니다):

| 형태 | 사용 | 예시 |
|------|------|------|
| **hen** | 직접 목적어 (전치사 뒤) | Ik zie **hen**. (나는 그들을 본다.) / voor **hen** (그들을 위해) |
| **hun** | 간접 목적어 | Ik geef **hun** het boek. (나는 그들에게 책을 준다.) |
| **ze** | 일상 회화에서 둘 다 대체 | Ik zie **ze**. / Ik geef **ze** het boek. |

> **실용 팁**: 일상 대화에서는 hen/hun 대신 거의 항상 **ze**를 씁니다. 글에서만 hen/hun을 정확히 구분하세요.

### 2인칭 사용법: jij / u / jullie

네덜란드어의 2인칭 선택은 사회적 맥락에 따라 달라집니다:

| 대명사 | 사용 상황 | 한국어 비교 |
|--------|---------|-----------|
| **jij/je** | 친구, 가족, 동료, 비슷한 나이의 사람 | 반말 "너" |
| **u** | 모르는 어른, 고객 서비스, 공식 상황 | 존댓말 "당신/선생님" |
| **jullie** | 복수의 비격식 대상 | "너희들" |

- 네덜란드에서는 **jij**를 매우 자유롭게 씁니다. 대부분의 일상 상황에서 jij를 사용합니다.
- **u**는 주로 고객 서비스, 공식 편지, 노인에게, 또는 벨기에(플란더스)에서 더 많이 사용합니다.
- 벨기에에서는 네덜란드보다 **u**를 훨씬 더 자주 씁니다.

### 비인칭 주어 het과 er

네덜란드어에서 **het**은 비인칭 주어로도 사용됩니다 (영어의 "it"과 같은 역할):

- **Het** regent. (비가 온다.)
- **Het** sneeuwt. (눈이 온다.)
- **Het** is koud. (추워.)
- **Het** is vijf uur. (5시야.)
- Hoe laat is **het**? (몇 시야?)
- **Het** gaat goed. (잘 되고 있어.)

**er**은 영어의 "there"에 해당하는 비인칭 표현에 사용됩니다:

- **Er** is een probleem. (문제가 있다.)
- **Er** zijn veel mensen. (사람이 많다.)
- **Er** staat een auto voor het huis. (집 앞에 차가 서 있다.)

### 인칭대명사 활용 예문

| 문장 | 번역 |
|------|------|
| **Ik** spreek Nederlands. | 나는 네덜란드어를 한다. |
| **Je** hebt een mooie tas. | 너는 예쁜 가방을 가지고 있어. |
| **Hij** werkt bij een bank. | 그는 은행에서 일해. |
| **Ze** studeert medicijnen. | 그녀는 의학을 공부해. |
| Ik geef **het** aan **haar**. | 나는 그것을 그녀에게 줘. |
| Kom **je** met **ons** mee? | 우리와 함께 갈래? |
| **Wij** wonen hier, maar **zij** wonen daar. | 우리는 여기 살고, 그들은 저기 살아. |
| Mag ik **u** iets vragen? | 제가 질문 하나 해도 될까요? |
| Kan ik **je** helpen? | 내가 도와줄까? |
| **Het** is lekker weer vandaag. | 오늘 날씨가 좋다. |

### 연습 대화

```
A: Hoi! Hoe gaat het met je?            (안녕! 어떻게 지내?)
B: Goed, en met jou?                     (잘 지내, 너는?)
A: Ook goed. Ken je hem?                 (나도 좋아. 그를 아니?)
B: Ja, hij is mijn collega.              (응, 그는 내 동료야.)
A: En zij? Ken je haar ook?              (그리고 그녀? 그녀도 아니?)
B: Nee, ik ken haar niet.               (아니, 그녀를 몰라.)
A: Zij is mijn zus. Ik stel jullie      (그녀는 내 여동생이야. 너희를
   aan elkaar voor.                       서로 소개해 줄게.)
B: Leuk! Wij gaan straks koffie          (좋아! 우리 이따 커피 마시러
   drinken. Ga je mee?                    갈 건데. 같이 갈래?)
A: Ja, ik ga met jullie mee.            (응, 너희와 같이 갈게.)
```

### 한국인이 자주 하는 실수

1. **주격과 목적격 혼동**: 한국어는 조사로 구분하므로 대명사 형태가 바뀌는 것을 놓치기 쉽습니다.
   - (X) Ik ken **hij**. → (O) Ik ken **hem**. (나는 그를 알아.)
   - (X) Zij ziet **wij**. → (O) Zij ziet **ons**. (그녀가 우리를 봐.)

2. **항상 강조형을 쓰는 실수**: 일상 대화에서는 비강조형이 더 자연스럽습니다.
   - (딱딱함) **Zij** gaat naar de winkel. → (자연스러움) **Ze** gaat naar de winkel.
   - (딱딱함) Ik zie **hem**. → (자연스러움) Ik zie **'m**. (구어체)

3. **zij(그녀)와 zij(그들) 혼동**: 문맥과 동사 활용으로 구분합니다.
   - Zij **is** mooi. (그녀는 예뻐.) — 단수 동사 is
   - Zij **zijn** mooi. (그들은 멋져.) — 복수 동사 zijn

4. **u를 너무 자주/적게 쓰는 실수**: 네덜란드에서는 대부분 jij를 쓰고, u는 공식적 상황에서만 씁니다.
