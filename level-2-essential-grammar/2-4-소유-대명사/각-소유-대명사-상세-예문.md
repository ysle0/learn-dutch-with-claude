# 각 소유 대명사 상세 예문

#### mijn / m'n (나의)

| 예문 | 번역 |
|------|------|
| **Mijn** naam is Kim. | 내 이름은 Kim이야. |
| Dit is **mijn** huis. | 이것은 내 집이야. |
| **Mijn** ouders wonen in Seoul. | 우리 부모님은 서울에 사셔. |
| Waar is **m'n** telefoon? | 내 전화기 어디 있어? (구어체) |
| **Mijn** broer is arts. | 내 형/오빠는 의사야. |

#### jouw / je (너의)

| 예문 | 번역 |
|------|------|
| Is dit **jouw** fiets? | 이거 네 자전거야? |
| Wat is **je** adres? | 네 주소가 뭐야? |
| **Jouw** idee is goed! | 네 아이디어가 좋아! |
| Ik vind **je** huis mooi. | 나는 네 집이 예쁘다고 생각해. |
| Hoe heet **je** kat? | 네 고양이 이름이 뭐야? |

#### zijn / z'n (그의, 그것의)

| 예문 | 번역 |
|------|------|
| **Zijn** vader werkt bij een bank. | 그의 아버지는 은행에서 일해. |
| Dit is **z'n** auto. | 이것은 그의 차야. (구어체) |
| **Zijn** hobby is voetbal. | 그의 취미는 축구야. |
| De kat likt **zijn** poot. | 고양이가 자기 발을 핥아. |
| Het boek en **zijn** auteur. | 그 책과 그것의 저자. |

#### haar / d'r (그녀의)

| 예문 | 번역 |
|------|------|
| **Haar** moeder is lerares. | 그녀의 어머니는 선생님이야. |
| Ik heb **haar** nummer niet. | 나는 그녀의 번호가 없어. |
| Dit is **d'r** kamer. | 이것은 그녀의 방이야. (구어체) |
| **Haar** Nederlandse is heel goed. | 그녀의 네덜란드어는 아주 좋아. |

#### uw (당신의 — 격식)

| 예문 | 번역 |
|------|------|
| Wat is **uw** naam? | 성함이 어떻게 되세요? |
| Hier is **uw** koffie. | 여기 커피 나왔습니다. |
| Mag ik **uw** paspoort zien? | 여권을 보여주시겠습니까? |
| **Uw** bestelling is klaar. | 주문하신 것이 준비됐습니다. |
| Wat is **uw** telefoonnummer? | 전화번호가 어떻게 되세요? |
