# 2-2. 명사의 복수형 (Meervoud)

네덜란드어 복수형은 크게 **-en**, **-s**, **-'s** 세 가지 패턴이 있습니다. 한국어의 "-들"처럼 간단하지 않고, 영어처럼 철자 변화가 따르기도 합니다. 네덜란드어 특유의 철자 규칙을 이해하면 복수형을 올바르게 쓸 수 있습니다.

### 주요 복수형 규칙

| 규칙 | 적용 조건 | 예시 |
|------|----------|------|
| **-en 추가** | 가장 일반적인 패턴 | boek → boek**en** (책들) |
| **-s 추가** | -el, -em, -en, -er, -je, -ie로 끝나는 단어 | tafel → tafel**s** (테이블들) |
| **-'s 추가** | -a, -i, -o, -u, -y로 끝나는 외래어 | auto → auto**'s** (자동차들) |

### -en 복수형 예시 (가장 일반적)

| 단수 | 복수 | 뜻 |
|------|------|-----|
| boek | boeken | 책들 |
| dag | dagen | 날들 |
| vraag | vragen | 질문들 |
| taal | talen | 언어들 |
| jaar | jaren | 년들 |
| dier | dieren | 동물들 |
| woord | woorden | 단어들 |
| stad | steden | 도시들 (불규칙!) |
| kind | kinderen | 아이들 (불규칙!) |
| land | landen | 나라들 |
| punt | punten | 점들 |
| brief | brieven | 편지들 (f→v) |

### -s 복수형 예시

| 단수 | 복수 | 뜻 |

### -'s 복수형 예시

### 철자 변화에 주의

네덜란드어의 핵심 철자 규칙입니다. 이 규칙을 이해하면 복수형뿐 아니라 동사 활용에서도 큰 도움이 됩니다.

#### 규칙 1: 단모음 보호 — 자음 겹침

단모음(a, e, i, o, u를 한 번만 쓴 것) + 자음 하나인 경우, 복수형 -en을 붙이면 음절이 열리면서 장모음으로 발음될 위험이 있습니다. 이를 방지하기 위해 **자음을 겹쳐 씁니다**.

| 단수 | 복수 | 설명 |
|------|------|------|
| man | ma**nn**en | 단모음 a 유지 → n을 겹침 |
| bos | bo**ss**en | 단모음 o 유지 → s를 겹침 |
| kat | ka**tt**en | 단모음 a 유지 → t를 겹침 |
| bed | be**dd**en | 단모음 e 유지 → d를 겹침 |
| les | le**ss**en | 단모음 e 유지 → s를 겹침 |
| bus | bu**ss**en | 단모음 u 유지 → s를 겹침 |
| fles | fle**ss**en | 단모음 e 유지 → s를 겹침 |
| pot | po**tt**en | 단모음 o 유지 → t를 겹침 |
| zon | zo**nn**en | 단모음 o 유지 → n을 겹침 |
| bal | ba**ll**en | 단모음 a 유지 → l을 겹침 |

#### 규칙 2: 장모음 유지 — 겹자음 줄임

장모음(aa, oo, uu, ee 등 겹모음)은 음절이 열리면 모음 하나로도 장모음을 나타낼 수 있으므로, **겹자음을 하나로 줄입니다**.

| 단수 | 복수 | 설명 |
|------|------|------|
| maan | ma**n**en | 장모음 aa → 열린 음절에서 a 하나로 충분 |
| boot | bo**t**en | 장모음 oo → 열린 음절에서 o 하나로 충분 |
| naam | na**m**en | 장모음 aa → a |
| straat | stra**t**en | 장모음 aa → a |
| school | scho**l**en | 장모음 oo → o |
| duur | du**r**en | 장모음 uu → u |
| been | be**n**en | 장모음 ee → e |
| week | we**k**en | 장모음 ee → e |
| buur | bu**r**en | 장모음 uu → u |

#### 규칙 3: f → v, s → z 변화

일부 단어에서 어말의 f와 s가 복수형에서 각각 v와 z로 바뀝니다.

| 단수 | 복수 | 변화 |
|------|------|------|
| huis | hu**iz**en | s → z |
| brief | br**iev**en | f → v |
| kaas | ka**z**en | s → z |
| prijs | pr**ijz**en | s → z |
| neus | ne**uz**en | s → z |
| golf | go**lv**en | f → v |
| dief | d**iev**en | f → v |
| druif | dr**uiv**en | f → v |
| reis | re**iz**en | s → z |
| graaf | gra**v**en | f → v |

> **참고**: 모든 -f/-s 단어가 이렇게 변하는 것은 아닙니다. 예를 들어 brief → brieven이지만, dak → daken (변화 없음)입니다. 자주 쓰는 단어의 복수형은 개별적으로 외우는 것이 좋습니다.

### 불규칙 복수형

일부 명사는 일반 규칙을 따르지 않습니다. 이들은 개별적으로 암기해야 합니다.

| 단수 | 복수 | 뜻 |
|------|------|-----|
| kind | **kinder**en | 아이들 (-eren 복수) |
| ei | **eier**en | 달걀들 (-eren 복수) |
| lied | **lieder**en | 노래들 (-eren 복수) |
| volk | **volker**en | 민족들 (-eren 복수) |
| stad | **sted**en | 도시들 (모음 변화) |
| lid | **led**en | 회원들 (모음 변화) |
| schip | **schep**en | 배들 (모음 변화) |
| museum | muse**a** | 박물관들 (라틴어식) |
| centrum | centr**a** | 센터들 (라틴어식) |

### 복합 명사의 복수형

네덜란드어는 복합 명사(합성어)가 매우 많습니다. 복합 명사의 복수형은 **마지막 단어**만 복수형으로 바꿉니다.

| 단수 | 복수 | 뜻 |
|------|------|-----|
| schoolboek | schoolboek**en** | 교과서들 |
| voetbalwedstrijd | voetbalwedstrijd**en** | 축구 경기들 |
| huisdier | huisdier**en** | 애완동물들 |
| slaapkamer | slaapkamer**s** | 침실들 |
| postkantoor | postkanto**r**en | 우체국들 (oo→o) |
| boekenkast | boekenkast**en** | 책장들 |
| zonnebril | zonnebril**len** | 선글라스들 |

### 복수형이 없거나 특별한 명사

- **항상 복수인 명사**: de kleren (옷), de kosten (비용), de ouders (부모)
- **셀 수 없는 명사** (복수형 없음): het geld (돈), de melk (우유), het weer (날씨), de rijst (쌀)
- **단복수 동형**: het schaap → de schapen (양), 하지만 een vis → vissen (물고기), een mens → mensen (사람)

### 한국인이 자주 하는 실수

1. **철자 변화를 무시하는 실수**:
   - (X) man → manen → (O) man → ma**nn**en
   - (X) boot → booten → (O) boot → bo**t**en

2. **f/s → v/z 변화를 잊는 실수**:
   - (X) huis → huisen → (O) huis → hu**iz**en
   - (X) brief → briefen → (O) brief → br**iev**en

3. **-'s와 -s를 혼동하는 실수**:
   - (X) auto → autos → (O) auto → auto**'s** (모음으로 끝나는 외래어는 어포스트로피 필요)
   - (O) tafel → tafels (자음으로 끝나는 -el 단어는 그냥 -s)

4. **불규칙 복수형을 규칙적으로 만드는 실수**:
   - (X) kind → kinden → (O) kind → **kinderen**
   - (X) stad → stadden → (O) stad → **steden**

> **규칙 정리**:
> - 단모음 + 자음 하나 → 복수형에서 자음을 겹쳐 단모음을 유지
> - 장모음(겹모음) + 자음 겹침 → 복수형에서 자음 하나로 줄임
> - 일부 단어에서 f → v, s → z 변화
> - 불규칙 복수형은 개별 암기 필요
