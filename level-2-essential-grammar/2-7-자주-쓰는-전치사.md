# 2-7. 자주 쓰는 전치사 (Voorzetsels)

전치사는 문장에서 명사/대명사와 결합하여 장소, 시간, 방향, 수단 등을 나타냅니다. 한국어의 조사(-에, -에서, -으로, -와 등)에 해당하지만, 네덜란드어 전치사는 명사 **앞에** 옵니다 (한국어 조사는 명사 뒤에 옴).

### 장소 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **in** | ~안에 | in de kamer (방 안에), in het water (물 안에) |
| **op** | ~위에 | op de tafel (테이블 위에), op straat (거리에서) |
| **aan** | ~에 (접촉/붙어) | aan de muur (벽에), aan tafel (식탁에 앉아) |
| **bij** | ~옆에, ~근처에 | bij het station (역 근처에), bij mij (내 옆에) |
| **voor** | ~앞에 | voor het huis (집 앞에), voor de deur (문 앞에) |
| **achter** | ~뒤에 | achter het huis (집 뒤에), achter de auto (차 뒤에) |
| **naast** | ~옆에 | naast de school (학교 옆에), naast mij (내 옆에) |
| **boven** | ~위에 (떨어져) | boven de deur (문 위에), boven de wolken (구름 위에) |
| **onder** | ~아래에 | onder de tafel (테이블 아래에), onder het bed (침대 아래에) |
| **tussen** | ~사이에 | tussen de huizen (집들 사이에), tussen ons (우리 사이에) |
| **tegenover** | ~맞은편에 | tegenover het station (역 맞은편에) |
| **binnen** | ~안에 (건물 등) | binnen het gebouw (건물 안에) |
| **buiten** | ~밖에 | buiten de stad (도시 밖에) |

#### in vs op 구분

| in (내부, 둘러싸인 공간) | op (표면, 위) |
|------------------------|-------------|
| in de kamer (방 안에) | op de tafel (테이블 위에) |
| in het water (물 속에) | op het dak (지붕 위에) |
| in de stad (도시 안에) | op straat (거리에서) |
| in de trein (기차 안에) | op de fiets (자전거를 타고) |
| in bed (침대에 누워) | op school (학교에서) |
| in het ziekenhuis (병원에 입원) | op kantoor (사무실에서) |

> **주의**: in/op의 사용은 항상 논리적이지 않습니다. "op school" (학교에서), "op kantoor" (사무실에서)처럼 표면 위가 아니라 장소를 나타내는 경우도 있습니다. 관용적 표현으로 외워야 합니다.

### 방향/이동 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **naar** | ~으로 (방향) | naar school (학교로), naar Amsterdam (암스테르담으로) |
| **van** | ~에서 (출발점) | van Amsterdam (암스테르담에서) |
| **uit** | ~에서 (출신/나옴) | uit Korea (한국에서 온), uit de kast (옷장에서 꺼내) |
| **door** | ~을 통해 | door de stad (도시를 통해), door het park (공원을 지나) |
| **langs** | ~을 따라/지나 | langs de rivier (강을 따라), langs het huis (집을 지나) |
| **tot** | ~까지 | tot Amsterdam (암스테르담까지), tot morgen (내일까지) |
| **richting** | ~방향으로 | richting het centrum (시내 방향으로) |

#### naar vs aan vs in (장소 이동)

| naar (이동 방향) | aan (도착/접촉) | in (안에 들어감) |
|----------------|----------------|----------------|
| Ik ga **naar** school. | Ik zit **aan** tafel. | Ik ben **in** de kamer. |
| Ik ga **naar** Amsterdam. | Ik woon **aan** zee. | Ik stap **in** de trein. |
| Ik ga **naar** huis. | Hij komt **aan** (도착하다). | Ik zwem **in** het water. |

### 시간 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **om** | ~시에 (시각) | om drie uur (3시에), om half vijf (4시 반에) |
| **in** | ~에 (월/계절/년) | in januari (1월에), in de zomer (여름에), in 2024 (2024년에) |
| **op** | ~에 (요일/날짜) | op maandag (월요일에), op 5 mei (5월 5일에) |
| **'s** | ~에 (하루 시간대) | 's ochtends (아침에), 's middags (오후에), 's avonds (저녁에), 's nachts (밤에) |
| **na** | ~후에 | na het eten (식사 후에), na de les (수업 후에) |
| **voor** | ~전에 | voor het eten (식사 전에), voor vijf uur (5시 전에) |
| **tijdens** | ~동안 | tijdens de les (수업 동안), tijdens het eten (식사 동안) |
| **sinds** | ~이래로 | sinds gisteren (어제 이래로), sinds 2020 (2020년 이래로) |
| **tot** | ~까지 | tot vijf uur (5시까지), tot morgen (내일까지) |
| **van...tot** | ~부터 ~까지 | van negen tot vijf (9시부터 5시까지) |
| **tegen** | ~즈음에 | tegen de avond (저녁 즈음에) |

시간 전치사 사용 예문:

| 예문 | 번역 |
|------|------|
| De les begint **om** negen uur. | 수업은 9시에 시작해. |
| Ik ga **in** juli **naar** Korea. | 나는 7월에 한국에 가. |
| **Op** zaterdag werk ik niet. | 토요일에는 일하지 않아. |
| Ik drink **'s ochtends** altijd koffie. | 아침에는 항상 커피를 마셔. |
| **Na** het werk ga ik sporten. | 일 끝나고 운동하러 가. |
| Ik woon hier **sinds** twee jaar. | 나는 여기 2년 전부터 살아. |
| De winkel is open **van** 9 **tot** 5. | 가게는 9시부터 5시까지 열어. |

### 기타 중요한 전치사

| 네덜란드어 | 한국어 | 예시 |
|-----------|-------|------|
| **met** | ~와 함께, ~로 (수단) | met mijn vriend (친구와 함께), met de trein (기차로) |
| **voor** | ~을 위해 | voor jou (너를 위해), voor het werk (일을 위해) |
| **van** | ~의 | het boek van Jan (Jan의 책) |
| **over** | ~에 대해 | over het weer (날씨에 대해), over politiek (정치에 대해) |
| **zonder** | ~없이 | zonder geld (돈 없이), zonder jou (너 없이) |
| **tegen** | ~에게 (말), ~에 반대 | tegen mij (나에게), tegen de muur (벽에 기대어) |
| **volgens** | ~에 따르면 | volgens mij (내 생각에), volgens de krant (신문에 따르면) |
| **ondanks** | ~에도 불구하고 | ondanks het weer (날씨에도 불구하고) |
| **dankzij** | ~덕분에 | dankzij jou (너 덕분에) |
| **vanwege** | ~때문에 | vanwege de regen (비 때문에) |
| **behalve** | ~을 제외하고 | behalve mij (나를 제외하고) |
| **per** | ~당, ~로 (수단) | per e-mail (이메일로), per dag (하루에) |

### 전치사 + 동사 고정 표현

네덜란드어에서 많은 동사가 특정 전치사와 결합합니다. 이것은 외워야 합니다.

| 동사 + 전치사 | 뜻 | 예문 |
|-------------|-----|------|
| denken **aan** | ~을 생각하다 | Ik denk **aan** je. (너를 생각해.) |
| wachten **op** | ~을 기다리다 | Ik wacht **op** de bus. (버스를 기다려.) |
| luisteren **naar** | ~을 듣다 | Ik luister **naar** muziek. (음악을 들어.) |
| kijken **naar** | ~을 보다 | Ik kijk **naar** een film. (영화를 봐.) |
| zoeken **naar** | ~을 찾다 | Ik zoek **naar** mijn sleutels. (열쇠를 찾아.) |
| houden **van** | ~을 좋아하다 | Ik houd **van** je. (너를 사랑해.) |
| praten **over** | ~에 대해 이야기하다 | We praten **over** het werk. (일에 대해 이야기해.) |
| beginnen **met** | ~을 시작하다 | Ik begin **met** de les. (수업을 시작해.) |
| stoppen **met** | ~을 그만두다 | Hij stopt **met** roken. (그는 담배를 끊어.) |
| dromen **van** | ~을 꿈꾸다 | Ik droom **van** vakantie. (휴가를 꿈꿔.) |
| lijken **op** | ~을 닮다 | Hij lijkt **op** zijn vader. (그는 아버지를 닮았어.) |
| vragen **om** | ~을 요청하다 | Ik vraag **om** hulp. (도움을 요청해.) |
| feliciteren **met** | ~을 축하하다 | Ik feliciteer je **met** je verjaardag! (생일 축하해!) |
| lachen **om** | ~을 보고 웃다 | We lachen **om** de grap. (농담에 웃어.) |
| rekenen **op** | ~을 기대하다/믿다 | Je kunt **op** mij rekenen. (나를 믿어도 돼.) |

### 전치사 축약형 (er + 전치사)

사물을 가리킬 때 "전치사 + het"을 쓰지 않고 **er + 전치사**를 씁니다:

| 원래 형태 (사용 X) | 축약형 (사용 O) | 뜻 |
|-------------------|----------------|-----|
| op het | **erop** | 그 위에 / 그것에 대해 |
| aan het | **eraan** | 그것에 |
| in het | **erin** | 그 안에 |
| over het | **erover** | 그것에 대해 |
| van het | **ervan** | 그것의 / 그것으로부터 |
| mee het | **ermee** | 그것으로 / 그것과 함께 |
| naar het | **ernaar** | 그것을 향해 |

예문:

| 예문 | 번역 |
|------|------|
| Ik denk **eraan**. | 나는 그것을 생각해. (ik denk aan het (X)) |
| Ik kijk **ernaar**. | 나는 그것을 봐. |
| Hij praat **erover**. | 그는 그것에 대해 이야기해. |
| Ik wacht **erop**. | 나는 그것을 기다려. |
| Ik houd **ervan**. | 나는 그것을 좋아해. |
| Wat doe je **ermee**? | 그것으로 뭐 해? |

> **규칙**: 사물에 대해 말할 때 → er + 전치사 (eraan, erop, erover...)
> 사람에 대해 말할 때 → 전치사 + 대명사 (aan hem, op haar, over hen...)
> - Ik denk **aan hem**. (나는 그를 생각해.) — 사람
> - Ik denk **eraan**. (나는 그것을 생각해.) — 사물

### 연습 대화

```
A: Waar ga je naartoe?                   (어디 가?)
B: Ik ga naar de supermarkt.             (슈퍼마켓에 가.)
A: Mag ik met je mee?                    (나도 같이 가도 돼?)
B: Ja, natuurlijk. We gaan met de        (응, 당연하지. 우리 자전거로 가자.)
   fiets.
A: Hoelang duurt het?                    (얼마나 걸려?)
B: Ongeveer tien minuten met de fiets.   (자전거로 약 10분.)
   De supermarkt is naast het station.   (슈퍼마켓은 역 옆에 있어.)
A: Na de boodschappen wil ik nog         (장 보고 나서 공원에도 가고 싶어.)
   naar het park.
B: Goed idee! We gaan eerst naar de      (좋은 생각! 먼저 가게에 가고,
   winkel, en dan door het park          그 다음에 공원을 지나
   naar huis.                            집으로 가자.)
```

### 한국인이 자주 하는 실수

1. **전치사 위치를 한국어 조사처럼 뒤에 놓는 실수**:
   - (X) school **naar** → (O) **naar** school (전치사는 명사 앞에!)

2. **naar와 in 혼동**:
   - Ik ga **naar** Amsterdam. (암스테르담으로 간다.) — 이동/방향
   - Ik woon **in** Amsterdam. (암스테르담에 산다.) — 위치/상태

3. **시간 전치사 혼동 (om/in/op)**:
   - **om** drie uur (3시에) — 정확한 시각
   - **in** januari (1월에) — 월/계절/년
   - **op** maandag (월요일에) — 요일/날짜

4. **동사 + 전치사 고정 표현을 모르는 실수**:
   - (X) Ik denk **over** je → (O) Ik denk **aan** je (denken aan = ~을 생각하다)
   - (X) Ik luister muziek → (O) Ik luister **naar** muziek (luisteren naar = ~을 듣다)

5. **er + 전치사를 사용하지 않는 실수**:
   - (X) Ik denk aan **het**. → (O) Ik denk **eraan**. (사물은 er + 전치사 사용)
