# 2-5. 동사 현재시제 (Tegenwoordige tijd)

네덜란드어 동사 활용은 영어보다 복잡하지만 한국어만큼 복잡하지는 않습니다. 기본 규칙을 익히면 대부분의 동사를 올바르게 활용할 수 있습니다.

### 규칙 동사 활용 패턴

동사 원형(infinitief)에서 **-en**을 제거하면 **어간(stam)**이 됩니다.

예: **werken** (일하다) → 어간: **werk**

| 인칭 | 활용 | 예시 (werken) |
|------|------|-------------|
| ik | 어간 | ik **werk** |
| jij/je | 어간 + **t** | jij **werkt** |
| u | 어간 + **t** | u **werkt** |
| hij/zij/het | 어간 + **t** | hij **werkt** |
| wij/we | 원형 | wij **werken** |
| jullie | 원형 | jullie **werken** |
| zij (복수) | 원형 | zij **werken** |

### 어간(stam) 만드는 법 상세

원형에서 -en을 빼면 어간이 되지만, **철자 규칙**에 주의해야 합니다. 복수형 규칙과 동일한 원리입니다.

#### 기본 어간

| 원형 | 어간 | 뜻 |
|------|------|-----|
| werken | werk | 일하다 |
| leren | leer | 배우다 |
| wonen | woon | 살다 |
| koken | kook | 요리하다 |
| praten | praat | 이야기하다 |
| spelen | speel | 놀다/연주하다 |
| horen | hoor | 듣다 |
| vragen | vraag | 질문하다 |
| wachten | wacht | 기다리다 |
| antwoorden | antwoord | 대답하다 |

#### 철자 변화가 있는 어간

**장모음 유지**: 원형에서 모음이 하나인데 장모음이면, 어간에서 모음을 겹쳐 씁니다.

| 원형 | 어간 | 설명 |
|------|------|------|
| lev**en** | lee**f** | "살다" — 열린 음절의 e → 닫힌 음절에서 ee로, v → f |
| schrijv**en** | schrij**f** | "쓰다" — v → f |
| gev**en** | gee**f** | "주다" — 열린 e → ee, v → f |
| lezen | lee**s** | "읽다" — 열린 e → ee, z → s |
| reizen | rei**s** | "여행하다" — z → s |
| verhuiz**en** | verhui**s** | "이사하다" — z → s |
| mak**en** | maak | "만들다" — 열린 a → aa |
| lek**en** | leek | 닫힌 음절에서 장모음 유지 |

> **핵심 규칙**: 어간이 단음절이 되면서 닫힌 음절이 되므로:
> - 장모음은 겹모음으로 써야 합니다: leven → leef (e→ee), maken → maak (a→aa)
> - 어말의 v → f, z → s로 바뀝니다 (네덜란드어에서 단어 끝에 v, z가 올 수 없음)

#### 어간이 -t로 끝나는 동사

어간이 이미 -t로 끝나면, jij/hij 형태에 -t를 더 추가하지 않습니다.

| 원형 | 어간 | jij 형태 | 설명 |
|------|------|---------|------|
| zitten | zit | jij **zit** | -t를 추가하지 않음 (X: zitt) |
| praten | praat | jij **praat** | 이미 -t로 끝남 |
| wachten | wacht | jij **wacht** | 이미 -t로 끝남 |
| chatten | chat | jij **chat** | 이미 -t로 끝남 |
| eten | eet | jij **eet** | 이미 -t로 끝남 |

#### 어간이 -d로 끝나는 동사

어간이 -d로 끝나면 jij/hij 형태에 -t를 추가합니다. 발음은 /t/로 합니다.

| 원형 | 어간 | jij 형태 | 발음 |
|------|------|---------|------|
| vinden | vind | jij **vindt** | /vint/ |
| worden | word | jij **wordt** | /wort/ |
| houden | houd | jij **houdt** | /hout/ |
| rijden | rijd | jij **rijdt** | /rijt/ |

### 도치(역순)에서 jij의 특별 규칙

**jij**가 동사 뒤에 오면 **-t**가 사라집니다. 이것은 네덜란드어의 매우 중요한 규칙입니다!

- Jij werk**t** hard. (너는 열심히 일해.)
- Werk jij hard? (너 열심히 일해?) ← **t 탈락!**

더 많은 예시:

| 정상 어순 | 도치 (의문문) |
|----------|-------------|
| Jij **leest** een boek. | **Lees** jij een boek? |
| Jij **woont** in Amsterdam. | **Woon** jij in Amsterdam? |
| Jij **spreekt** Nederlands. | **Spreek** jij Nederlands? |
| Jij **hebt** een auto. | **Heb** jij een auto? |
| Jij **vindt** het leuk. | **Vind** jij het leuk? |
| Jij **eet** veel. | **Eet** jij veel? (어간이 -t로 끝나면 변화 없음) |

> **주의**: 이 규칙은 **jij/je**에만 적용됩니다. u나 hij에게는 적용되지 않습니다.
> - Werkt u hard? (t 유지)
> - Werkt hij hard? (t 유지)

### 주요 불규칙 동사

#### zijn (이다) — 가장 중요한 동사

| 인칭 | 활용 | 예문 |
|------|------|------|
| ik | **ben** | Ik **ben** student. (나는 학생이야.) |
| jij/je | **bent** | Jij **bent** laat. (너 늦었어.) |
| u | **bent** / **is** | U **bent** welkom. (환영합니다.) |
| hij/zij/het | **is** | Het **is** mooi weer. (날씨가 좋아.) |
| wij/we | **zijn** | Wij **zijn** vrienden. (우리는 친구야.) |
| jullie | **zijn** | Jullie **zijn** klaar. (너희 다 됐어.) |
| zij (복수) | **zijn** | Zij **zijn** uit Korea. (그들은 한국에서 왔어.) |

zijn의 일상 표현:
- Hoe **is** het? (어때?)
- Waar **ben** je? (너 어디야?)
- **Ben** jij Koreaans? (너 한국인이야?) — 도치에서도 t 탈락 없음 (불규칙이므로)
- Het **is** half drie. (2시 반이야.)
- Er **zijn** veel mensen. (사람이 많아.)

#### hebben (가지다)

| 인칭 | 활용 | 예문 |
|------|------|------|
| ik | **heb** | Ik **heb** een vraag. (질문이 있어요.) |
| jij/je | **hebt** | Jij **hebt** een mooie fiets. (너 예쁜 자전거 가지고 있네.) |
| u | **hebt** / **heeft** | **Heeft** u een moment? (시간 좀 있으세요?) |
| hij/zij/het | **heeft** | Hij **heeft** twee kinderen. (그는 아이가 둘이야.) |
| wij/we | **hebben** | Wij **hebben** honger. (우리 배고파.) |
| jullie | **hebben** | Jullie **hebben** geluk. (너희 운이 좋아.) |
| zij (복수) | **hebben** | Zij **hebben** geen geld. (그들은 돈이 없어.) |

hebben의 일상 표현:
- Ik **heb** honger. (나 배고파.) — 한국어: "배고프다" (형용사), 네덜란드어: "배고픔을 가지다" (동사)
- Ik **heb** dorst. (나 목말라.)
- Ik **heb** het koud. (나 추워.)
- Ik **heb** het warm. (나 더워.)
- **Heb** je zin in koffie? (커피 마실래?)

#### 조동사 (Hulpwerkwoorden)

조동사는 다른 동사와 함께 쓰여 의미를 보충합니다. 조동사 + 원형동사 구조로 사용됩니다.

| 원형 | 뜻 | ik | jij | hij/zij | wij |
|------|-----|-----|------|---------|------|
| zullen | ~할 것이다 | zal | zult/zal | zal | zullen |
| kunnen | ~할 수 있다 | kan | kunt/kan | kan | kunnen |
| mogen | ~해도 된다 | mag | mag | mag | mogen |
| moeten | ~해야 한다 | moet | moet | moet | moeten |
| willen | ~하고 싶다 | wil | wilt/wil | wil | willen |

조동사 예문:

| 예문 | 번역 |
|------|------|
| Ik **kan** Nederlands spreken. | 나는 네덜란드어를 할 수 있어. |
| **Mag** ik hier zitten? | 여기 앉아도 되나요? |
| Je **moet** hard studeren. | 너는 열심히 공부해야 해. |
| Wij **willen** naar Amsterdam gaan. | 우리는 암스테르담에 가고 싶어. |
| Het **zal** morgen regenen. | 내일 비가 올 거야. |
| **Kun** jij zwemmen? | 너 수영할 수 있어? (도치 t 탈락) |
| Je **mag** hier niet roken. | 여기서 담배 피면 안 돼. |
| Ik **moet** om 8 uur beginnen. | 나는 8시에 시작해야 해. |
| **Wil** je koffie of thee? | 커피 마실래, 차 마실래? |
| Ze **kunnen** goed voetballen. | 그들은 축구를 잘 할 수 있어. |

> **조동사 + 원형 어순**: 조동사가 두 번째 위치에 오고, 원형 동사는 문장 **끝**에 갑니다.
> - Ik **wil** een boek **lezen**. (나는 책을 읽고 싶어.)
> - Hij **moet** morgen vroeg **opstaan**. (그는 내일 일찍 일어나야 해.)
> - Wij **kunnen** goed Nederlands **spreken**. (우리는 네덜란드어를 잘 할 수 있어.)

### 연습 대화

```
A: Waar werk je?                         (어디서 일해?)
B: Ik werk bij een IT-bedrijf.           (IT 회사에서 일해.)
   En jij? Wat doe jij?                  (너는? 뭐 해?)
A: Ik studeer nog. Ik leer Nederlands.   (나는 아직 학생이야. 네덜란드어를 배워.)
B: Spreek je al een beetje?              (벌써 좀 할 줄 아니?)
A: Ja, ik kan een beetje spreken.        (응, 좀 할 수 있어.)
   Maar ik moet meer oefenen.            (하지만 더 연습해야 해.)
B: Wil je samen oefenen?                 (같이 연습할래?)
A: Ja, graag! Wanneer heb je tijd?       (응, 좋아! 언제 시간 있어?)
B: Ik heb morgenavond tijd.              (내일 저녁에 시간 있어.)
```

### 한국인이 자주 하는 실수

1. **도치에서 -t 탈락을 잊는 실수**:
   - (X) Werk**t** jij morgen? → (O) **Werk** jij morgen?
   - (X) Heb**t** jij een auto? → (O) **Heb** jij een auto?

2. **어간 철자 실수**:
   - (X) Ik **lef** → (O) Ik **leef** (leven의 어간은 leef)
   - (X) Ik **schrijv** → (O) Ik **schrijf** (v→f)
   - (X) Ik **lez** → (O) Ik **lees** (z→s, 열린 e→ee)

3. **조동사 뒤에 활용형을 쓰는 실수**: 조동사 뒤의 동사는 항상 원형!
   - (X) Ik wil **werk**. → (O) Ik wil **werken**.
   - (X) Hij kan **spreekt**. → (O) Hij kan **spreken**.

4. **어간이 -t로 끝나는 동사에 -t를 더 붙이는 실수**:
   - (X) Jij **zitt** → (O) Jij **zit**
   - (X) Hij **eet**t → (O) Hij **eet**

5. **현재시제의 의미 차이**: 네덜란드어 현재시제는 영어의 현재진행형도 포함합니다.
   - Ik **lees** een boek. = I read a book. / I am reading a book.
   - 네덜란드어에는 별도의 진행형이 없습니다!
