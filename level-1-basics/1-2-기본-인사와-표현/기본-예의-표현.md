# 기본 예의 표현

| 네덜란드어 | 발음 | 한국어 | 사용 상황 |
|-----------|------|-------|----------|
| Alstublieft | 알스튀블리프트 | ~주세요 / 여기 있습니다 | 격식 — 뭔가를 건네거나 요청할 때 |
| Alsjeblieft | 알셔블리프트 | ~줘 / 여기 | 비격식 |
| Dank u wel | 당크 위 뷸 | 감사합니다 | 격식 |
| Dank je wel | 당크 여 뷸 | 고마워 | 비격식 |
| Bedankt | 버당크트 | 고마워요 | 일반 |
| Hartelijk bedankt | 하르틀럭 버당크트 | 진심으로 감사합니다 | 격식 |
| Heel erg bedankt | 헬 에르흐 버당크트 | 정말 감사합니다 | 강조 |
| Graag gedaan | 흐라흐 허단 | 천만에요 | 일반 |
| Geen dank | 헤인 당크 | 뭘요, 별말씀을 | 일반 |
| Geen probleem | 헤인 프로블렘 | 문제 없어요 | 비격식 |
| Sorry | 소리 | 미안해 / 죄송해 | 일반 (가벼운 사과) |
| Pardon | 파르돈 | 실례합니다 | 일반 (길을 지나갈 때 등) |
| Neem me niet kwalijk | 넴 머 닛 크발럭 | 실례합니다 / 죄송합니다 | 격식 |
| Het spijt me | 헷 스페이트 머 | 미안합니다 (진심) | 격식/진지한 사과 |
| Excuus / Excuseer me | 엑스퀴스 / 엑스퀴제이르 머 | 실례합니다 | 격식 |
| Ja | 야 | 예 | - |
| Nee | 네이 | 아니오 | - |
| Ja hoor | 야 호르 | 네, 물론이요 | 동의 강조 |
| Nee hoor | 네이 호르 | 아니에요, 아닌데요 | 부정 부드럽게 |
| Natuurlijk | 나튀를럭 | 당연하죠 | 긍정 강조 |
