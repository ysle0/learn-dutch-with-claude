# 1-5. 요일, 월, 계절

### 요일 (Dagen van de week)

| 네덜란드어 | 발음 | 한국어 | 약어 |
|-----------|------|-------|------|
| maandag | 만닥 | 월요일 | ma |
| dinsdag | 딘스닥 | 화요일 | di |
| woensdag | 운스닥 | 수요일 | wo |
| donderdag | 돈더르닥 | 목요일 | do |
| vrijdag | 브레이닥 | 금요일 | vr |
| zaterdag | 자터르닥 | 토요일 | za |
| zondag | 존닥 | 일요일 | zo |

> **참고**: 네덜란드어에서 요일은 **소문자**로 씁니다 (영어와 다름!).

#### 요일 관련 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| Welke dag is het vandaag? | 오늘 무슨 요일이에요? |
| Vandaag is het maandag. | 오늘은 월요일이에요. |
| Gisteren was het zondag. | 어제는 일요일이었어요. |
| Morgen is het dinsdag. | 내일은 화요일이에요. |
| op maandag | 월요일에 |
| elke maandag / iedere maandag | 매주 월요일 |
| 's maandags | 월요일마다 (반복) |
| afgelopen maandag | 지난 월요일 |
| aankomende vrijdag | 다가오는 금요일 |
| vorige week dinsdag | 지난주 화요일 |
| volgende week woensdag | 다음주 수요일 |
| het weekend | 주말 |
| doordeweeks | 평일에 |
| in het weekend | 주말에 |

```
Wat doe je op zaterdag?          (토요일에 뭐 해?)
Ik werk van maandag tot vrijdag. (월요일부터 금요일까지 일해요.)
Op zondag slaap ik lang.         (일요일에는 늦잠을 자요.)
Elke woensdag ga ik sporten.     (매주 수요일에 운동해요.)
Zaterdag ga ik boodschappen doen. (토요일에 장 보러 가요.)
```

### 월 (Maanden)

| 네덜란드어 | 발음 | 한국어 | 약어 |
|-----------|------|-------|------|
| januari | 야뉘아리 | 1월 | jan. |
| februari | 페브뤼아리 | 2월 | feb. |
| maart | 마르트 | 3월 | mrt. |
| april | 아프릴 | 4월 | apr. |
| mei | 메이 | 5월 | mei |
| juni | 유니 | 6월 | jun. |
| juli | 율리 | 7월 | jul. |
| augustus | 아우휘스튀스 | 8월 | aug. |
| september | 셉템버르 | 9월 | sep. |
| oktober | 옥토버르 | 10월 | okt. |
| november | 노벰버르 | 11월 | nov. |
| december | 데셈버르 | 12월 | dec. |

> **참고**: 월 이름도 네덜란드어에서는 **소문자**로 씁니다.

#### 월 관련 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| in januari | 1월에 |
| in de maand maart | 3월 달에 |
| sinds april | 4월부터 |
| tot en met juni | 6월까지 (포함) |
| van september tot december | 9월부터 12월까지 |
| begin januari | 1월 초 |
| midden februari | 2월 중순 |
| eind maart | 3월 말 |
| deze maand | 이번 달 |
| vorige maand | 지난달 |
| volgende maand | 다음 달 |
| twee maanden geleden | 두 달 전 |
| over drie maanden | 세 달 후 |

```
Mijn verjaardag is in mei.       (제 생일은 5월이에요.)
In december is het Kerstmis.     (12월에는 크리스마스예요.)
Het regent veel in november.     (11월에는 비가 많이 와요.)
In augustus gaan we op vakantie.  (8월에 우리는 휴가를 가요.)
```

### 계절 (Seizoenen)

| 네덜란드어 | 발음 | 한국어 | 기간 |
|-----------|------|-------|------|
| de lente / het voorjaar | 더 렌터 / 헷 보르야르 | 봄 | 3~5월 |
| de zomer | 더 조머르 | 여름 | 6~8월 |
| de herfst / het najaar | 더 헤르프스트 / 헷 나야르 | 가을 | 9~11월 |
| de winter | 더 빈터르 | 겨울 | 12~2월 |

#### 계절 관련 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| in de lente / in het voorjaar | 봄에 |
| in de zomer | 여름에 |
| in de herfst / in het najaar | 가을에 |
| in de winter | 겨울에 |
| deze zomer | 이번 여름 |
| vorige winter | 지난 겨울 |
| volgende lente | 다음 봄 |

#### 계절별 네덜란드 특색

```
De lente (봄):
- De tulpen bloeien. (튤립이 핍니다.)
- Koningsdag is op 27 april. (왕의 날은 4월 27일입니다.)
- Het wordt warmer. (날씨가 따뜻해집니다.)

De zomer (여름):
- We gaan naar het strand. (해변에 갑니다.)
- Het is lang licht. (해가 오래 있습니다.)
- Veel mensen gaan op vakantie. (많은 사람들이 휴가를 갑니다.)
- Terrassen zijn open. (야외 테라스가 열립니다.)

De herfst (가을):
- De bladeren vallen. (나뭇잎이 떨어집니다.)
- Het waait hard. (바람이 세게 붑니다.)
- Sint-Maarten is op 11 november. (성 마르틴 축제는 11월 11일입니다.)
- Het wordt vroeg donker. (일찍 어두워집니다.)

De winter (겨울):
- Sinterklaas komt op 5 december. (신터클라스가 12월 5일에 옵니다.)
- Kerstmis is op 25 december. (크리스마스는 12월 25일입니다.)
- Oud en Nieuw (새해 전야/새해)
- Soms kunnen we schaatsen. (가끔 스케이트를 탈 수 있습니다.)
- Oliebollen eten! (올리볼런을 먹습니다!)
```

```
Ik woon hier al drie jaar.          (여기 산 지 벌써 3년이에요.)
Ik ga twee weken op vakantie.        (2주간 휴가를 갑니다.)
Het duurt een kwartier.              (15분 걸려요.)

Ik heb een uur gewacht.              (한 시간 기다렸어요.)
Ik ben hier sinds vorig jaar.        (작년부터 여기 있어요.)
```
