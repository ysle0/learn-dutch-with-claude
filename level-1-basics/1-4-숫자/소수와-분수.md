# 소수와 분수

#### 소수 (Decimale getallen)

| 표기 | 네덜란드어 | 한국어 |
|------|-----------|-------|
| 0,5 | nul komma vijf | 영 쉼표 오 |
| 1,5 | een komma vijf / anderhalve | 일 점 오 / 하나 반 |
| 2,75 | twee komma vijfenzeventig | 이 점 칠오 |
| 3,14 | drie komma veertien | 삼 점 일사 |
| 0,1 | nul komma een | 영 쉼표 일 |
| 0,01 | nul komma nul een | 영 쉼표 영일 |
| 10,33 | tien komma drieëndertig | 십 점 삼삼 |

> **참고**: 네덜란드어에서 소수점은 **komma(쉼표)** 라고 읽습니다 (영어의 point와 다름).

#### 분수 (Breuken)

| 표기 | 네덜란드어 | 한국어 |
|------|-----------|-------|
| ½ | een half | 이분의 일 |
| ⅓ | een derde | 삼분의 일 |
| ¼ | een kwart / een vierde | 사분의 일 |
| ¾ | driekwart / drie vierde | 사분의 삼 |
| ⅕ | een vijfde | 오분의 일 |
| ⅔ | twee derde | 삼분의 이 |
| 1½ | anderhalve | 하나 반 |
| 2½ | tweeënhalf | 둘 반 |
