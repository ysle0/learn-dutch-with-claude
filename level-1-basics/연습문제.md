# 연습 문제

### 연습 1: 자기소개
네덜란드어로 자기소개를 해보세요 (이름, 출신, 거주지, 나이, 직업, 취미):

```
예시 답안:
Hallo! Ik heet ___. Ik kom uit Zuid-Korea en ik woon in ___.
Ik ben ___ jaar oud. Ik werk als ___ / Ik studeer ___.
In mijn vrije tijd ___ ik graag.
Ik leer Nederlands. Aangenaam!
```

### 연습 2: 숫자 읽기
1~20까지 네덜란드어로 세어보세요. 그리고 다음 숫자를 읽어보세요:
- 25, 38, 47, 63, 79, 84, 91, 100

### 연습 3: 요일과 달
- 오늘 요일과 이번 달을 네덜란드어로 말해보세요.
- "다음 주 금요일"을 네덜란드어로?
- "3월에"를 네덜란드어로?

### 연습 4: 번역하세요 (한국어 → 네덜란드어)
1. "안녕하세요, 만나서 반갑습니다."
2. "이것은 얼마예요?"
3. "감사합니다, 안녕히 계세요."
4. "저는 한국에서 왔습니다."
5. "네덜란드어를 조금 합니다."
6. "당신의 이름은 무엇입니까?"
7. "어디에 사세요?"
8. "오늘 날씨가 어때요?"

### 연습 5: 번역하세요 (네덜란드어 → 한국어)
1. Ik woon in Amsterdam sinds drie jaar.
2. Mijn broer is vijfentwintig jaar oud.
3. Hoeveel kost een kopje koffie?
4. Welke talen spreek je?
5. Waarom leer je Nederlands?
6. Elke zaterdag ga ik naar de markt.
7. Het is kwart over drie.
8. Mijn verjaardag is op tien mei.

### 연습 6: 대화 완성
빈칸을 채워 대화를 완성하세요:

```
A: Hallo! Hoe ___ je?
B: Ik ___ Minho. ___ jij?
A: Ik heet Sophie. ___ kom je vandaan?
B: Ik ___ uit Zuid-Korea. Ik ___ nu in Rotterdam.
A: Wat ___! Wat ___ je voor werk?
B: Ik ___ als ingenieur. En jij?
A: Ik ___ geneeskunde aan de universiteit.
B: ___! Hoe ___ ben je?
A: Ik ben drieëntwintig.
```

### 연습 7: 듣기 연습 준비
다음 문장을 소리 내어 읽어보세요 (발음에 주의):

```
1. Goedemorgen! Hoe gaat het met u?
2. Ik kom uit Zuid-Korea en ik woon in Den Haag.
3. Het is kwart voor vijf op vrijdag drieëntwintig maart.
4. Mijn telefoonnummer is nul zes, tweeëndertig, vijfenveertig, achtenzeventig.
5. Ik heb twee broers en een zus. Zij wonen in Seoul.
```
