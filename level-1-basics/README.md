### 핵심 발음 규칙

#### 열린 음절 vs 닫힌 음절

네덜란드어 발음의 가장 중요한 기초는 **음절의 열림/닫힘**입니다.

- **닫힌 음절**: 자음으로 끝남 → 모음이 **짧게** 발음됨
  - m**a**n (남자), b**e**d (침대), b**o**s (숲)
- **열린 음절**: 모음으로 끝남 → 모음이 **길게** 발음됨
  - n**a**-me (이름), l**e**-ven (살다), b**o**-men (나무들)

```
man (닫힌 → 짧은 a) vs maan (장모음 aa → 긴 a)
man → mannen (복수: 닫힌 유지 → nn으로 겹침)
maan → manen (복수: 열린 음절 → a 하나로 충분)
```

> **이것이 중요한 이유**: 모음의 길이에 따라 단어의 의미가 완전히 달라집니다!
> - man (남자) vs maan (달)
> - bos (숲) vs boos (화난)
> - mus (참새) vs muus/muur (벽)

#### 자음 조합 상세

| 조합 | 발음 | 예시 |
|------|------|------|
| **th** | ㅌ (영어 th와 다름!) | thee (차) → '테이', thuis (집) → '타위스' |
| **ph** | ㅍ | telefoon/phone → 거의 안 쓰임, f로 대체 |
| **kn** | 크ㄴ (k도 발음!) | knie (무릎) → '크니', knoop (매듭) → '크노프' |
| **wr** | 브ㄹ (w도 발음!) | wreed (잔인한), wraak (복수) |
| **ts** | ㅊ | ets (판화), fiets (자전거) |
| **nk** | ㅇㅋ | denken (생각하다), drinken (마시다) |
| **sch-** (어두) | 스흐 | school (학교), schip (배) |
| **-isch** (어말) | -이스 | typisch (전형적인), logisch (논리적인) |

> **영어와의 핵심 차이 5가지**:
> 1. **th**는 그냥 **t** 발음 (영어처럼 혀를 이 사이에 넣지 않음)
> 2. **kn**의 **k**를 발음함 (영어 knee에서 k는 묵음이지만 네덜란드어에선 발음)
> 3. 어말의 **d**는 **t**로, **b**는 **p**로 발음 (유성음의 무성음화)
> 4. **g/ch**는 목 뒤쪽 마찰음 (영어에 없는 소리)
> 5. **r**은 혀를 굴리거나 목울림 (지역마다 다름)

#### 발음의 지역 차이

네덜란드어의 발음은 지역에 따라 상당히 다릅니다:

| 특징 | 북부/표준 (Randstad) | 남부 (Brabant, Limburg) | 벨기에 (Vlaanderen) |
|------|---------------------|----------------------|-------------------|
| **g/ch** | 거친 흐 (hard g) | 부드러운 흐 (zachte g) | 매우 부드러운 흐 |
| **r** | 목울림 (Gooise r) | 혀 굴림 | 혀 굴림 |
| **n** (어말) | 종종 탈락 | 발음 | 발음 |
| 전체 느낌 | 거칠고 직접적 | 부드럽고 노래하는 듯 | 차분하고 또렷한 |

> **초보자 팁**: 남부/벨기에식 부드러운 g가 발음하기 더 쉽습니다. 처음에는 부드러운 g로 시작하고 점차 표준 발음에 익숙해지세요.

#### 철자와 발음 규칙 — 복수형/활용형에서의 변화

단어가 변화할 때 발음을 유지하기 위해 철자가 바뀝니다:

```
1. 단모음 유지를 위한 자음 겹침:
   man → mannen (남자들) — a가 짧아야 하므로 nn
   kat → katten (고양이들) — a가 짧아야 하므로 tt
   bos → bossen (숲들) — o가 짧아야 하므로 ss

2. 장모음이면 자음을 하나로:
   maan → manen (달들) — aa=a, 열린 음절이므로 n 하나
   boom → bomen (나무들) — oo=o, 열린 음절이므로 m 하나
   duur → dure (비싼+e) — uu=u, 열린 음절이므로 r 하나

3. 유성/무성 교체:
   huis → huizen (집들) — s → z
   brief → brieven (편지들) — f → v
   lief → lieve (다정한+e) — f → v
   ijs → ijzig (얼음의) — s → z
```

#### 발음 연습: 최소 대립쌍 (Minimale paren)

발음 차이로 의미가 달라지는 단어쌍을 연습하세요:

```
짧은 모음 vs 긴 모음:
man (남자) ↔ maan (달)
tak (가지) ↔ taak (과제)
bom (폭탄) ↔ boom (나무)
mus (참새) ↔ muur (벽)
lek (누수) ↔ leek (외부인)
bon (영수증) ↔ boon (콩)
pot (냄비) ↔ poot (발/다리)
zon (태양) ↔ zoon (아들)
hel (지옥) ↔ heel (전체/매우)
pit (씨앗) ↔ piet (이름)

이중모음 연습:
huis (집) — ui
muis (쥐) — ui
trein (기차) — ei
wijn (와인) — ij
blauw (파란) — au
koud (추운) — ou
boek (책) — oe
deur (문) — eu
```

#### 발음 연습: 혀 꼬이는 문장 (Tongbrekers)

```
1. "Acht achtentachtig prachtige grachten."
   (여든여덟 개의 아름다운 운하.)
   → ch/g 연습

2. "De kat krabt de krullen van de trap."
   (고양이가 계단에서 곱슬을 긁는다.)
   → kr, tr 연습

3. "Schietschijven schieten is moeilijk."
   (사격 과녁 쏘기는 어렵다.)
   → sch 연습

4. "Een heel klein konijntje."
   (아주 작은 토끼.)
   → ij, ei, ee 연습

5. "De groene groenten groeien in de grond."
   (초록 채소들이 땅에서 자란다.)
   → gr, oe 연습
```

> **팁**: 네덜란드어의 `g`와 `ch` 발음은 한국어에 없는 소리입니다. 목 뒤쪽에서 가래를 뱉듯이 거칠게 '흐' 소리를 내세요. 매일 5분씩 위의 혀 꼬이는 문장을 소리내어 읽으면 빠르게 익숙해집니다.

### 격식 vs 비격식 상세

네덜란드어에서는 상대방에 따라 **u** (격식)와 **je/jij** (비격식)를 구분합니다.

#### 격식 (u) 사용 상황

- 처음 만난 어른
- 나보다 나이가 많은 사람
- 직장 상사 (일부 회사)
- 관공서, 병원 등 공식 장소
- 고객 서비스 (전화, 이메일)
- 왕실/공식 행사

#### 비격식 (je/jij) 사용 상황

- 친구, 가족
- 또래, 동료 (대부분의 직장)
- 아이들
- 학생끼리
- SNS, 채팅
- 대부분의 일상 상황

> **문화 팁**: 네덜란드는 비교적 격식을 덜 따지는 문화입니다.
> - 많은 직장에서 상사에게도 **je/jij**를 사용합니다.
> - 모르는 사람에게도 비격식을 쓰는 경우가 많아지고 있습니다.
> - 확실하지 않으면 **u**로 시작하고, 상대가 **je로 해도 돼요 (Je mag je zeggen)**라고 하면 바꾸세요.
> - 벨기에에서는 네덜란드보다 격식을 더 중시하는 편입니다.

### 예시 대화: 격식 vs 비격식 비교

#### 격식 대화 (공식 상황)

```
A: Goedemiddag. Hoe maakt u het?
   (좋은 오후입니다. 안녕하십니까?)

B: Goed, dank u wel. En u?
   (잘 지냅니다, 감사합니다. 당신은요?)

A: Ook goed, dank u. Kan ik u helpen?
   (저도 잘 지냅니다, 감사합니다. 도와드릴까요?)

B: Ja, alstublieft. Ik zoek het station.
   (네, 부탁드립니다. 역을 찾고 있습니다.)

A: Het station is daar, rechtsaf.
   (역은 저기, 오른쪽입니다.)

B: Hartelijk bedankt. Prettige dag nog!
   (진심으로 감사합니다. 좋은 하루 보내세요!)

A: U ook! Tot ziens.
   (당신도요! 안녕히 가세요.)
```

#### 비격식 대화 (친구 사이)

```
A: Hoi! Hoe is het?
   (안녕! 어때?)

B: Goed hoor, en met jou?
   (잘 지내, 너는?)

A: Prima! Ga je mee koffie drinken?
   (좋아! 커피 마시러 갈래?)

B: Ja, leuk! Waar?
   (응, 좋아! 어디서?)

A: Bij het café op de hoek.
   (모퉁이에 있는 카페에서.)

B: Oké, tot zo!
   (오케이, 이따 봐!)

A: Doei!
   (안녕!)
```

### 예시 대화 1: 친근한 자기소개 (비격식)

```
A: Hallo! Hoe heet je?
   (안녕! 이름이 뭐야?)

B: Hoi! Ik ben Minho. En jij?
   (안녕! 나는 민호야. 너는?)

A: Ik heet Anna. Waar kom je vandaan?
   (나는 안나야. 어디에서 왔어?)

B: Ik kom uit Zuid-Korea. Ik woon nu in Amsterdam.
   (한국에서 왔어. 지금 암스테르담에 살고 있어.)

A: Leuk! Spreek je Nederlands?
   (좋다! 네덜란드어 할 줄 알아?)

B: Een beetje. Ik leer Nederlands. Het is moeilijk!
   (조금. 네덜란드어를 배우고 있어. 어려워!)

A: Haha, dat klopt. Wat doe je hier in Amsterdam?
   (하하, 맞아. 암스테르담에서 뭐 해?)

B: Ik werk als programmeur bij een tech bedrijf. En jij?
   (테크 회사에서 프로그래머로 일해. 너는?)

A: Ik studeer geneeskunde aan de universiteit.
   (대학교에서 의학을 공부해.)

B: Wat leuk! Hoe oud ben je?
   (멋지다! 몇 살이야?)

A: Ik ben drieëntwintig. En jij?
   (스물셋이야. 너는?)

B: Ik ben achtentwintig.
   (스물여덟이야.)
```

### 예시 대화 2: 격식있는 자기소개 (비즈니스/공식)

```
A: Goedemiddag. Mag ik me voorstellen? Mijn naam is Jan de Vries.
   (좋은 오후입니다. 제 소개를 해도 될까요? 이름은 얀 더 프리스입니다.)

B: Aangenaam, meneer De Vries. Ik ben Kim Minho.
   (반갑습니다, 더 프리스 씨. 저는 김민호입니다.)

A: Aangenaam. Komt u uit Korea?
   (반갑습니다. 한국에서 오셨나요?)

B: Ja, ik kom uit Seoul. Ik werk nu bij Samsung in Nederland.
   (네, 서울에서 왔습니다. 지금 네덜란드에서 삼성에서 일하고 있습니다.)

A: Wat interessant. Hoe lang woont u al in Nederland?
   (흥미롭군요. 네덜란드에 얼마나 사셨나요?)

B: Ik woon hier sinds twee jaar. Ik vind het een mooi land.
   (2년 전부터 살고 있습니다. 아름다운 나라라고 생각합니다.)

A: Dank u wel. Spreekt u ook Nederlands?
   (감사합니다. 네덜란드어도 하시나요?)

B: Ik leer Nederlands. Het is niet makkelijk, maar ik doe mijn best.
   (네덜란드어를 배우고 있습니다. 쉽지 않지만 최선을 다하고 있어요.)
```

#### 전화번호 읽기

네덜란드 전화번호는 숫자를 **한 자리 또는 두 자리씩** 끊어 읽습니다:

```

"nul zes, twaalf, vierendertig, zesenvijftig, achtenzeventig"

"nul twintig, een twee drie, vijfenveertig, zevenenzestig"

Waar is de dichtstbijzijnde supermarkt?  (가장 가까운 슈퍼마켓이 어디에요?)
```

> **파생 의문사**:
> - **Waarnaartoe?** = 어디로? (Waar ga je naartoe?)
> - **Waarvandaan?** = 어디에서? (Waar kom je vandaan?)
> - **Waarin?** = 무엇 안에? (Waarin zit het?)
> - **Waarop?** = 무엇 위에? (Waarop zit je?)

#### Wanneer? (언제?)

```
Wanneer kom je?               (언제 와?)
Wanneer begint de film?       (영화가 언제 시작해?)
Wanneer is je verjaardag?     (생일이 언제야?)
Wanneer heb je tijd?          (언제 시간 있어?)
Wanneer ben je klaar?         (언제 끝나?)
Wanneer vertrek je?           (언제 떠나?)
Sinds wanneer woon je hier?   (언제부터 여기에 살았어?)
Tot wanneer ben je er?        (언제까지 있을 거야?)
```

#### Waarom? (왜?)

```
Waarom leer je Nederlands?    (왜 네덜란드어를 배워?)
Waarom niet?                  (왜 안 돼?)
Waarom ben je laat?           (왜 늦었어?)
Waarom ga je weg?             (왜 가?)
Waarom is dat zo?             (왜 그래?)
Waarom doe je dat?            (왜 그렇게 해?)
Waarom vraag je dat?          (왜 그걸 물어봐?)
```

#### Hoe? (어떻게?) — 다양한 조합

```
Hoe gaat het?                 (어떻게 지내?)
Hoe heet je?                  (이름이 뭐야? — 직역: 어떻게 불려?)
Hoe kom ik daar?              (거기에 어떻게 가?)
Hoe zeg je dat in het Nederlands? (그걸 네덜란드어로 어떻게 말해?)
Hoe laat is het?              (몇 시야?)
Hoe lang duurt het?           (얼마나 걸려?)
Hoe ver is het?               (얼마나 먼 거야?)
Hoe oud ben je?               (몇 살이야?)
Hoe groot is je huis?         (집이 얼마나 커?)
Hoe duur is het?              (얼마나 비싸?)
Hoe vaak ga je naar de sportschool? (얼마나 자주 헬스장에 가?)
Hoe gaat het met je moeder?   (어머니는 어떻게 지내셔?)
Hoe was je vakantie?          (휴가는 어땠어?)
Hoe smaakt het?               (맛이 어때?)
```

#### Hoeveel? (얼마나? — 수량/가격)

```
Hoeveel kost het?             (얼마예요?)
Hoeveel kinderen heb je?      (아이가 몇 명이야?)
Hoeveel tijd heb je nodig?    (시간이 얼마나 필요해?)
Hoeveel mensen komen er?      (몇 명이 와?)
Hoeveel dagen blijf je?       (며칠 있어?)
Hoeveel talen spreek je?      (몇 개 언어를 해?)
Hoeveel is twee plus drie?    (2 더하기 3은 얼마야?)
```

#### Welk(e)? (어떤/어느?)

```
Welke kleur wil je?           (어떤 색을 원해?)
Welk boek lees je?            (어떤 책 읽어?) — het boek → welk
Welke taal spreek je?         (어떤 언어를 해?) — de taal → welke
Welke dag past jou?           (어떤 요일이 너한테 맞아?)
Welk station is het?          (어느 역이야?)
In welke stad woon je?        (어느 도시에 살아?)
Welke maat heb je?            (사이즈가 어떻게 돼?)
Welke film wil je zien?       (어떤 영화를 보고 싶어?)
```

> **welk vs welke 규칙**:
> - **het 명사** 앞 → **welk** (welk boek, welk huis)
> - **de 명사** 앞 → **welke** (welke man, welke vrouw)
> - **복수 명사** 앞 → **welke** (welke boeken, welke huizen)

### 대답 패턴

의문사 질문에 대한 답변 구조:

```
Wie? → 사람으로 답
  Wie is dat? → Dat is mijn broer. (그건 내 형이야.)

Wat? → 사물/행동으로 답
  Wat doe je? → Ik ben student. (나는 학생이야.)

Waar? → 장소로 답
  Waar woon je? → Ik woon in Utrecht. (위트레흐트에 살아.)

Wanneer? → 시간으로 답
  Wanneer kom je? → Ik kom morgen. (내일 와.)

Waarom? → 이유로 답 (Omdat...)
  Waarom leer je Nederlands? → Omdat ik in Nederland woon.
  (네덜란드에 살기 때문에.)

Hoe? → 방법/상태로 답
  Hoe gaat het? → Goed, dank je! (잘 지내, 고마워!)

Hoeveel? → 숫자로 답
  Hoeveel kost het? → Het kost vijf euro. (5유로예요.)

Welk(e)? → 특정 항목으로 답
  Welke kleur? → Blauw. (파란색.)
```

### 예시 대화: 의문사 활용

```
A: Hoi! Waar ga je naartoe?
   (안녕! 어디 가?)

B: Ik ga naar de supermarkt. En jij?
   (슈퍼마켓에 가. 너는?)

A: Ik ga naar huis. Hoelang ben je al in Nederland?
   (나는 집에 가. 네덜란드에 얼마나 있었어?)

B: Sinds september. Vier maanden nu.
   (9월부터. 지금 4개월째.)

A: Waarom ben je naar Nederland gekomen?
   (왜 네덜란드에 왔어?)

B: Omdat ik hier werk. Ik werk bij een IT bedrijf.
   (여기서 일하니까. IT 회사에서 일해.)

A: Welk bedrijf?
   (어느 회사?)

B: Booking.com, in Amsterdam.
   (부킹닷컴, 암스테르담에서.)

A: Wat leuk! Hoe ga je naar je werk?
   (좋겠다! 출근은 어떻게 해?)

B: Met de fiets, natuurlijk! Het is maar tien minuten.
   (자전거로, 당연히! 10분밖에 안 걸려.)

A: Haha, typisch Nederlands! Hoe vind je het hier?
   (하하, 전형적인 네덜란드식이네! 여기 어때?)

B: Ik vind het leuk, maar het weer is niet zo goed.
   (좋아, 근데 날씨가 별로야.)
```
