# Level 1: 기초 입문

> 네덜란드어의 알파벳, 발음, 기본 인사와 자기소개를 배웁니다.

## 1-1. 알파벳과 발음

네덜란드어는 라틴 알파벳 26자를 사용합니다. 영어와 같은 글자를 쓰지만 발음이 다릅니다.

### 알파벳

| 글자 | 네덜란드어 이름 | 발음 (한국어 근사) |
|------|----------------|-------------------|
| A a | aa | 아 |
| B b | bee | 베 |
| C c | cee | 세 |
| D d | dee | 데 |
| E e | ee | 에 |
| F f | ef | 에프 |
| G g | gee | 헤 (목 뒤쪽에서 거칠게) |
| H h | ha | 하 |
| I i | ie | 이 |
| J j | jee | 예 |
| K k | ka | 카 |
| L l | el | 엘 |
| M m | em | 엠 |
| N n | en | 엔 |
| O o | oo | 오 |
| P p | pee | 페 |
| Q q | ku | 퀴 |
| R r | er | 에르 (혀를 굴리거나 목울림) |
| S s | es | 에스 |
| T t | tee | 테 |
| U u | uu | 위 (입을 둥글게, 프랑스어 u와 비슷) |
| V v | vee | 베/페 (영어 v보다 부드럽게) |
| W w | wee | 붸 (영어 w와 v 사이) |
| X x | iks | 익스 |
| Y y | ij/Griekse ij | 에이 |
| Z z | zet | 젯 |

### 핵심 발음 규칙

#### 모음

| 표기 | 발음 | 예시 |
|------|------|------|
| **a** (닫힌 음절) | 짧은 '아' | **m**a**n** (남자) |
| **aa** / **a** (열린 음절) | 긴 '아' | **m**a**a**n (달), **n**a**me** (이름) |
| **e** (닫힌 음절) | 짧은 '에' | **b**e**d** (침대) |
| **ee** / **e** (열린 음절) | 긴 '에이' | **b**ee**n** (다리), **l**e**ven** (살다) |
| **i** (닫힌 음절) | 짧은 '이' | **d**i**t** (이것) |
| **ie** | 긴 '이' | **d**ie**r** (동물) |
| **o** (닫힌 음절) | 짧은 '오' | **b**o**s** (숲) |
| **oo** / **o** (열린 음절) | 긴 '오' | **b**oo**m** (나무), **z**o (그래서) |
| **u** (닫힌 음절) | 짧은 '위' | **b**u**s** (버스) |
| **uu** / **u** (열린 음절) | 긴 '위' | **d**uu**r** (비싼), **m**u**ren** (벽들) |

#### 이중모음

| 표기 | 발음 | 예시 |
|------|------|------|
| **ij / ei** | '에이' | **ij**s (아이스크림), **ei** (알) |
| **ou / au** | '아우' | **ou**d (오래된), **au**to (자동차) |
| **ui** | '아위' (독특한 발음) | **ui**t (밖으로), h**ui**s (집) |
| **oe** | '우' | g**oe**d (좋은), **oe**ver (강변) |
| **eu** | '외' | d**eu**r (문), l**eu**k (재미있는) |
| **ie** | '이' | d**ie** (그/그것), br**ie**f (편지) |

#### 자음 — 영어와 다른 점

| 표기 | 발음 | 설명 |
|------|------|------|
| **g** | ㅎ/흐 (거친) | 목 뒤에서 마찰음, **g**oed (좋은) |
| **ch** | ㅎ/흐 (거친) | g와 거의 동일, na**ch**t (밤) |
| **sch** | 스흐 | **sch**ool (학교) |
| **j** | ㅇ+야 | 영어 y 소리, **j**a (예) |
| **v** | ㅂ/ㅍ | 영어 v보다 약함, **v**riend (친구) |
| **w** | ㅂ+ㅜ | 영어 w와 v 사이, **w**ater (물) |
| **r** | ㄹ (굴림) | 지역마다 다름, **r**ood (빨간) |
| **-d** (어말) | ㅌ | goe**d** → '훗' (어미의 d는 t 발음) |
| **-b** (어말) | ㅂ→ㅍ | he**b** → '헵' |

> **팁**: 네덜란드어의 `g`와 `ch` 발음은 한국어에 없는 소리입니다. 목 뒤쪽에서 가래를 뱉듯이 거칠게 '흐' 소리를 내세요.

## 1-2. 기본 인사와 표현

### 인사

| 네덜란드어 | 발음 | 한국어 |
|-----------|------|-------|
| Hallo | 할로 | 안녕하세요 (일반) |
| Hoi | 호이 | 안녕 (친근) |
| Goedemorgen | 후더모르헌 | 좋은 아침이에요 |
| Goedemiddag | 후더미닥 | 좋은 낮이에요 |
| Goedenavond | 후더나본트 | 좋은 저녁이에요 |
| Goedenacht | 후더나흐트 | 잘 자요 |
| Dag! | 닥 | 안녕! (인사/작별 모두) |
| Tot ziens | 톳 진스 | 안녕히 가세요/계세요 |
| Doei | 두이 | 잘 가 (친근) |

### 기본 예의 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| Alstublieft (격식) / Alsjeblieft (비격식) | ~주세요 / 여기 있습니다 |
| Dank u wel (격식) / Dank je wel (비격식) | 감사합니다 |
| Bedankt | 고마워요 |
| Graag gedaan | 천만에요 |
| Sorry / Pardon | 죄송합니다 / 실례합니다 |
| Neem me niet kwalijk | 실례합니다 (격식) |
| Ja | 예 |
| Nee | 아니오 |

### 격식 vs 비격식

네덜란드어에서는 상대방에 따라 **u** (격식)와 **je/jij** (비격식)를 구분합니다.

- **u**: 모르는 사람, 윗사람, 공식 상황
- **je/jij**: 친구, 가족, 또래

> 네덜란드는 비교적 격식을 덜 따지는 문화여서, 많은 상황에서 **je/jij**를 사용합니다.

## 1-3. 자기소개

### 핵심 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik ben [이름]. | 저는 [이름]입니다. |
| Mijn naam is [이름]. | 제 이름은 [이름]입니다. |
| Ik kom uit [나라]. | 저는 [나라]에서 왔습니다. |
| Ik woon in [도시]. | 저는 [도시]에 살고 있습니다. |
| Ik ben [숫자] jaar oud. | 저는 [숫자]살입니다. |
| Ik spreek Koreaans. | 저는 한국어를 합니다. |
| Ik leer Nederlands. | 저는 네덜란드어를 배우고 있습니다. |
| Aangenaam! | 만나서 반갑습니다! |
| Hoe heet je? (비격식) | 이름이 뭐예요? |
| Hoe heet u? (격식) | 성함이 어떻게 되세요? |
| Hoe gaat het? | 어떻게 지내세요? |
| Goed, dank je! | 잘 지내요, 고마워요! |

### 예시 대화

```
A: Hallo! Hoe heet je?
   (안녕! 이름이 뭐야?)

B: Hoi! Ik ben Minho. En jij?
   (안녕! 나는 민호야. 너는?)

A: Ik heet Anna. Waar kom je vandaan?
   (나는 안나야. 어디에서 왔어?)

B: Ik kom uit Zuid-Korea. Ik woon nu in Amsterdam.
   (한국에서 왔어. 지금 암스테르담에 살고 있어.)

A: Leuk! Spreek je Nederlands?
   (좋다! 네덜란드어 할 줄 알아?)

B: Een beetje. Ik leer Nederlands.
   (조금. 네덜란드어를 배우고 있어.)
```

| 0 | nul | 11 | elf |
| 1 | een | 12 | twaalf |
| 2 | twee | 13 | dertien |
| 3 | drie | 14 | veertien |
| 4 | vier | 15 | vijftien |
| 5 | vijf | 16 | zestien |
| 6 | zes | 17 | zeventien |
| 7 | zeven | 18 | achttien |
| 8 | acht | 19 | negentien |
| 9 | negen | 20 | twintig |
| 10 | tien | 100 | honderd |

### 21~99: 일의 자리를 먼저 말함

- 21 = **eenentwintig** (een-en-twintig, 1 그리고 20)
- 35 = **vijfendertig** (vijf-en-dertig, 5 그리고 30)
- 99 = **negenennegentig** (negen-en-negentig, 9 그리고 90)

> **주의**: 독일어처럼 일의 자리를 먼저 읽습니다! 25 = vijfentwintig (5와 20)

## 연습 문제

1. 네덜란드어로 자기소개를 해보세요 (이름, 출신, 거주지).
2. 1~20까지 네덜란드어로 세어보세요.
3. 오늘 요일과 이번 달을 네덜란드어로 말해보세요.
4. 다음을 네덜란드어로 번역하세요:
   - "안녕하세요, 만나서 반갑습니다."
   - "이것은 얼마예요?"
   - "감사합니다, 안녕히 계세요."
