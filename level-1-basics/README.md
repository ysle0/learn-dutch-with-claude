# Level 1: 기초 입문

> 네덜란드어의 알파벳, 발음, 기본 인사와 자기소개를 배웁니다.

## 1-1. 알파벳과 발음

네덜란드어는 라틴 알파벳 26자를 사용합니다. 영어와 같은 글자를 쓰지만 발음이 다릅니다.

| C c | cee | 세 |
| D d | dee | 데 |

### 핵심 발음 규칙

#### 모음

| 표기 | 발음 | 예시 |
|------|------|------|
| **a** (닫힌 음절) | 짧은 '아' | **m**a**n** (남자) |
| **aa** / **a** (열린 음절) | 긴 '아' | **m**a**a**n (달), **n**a**me** (이름) |
| **e** (닫힌 음절) | 짧은 '에' | **b**e**d** (침대) |
| **ee** / **e** (열린 음절) | 긴 '에이' | **b**ee**n** (다리), **l**e**ven** (살다) |
| **i** (닫힌 음절) | 짧은 '이' | **d**i**t** (이것) |
| **ie** | 긴 '이' | **d**ie**r** (동물) |
| **o** (닫힌 음절) | 짧은 '오' | **b**o**s** (숲) |
| **oo** / **o** (열린 음절) | 긴 '오' | **b**oo**m** (나무), **z**o (그래서) |
| **u** (닫힌 음절) | 짧은 '위' | **b**u**s** (버스) |
| **uu** / **u** (열린 음절) | 긴 '위' | **d**uu**r** (비싼), **m**u**ren** (벽들) |

#### 이중모음

| 표기 | 발음 | 예시 |
|------|------|------|
| **ij / ei** | '에이' | **ij**s (아이스크림), **ei** (알) |
| **ou / au** | '아우' | **ou**d (오래된), **au**to (자동차) |
| **ui** | '아위' (독특한 발음) | **ui**t (밖으로), h**ui**s (집) |
| **oe** | '우' | g**oe**d (좋은), **oe**ver (강변) |
| **eu** | '외' | d**eu**r (문), l**eu**k (재미있는) |
| **ie** | '이' | d**ie** (그/그것), br**ie**f (편지) |

#### 자음 — 영어와 다른 점

| 표기 | 발음 | 설명 |
|------|------|------|
| **g** | ㅎ/흐 (거친) | 목 뒤에서 마찰음, **g**oed (좋은) |
| **ch** | ㅎ/흐 (거친) | g와 거의 동일, na**ch**t (밤) |
| **sch** | 스흐 | **sch**ool (학교) |
| **j** | ㅇ+야 | 영어 y 소리, **j**a (예) |
| **v** | ㅂ/ㅍ | 영어 v보다 약함, **v**riend (친구) |
| **w** | ㅂ+ㅜ | 영어 w와 v 사이, **w**ater (물) |
| **r** | ㄹ (굴림) | 지역마다 다름, **r**ood (빨간) |
| **-d** (어말) | ㅌ | goe**d** → '훗' (어미의 d는 t 발음) |
| **-b** (어말) | ㅂ→ㅍ | he**b** → '헵' |

> **팁**: 네덜란드어의 `g`와 `ch` 발음은 한국어에 없는 소리입니다. 목 뒤쪽에서 가래를 뱉듯이 거칠게 '흐' 소리를 내세요.

### 기본 예의 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| Alstublieft (격식) / Alsjeblieft (비격식) | ~주세요 / 여기 있습니다 |
| Dank u wel (격식) / Dank je wel (비격식) | 감사합니다 |
| Bedankt | 고마워요 |
| Graag gedaan | 천만에요 |
| Sorry / Pardon | 죄송합니다 / 실례합니다 |
| Neem me niet kwalijk | 실례합니다 (격식) |
| Ja | 예 |
| Nee | 아니오 |

### 격식 vs 비격식

네덜란드어에서는 상대방에 따라 **u** (격식)와 **je/jij** (비격식)를 구분합니다.

- **u**: 모르는 사람, 윗사람, 공식 상황
- **je/jij**: 친구, 가족, 또래

> 네덜란드는 비교적 격식을 덜 따지는 문화여서, 많은 상황에서 **je/jij**를 사용합니다.

## 1-3. 자기소개

### 핵심 표현

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik ben [이름]. | 저는 [이름]입니다. |
| Mijn naam is [이름]. | 제 이름은 [이름]입니다. |
| Ik kom uit [나라]. | 저는 [나라]에서 왔습니다. |
| Ik woon in [도시]. | 저는 [도시]에 살고 있습니다. |
| Ik ben [숫자] jaar oud. | 저는 [숫자]살입니다. |
| Ik spreek Koreaans. | 저는 한국어를 합니다. |
| Ik leer Nederlands. | 저는 네덜란드어를 배우고 있습니다. |
| Aangenaam! | 만나서 반갑습니다! |
| Hoe heet je? (비격식) | 이름이 뭐예요? |
| Hoe heet u? (격식) | 성함이 어떻게 되세요? |
| Hoe gaat het? | 어떻게 지내세요? |
| Goed, dank je! | 잘 지내요, 고마워요! |

### 예시 대화

```
A: Hallo! Hoe heet je?
   (안녕! 이름이 뭐야?)

B: Hoi! Ik ben Minho. En jij?
   (안녕! 나는 민호야. 너는?)

A: Ik heet Anna. Waar kom je vandaan?
   (나는 안나야. 어디에서 왔어?)

B: Ik kom uit Zuid-Korea. Ik woon nu in Amsterdam.
   (한국에서 왔어. 지금 암스테르담에 살고 있어.)

A: Leuk! Spreek je Nederlands?
   (좋다! 네덜란드어 할 줄 알아?)

B: Een beetje. Ik leer Nederlands.
   (조금. 네덜란드어를 배우고 있어.)
```

### 숫자의 실생활 활용

#### 나이 말하기

```
Hoe oud ben je?        (몇 살이에요?)
Ik ben 25 jaar oud.    (저는 25살이에요.)
Ik ben vijfentwintig.  (저는 스물다섯이에요.)
Hij is 40 jaar oud.    (그는 40살이에요.)
Zij is 8 jaar oud.     (그녀는 8살이에요.)
Mijn oma is 80.        (우리 할머니는 80세에요.)
De baby is 6 maanden oud. (아기는 6개월이에요.)
```

#### 가격과 쇼핑

```
Hoeveel kost het?                      (이거 얼마예요?)
Het kost vijf euro.                    (5유로예요.)
Dat is drie euro vijftig.              (3유로 50센트예요.) → €3,50
Het kost twaalf euro negenennegentig.  (12유로 99센트예요.) → €12,99
Twee broodjes voor vier euro.          (빵 두 개에 4유로.)
Een kopje koffie kost twee vijftig.    (커피 한 잔에 2유로 50센트.)
De huur is achthonderd euro per maand. (월세는 800유로예요.)
Dat is te duur!                        (그건 너무 비싸요!)
Heeft u korting?                       (할인 있나요?)
Het is in de uitverkoop: vijftig procent korting. (세일 중이에요: 50% 할인.)
```

#### 전화번호 읽기

네덜란드 전화번호는 숫자를 **한 자리 또는 두 자리씩** 끊어 읽습니다:

```
06-1234-5678:
"nul zes, twaalf, vierendertig, zesenvijftig, achtenzeventig"

020-123-4567 (암스테르담 지역번호):

#### 시간 말하기 (기본)

```
Hoe laat is het?              (몇 시예요?)
Het is drie uur.              (3시예요.)
Het is half vier.             (3시 반이에요.) ⚠️ 네덜란드어에서는 "4의 반"이라고 표현!
Het is kwart over twee.       (2시 15분이에요.)
Het is kwart voor vijf.       (4시 45분이에요. = 5시 15분 전)
Het is tien over zes.         (6시 10분이에요.)
Het is vijf voor acht.        (7시 55분이에요. = 8시 5분 전)
Het is tien voor half negen.  (8시 20분이에요. = 8시 반의 10분 전)
Het is vijf over half elf.    (10시 35분이에요. = 10시 반의 5분 후)
Om hoe laat?                  (몇 시에?)
Om negen uur.                 (9시에.)
Om half twee.                 (1시 반에.)
```

> **⚠️ 중요**: 네덜란드어 시간에서 **half**는 **다음 시간의 반**입니다!
> - half drie = 2:30 (3시의 반 = 2시 30분) ❌ 3시 30분이 아님!
> - half acht = 7:30 (8시의 반 = 7시 30분) ❌ 8시 30분이 아님!

```
