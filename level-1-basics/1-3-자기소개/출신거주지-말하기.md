# 출신/거주지 말하기

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik kom uit [나라]. | 저는 [나라]에서 왔습니다. |
| Ik kom uit Zuid-Korea. | 저는 한국에서 왔습니다. |
| Waar kom je vandaan? | 어디에서 왔어? (비격식) |
| Waar komt u vandaan? | 어디에서 오셨어요? (격식) |
| Ik woon in [도시]. | 저는 [도시]에 살고 있습니다. |
| Ik woon in Amsterdam. | 저는 암스테르담에 살고 있어요. |
| Waar woon je? | 어디에 살아? |
| Ik ben [국적]. | 저는 [국적]입니다. |
| Ik ben Koreaans. (남) / Koreaanse. (여) | 저는 한국인입니다. |

#### 주요 나라와 국적

| 나라 (네덜란드어) | 한국어 | 국적 (남/여) |
|-----------------|-------|-------------|
| Zuid-Korea | 한국 | Koreaans / Koreaanse |
| Nederland | 네덜란드 | Nederlands / Nederlandse |
| België | 벨기에 | Belgisch / Belgische |
| Duitsland | 독일 | Duits / Duitse |
| Frankrijk | 프랑스 | Frans / Franse |
| Engeland | 영국 | Engels / Engelse |
| Amerika (VS) | 미국 | Amerikaans / Amerikaanse |
| Japan | 일본 | Japans / Japanse |
| China | 중국 | Chinees / Chinese |
| Spanje | 스페인 | Spaans / Spaanse |
| Italië | 이탈리아 | Italiaans / Italiaanse |
| Turkije | 터키 | Turks / Turkse |
| Marokko | 모로코 | Marokkaans / Marokkaanse |
| Indonesië | 인도네시아 | Indonesisch / Indonesische |
| Suriname | 수리남 | Surinaams / Surinaamse |

| 네덜란드어 | 한국어 | 참고 |
|-----------|-------|------|
| Amsterdam | 암스테르담 | 수도, 가장 큰 도시 |
| Rotterdam | 로테르담 | 유럽 최대 항구도시 |

| Groningen | 흐로닝언 | 북부 대학도시 |
| Leiden | 레이던 | 역사적 대학도시 |
| Maastricht | 마스트리흐트 | 남부 문화도시 |
| Delft | 델프트 | 도자기와 공대로 유명 |
| Breda | 브레다 | 남부 도시 |
