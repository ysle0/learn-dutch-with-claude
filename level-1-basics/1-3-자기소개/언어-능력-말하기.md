# 언어 능력 말하기

| 네덜란드어 | 한국어 |
|-----------|-------|
| Ik spreek Koreaans. | 저는 한국어를 합니다. |
| Ik spreek een beetje Nederlands. | 네덜란드어를 조금 해요. |
| Ik leer Nederlands. | 네덜란드어를 배우고 있어요. |
| Ik spreek ook Engels. | 영어도 해요. |
| Welke talen spreek je? | 무슨 언어를 해? |
| Ik spreek vloeiend Engels. | 영어를 유창하게 해요. |
| Ik kan een beetje Duits. | 독일어를 조금 할 수 있어요. |
