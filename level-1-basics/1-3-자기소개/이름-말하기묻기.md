# 자기소개 템플릿

자신의 정보를 넣어서 연습해 보세요:

```
Hallo! Ik heet __________.
Ik kom uit __________ en ik woon nu in __________.
Ik ben __________ jaar oud.
Ik werk als __________ / Ik studeer __________.
Ik spreek __________ en een beetje __________.
In mijn vrije tijd __________ ik graag.
Ik ben hier sinds __________.
Ik vind Nederland __________ (leuk/mooi/interessant).
Aangenaam!
```
