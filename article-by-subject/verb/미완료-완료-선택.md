# 미완료 vs. 완료 선택

네덜란드어 과거 시제인 미완료(imperfectum)와 완료(perfectum)를 언제 사용하는지 알아봅니다.

## 미완료 (Imperfectum) 사용

과거의 **배경, 맥락, 상황**을 설명할 때 사용합니다.

### 사용 상황

| 상황 | 예문 |
|------|------|
| 반복 행동 | Ik **fietste** elke dag naar school. (매일 자전거로 학교에 갔다.) |
| 특정 시간 | Gisteren **regende** het hard. (어제 비가 많이 왔다.) |
| 묘사/배경 | De zon **scheen** en de vogels **zongen**. (해가 빛나고 새가 울었다.) |
| 지속, 결과 없음 | Ik **woonde** in Amsterdam. (암스테르담에 살았다.) |
| 동시 행동 | Terwijl ik **kookte**, **keek** hij tv. (내가 요리하는 동안 그가 TV를 봤다.) |

---

## 완료 (Perfectum) 사용

과거에 **완료된 사건**이 현재와 관련이 있을 때 사용합니다.

### 사용 상황

| 상황 | 예문 |
|------|------|
| 일회성 사건 | Ik **heb** gisteren een boek **gekocht**. (어제 책을 샀다.) |
| 시간 무관 | Ik **heb** in Amsterdam **gewoond**. (암스테르담에 살았던 적이 있다.) |
| 사실 진술 | Hij **is** in 1990 **geboren**. (그는 1990년에 태어났다.) |
| 현재까지 구체적 결과 | Ik **heb** mijn sleutel **verloren**. (열쇠를 잃어버렸다.) — 지금도 없다 |

---

## 핵심 구별법

| 질문 | 미완료 | 완료 |
|------|--------|------|
| 배경/맥락인가? | ✔ | |
| 반복적이었는가? | ✔ | |
| 일회성 사건인가? | | ✔ |
| 현재와 관련 있는가? | | ✔ |
| 묘사/상황 설명인가? | ✔ | |

### 비교 예시

- Ik **woonde** in Utrecht. (위트레흐트에 살았다.) — 배경, 지금은 아님
- Ik **heb** in Utrecht **gewoond**. (위트레흐트에 살았던 적이 있다.) — 경험

- Het **regende** toen ik buitenkwam. (내가 나왔을 때 비가 오고 있었다.) — 배경
- Het **heeft** vandaag **geregend**. (오늘 비가 왔다.) — 완료된 사실
