# 위치 동사 (Werkwoorden van positie)

네덜란드어에서 물건의 위치를 설명할 때 사용하는 동사입니다. 한국어의 "있다"와 달리 물건의 형태와 자세에 따라 다른 동사를 사용합니다.

## Staan (서 있다)

**수직으로 서 있는 것, 다리/바퀴 위에 있는 것, 텍스트/사진**에 사용합니다.

| 시제 | 형태 |
|------|------|
| 현재형 | sta / staat / staan |
| 과거형 | stond / stonden |
| 완료형 | heb gestaan |

- De fles **staat** op tafel. (병이 탁자 위에 서 있다.)
- Het **staat** in de krant. (신문에 실려 있다.)
- 이동 동사: **zetten** — Ik **zet** de fles op tafel. (병을 탁자 위에 놓다.)

## Liggen (누워 있다)

**수평으로 놓인 것, 지리적 위치**에 사용합니다.

| 시제 | 형태 |
|------|------|
| 현재형 | lig / ligt / liggen |
| 과거형 | lag / lagen |
| 완료형 | heb gelegen |

- Het boek **ligt** op de bank. (책이 소파 위에 놓여 있다.)
- Amsterdam **ligt** in Nederland. (암스테르담은 네덜란드에 있다.)
- 이동 동사: **leggen** — Ik **leg** het boek op de bank. (책을 소파에 놓다.)

## Hangen (매달려 있다)

**벽이나 천장에 매달린 것**에 사용합니다.

| 시제 | 형태 |
|------|------|
| 현재형 | hang / hangt / hangen |
| 과거형 | hing / hingen |
| 완료형 | heb gehangen |

- De foto **hangt** aan de muur. (사진이 벽에 걸려 있다.)
- 이동 동사: **ophangen** — Ik **hang** de foto **op**. (사진을 걸다.)

## Zitten (들어 있다)

**용기/공간 안에 있는 것**에 사용합니다.

| 시제 | 형태 |
|------|------|
| 현재형 | zit / zit / zitten |
| 과거형 | zat / zaten |
| 완료형 | heb gezeten |

- De melk **zit** in de koelkast. (우유가 냉장고 안에 있다.)
- Er **zit** een gat in mijn broek. (내 바지에 구멍이 있다.)
- 이동 동사: **doen** — Ik **doe** de melk in de koelkast. (우유를 냉장고에 넣다.)

## Lopen (이어지다)

**A에서 B로 이어지는 긴 형태**에 사용합니다.

- De weg **loopt** van Amsterdam naar Utrecht. (길이 암스테르담에서 위트레흐트까지 이어진다.)
- Er **loopt** een scheur door de muur. (벽에 금이 가 있다.)
