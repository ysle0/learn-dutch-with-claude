# 감정과 원인 (Feelings & Emotions and Their Causes)

네덜란드어에서는 **상황(원인)**과 **사람(감정)**을 서로 다른 형용사로 표현합니다. 원인을 나타내는 형용사와 그에 대응하는 감정 형용사를 함께 익히면 자연스러운 표현이 가능합니다.

> 출처: [Zichtbaar Nederlands - Adjective](https://zichtbaarnederlands.nl/en/adjective/)

---

### 기본 원인-감정 형용사 조합

| 원인 (상황) | 감정 (사람) | 한국어 뜻 |
|---|---|---|
| saai (지루한) | verveeld (지루해하는) | 지루한 상황 → 지루해하는 사람 |
| vervelend / irritant (짜증나는) | boos / kwaad / geïrriteerd (화난) | 짜증나는 상황 → 화가 난 사람 |
| grappig (재미있는) | vrolijk (즐거운) | 재미있는 상황 → 즐거운 사람 |
| eng (무서운) | bang (두려운) | 무서운 상황 → 두려워하는 사람 |
| vermoeiend (피곤하게 하는) | moe (피곤한) | 피곤하게 하는 상황 → 피곤한 사람 |
| treurig / triest (슬픈) | verdrietig (슬퍼하는) | 슬픈 상황 → 슬퍼하는 사람 |
| interessant (흥미로운) | geïnteresseerd (관심 있는) | 흥미로운 상황 → 관심 있는 사람 |
| spannend / stressvol (긴장되는) | nerveus / gestrest / gespannen (긴장한) | 긴장되는 상황 → 긴장한 사람 |

### 예문

- De film is **saai**. Ik ben **verveeld**. (영화가 지루하다. 나는 지루해한다.)
- Het werk is **vermoeiend**. Zij is **moe**. (일이 피곤하게 한다. 그녀는 피곤하다.)
- Het verhaal is **grappig**. Wij zijn **vrolijk**. (이야기가 재미있다. 우리는 즐겁다.)

---

### B2 수준: 고급 패턴

원인 형용사와 감정 형용사는 일정한 형태적 패턴을 따릅니다.

- **원인 형용사**: 기본 동사 + **-d** 또는 **-ant** (예: vermoei**end**, irrit**ant**)
- **감정 형용사**: **ge-** + 과거분사 (예: **ge**irriteer**d**, **ge**stres**t**)

---

### 추가 고급 어휘 (B2)

| 원인 (상황) | 감정 (사람) | 한국어 뜻 |
|---|---|---|
| alarmerend (경각심을 주는) | gealarmeerd (경각심을 느끼는) | 경고하는 → 경각 상태인 |
| amusant (즐겁게 하는) | geamuseerd (즐거워하는) | 재미있는 → 즐거워하는 |
| choquerend (충격적인) | gechoqueerd (충격받은) | 충격적인 → 충격받은 |
| deprimerend (우울하게 하는) | gedeprimeerd (우울한) | 우울하게 하는 → 우울해하는 |
| fascinerend (매혹적인) | gefascineerd (매혹된) | 매혹적인 → 매혹된 |
| teleurstellend (실망스러운) | teleurgesteld (실망한) | 실망스러운 → 실망한 |
| verbazend (놀라운) | verbaasd (놀란) | 놀라운 → 놀란 |

### 고급 예문

- Het nieuws is **choquerend**. De mensen zijn **gechoqueerd**. (뉴스가 충격적이다. 사람들은 충격받았다.)
- De resultaten zijn **teleurstellend**. De coach is **teleurgesteld**. (결과가 실망스럽다. 코치는 실망했다.)
