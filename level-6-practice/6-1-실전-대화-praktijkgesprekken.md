# 6-1. 실전 대화: 일상 시나리오

### 시나리오 1: 카페에서

```
A: Hoi! Wat wil je drinken?
   (안녕! 뭐 마실래?)

B: Doe mij maar een koffie verkeerd.
   (카페라테로 할게.)

A: Met suiker?
   (설탕 넣을까?)

B: Nee, zonder suiker, alsjeblieft.
   (아니, 설탕 없이, 부탁해.)

A: En een gebakje erbij?
   (케이크도 곁들일래?)

B: Ja, lekker! Een appeltaart graag.
   (응, 좋아! 사과파이 주세요.)

A: Komt eraan!
   (바로 나올게요!)
```

> **koffie verkeerd**: 직역하면 "잘못된 커피". 우유를 많이 넣은 커피 = 카페라테. 네덜란드 카페 필수 메뉴!

### 시나리오 2: 기차역에서

```
A: Goedemiddag. Een enkeltje naar Rotterdam, alstublieft.
   (안녕하세요. 로테르담 편도 한 장 주세요.)

B: Eerste of tweede klas?
   (1등석인가요, 2등석인가요?)

A: Tweede klas, graag. Van welk spoor vertrekt de trein?
   (2등석이요. 어느 플랫폼에서 출발하나요?)

B: Spoor 5. De trein vertrekt om 14:23.
   (5번 플랫폼이요. 기차는 14시 23분에 출발합니다.)

A: Moet ik overstappen?
   (환승해야 하나요?)

B: Nee, het is een directe verbinding.
   (아니요, 직행입니다.)

A: Prima, dank u wel!
   (좋아요, 감사합니다!)
```

> **enkeltje**: 편도 / **retourtje**: 왕복
> 네덜란드에서는 OV-chipkaart (교통카드)를 사용합니다.

### 시나리오 3: 의사 방문

```
A: Goedemorgen, wat kan ik voor u doen?
   (좋은 아침, 무엇을 도와드릴까요?)

B: Ik voel me niet zo lekker. Ik heb hoofdpijn en ik ben erg moe.
   (몸이 좀 안 좋아요. 두통이 있고 매우 피곤해요.)

A: Hoe lang heeft u deze klachten al?
   (이 증상이 얼마나 되셨나요?)

B: Sinds drie dagen.
   (3일 전부터요.)

A: Heeft u ook koorts?
   (열도 있나요?)

B: Ja, een beetje. 38 graden.
   (네, 약간. 38도요.)

A: Ik schrijf een recept voor u uit. Neem dit medicijn drie keer per dag.
   (처방전을 써드리겠습니다. 이 약을 하루 세 번 드세요.)

B: Dank u wel, dokter.
   (감사합니다, 선생님.)
```

### 시나리오 4: 집 구하기

```
A: Ik ben op zoek naar een huurwoning in Amsterdam.
   (암스테르담에서 임대 주택을 찾고 있어요.)

B: Wat voor type woning zoekt u?
   (어떤 유형의 주택을 찾고 계세요?)

A: Een tweekamerappartement, liefst in het centrum.
   (투룸 아파트, 가능하면 도심에요.)

B: Wat is uw budget?
   (예산이 어떻게 되세요?)

A: Maximaal 1500 euro per maand, inclusief servicekosten.
   (관리비 포함 월 최대 1500유로요.)

B: Ik heb iets op de Prinsengracht. Wilt u een bezichtiging plannen?
   (프린센허라흐트에 하나 있어요. 방문 예약하시겠어요?)

A: Ja, graag! Wanneer kan dat?
   (네, 부탁해요! 언제 가능할까요?)
```

### 시나리오 5: 전화 통화

```
A: Met Jan de Vries.
   (얀 더 프리스입니다.)

B: Goedemiddag, u spreekt met Kim. Is mevrouw Bakker aanwezig?
   (안녕하세요, 킴입니다. 바커 씨 계신가요?)

A: Nee, ze is op dit moment niet bereikbaar. Kan ik een boodschap doorgeven?
   (아니요, 지금 통화가 안 됩니다. 메시지를 전해드릴까요?)

B: Ja, wilt u vragen of ze mij kan terugbellen?
   (네, 저한테 다시 전화해 달라고 전해주시겠어요?)

A: Natuurlijk. Wat is uw telefoonnummer?
   (물론이죠. 전화번호가 어떻게 되세요?)

B: 06-12345678. Dank u wel!
   (06-12345678이요. 감사합니다!)
```
